## Introduction

Shopping for clothes in Dutch: **de maat**, **passen**, **het past**. Uses [Clothing and Colours](/notes/a1/22-clothing-and-colours) and [Shopping](/notes/a1/08-shopping). Vocabulary: [Word Hub → Clothing](/notes/word-hub/nouns#clothing-and-accessories).

## Quick Review

| From your course | You use it here |
| --- | --- |
| [A1 L22](/notes/a1/22-clothing-and-colours): sizes, colours | **Maat M, blauw** |
| [A1 L8](/notes/a1/08-shopping): **passen**, paying | **Mag ik dit passen?** |
| [A1 L9](/notes/a1/09-demonstratives): **deze/die** | **Deze trui** |

## Key patterns

| Pattern | Example |
| --- | --- |
| **Mag ik … passen?** | **Mag ik deze broek passen?** |
| **Heeft u maat …?** | **Heeft u maat 40?** |
| **Het past niet.** | It does not fit. |
| **Heeft u een maat groter?** | Do you have a bigger size? |

## Core Vocabulary

| Dutch | English |
| --- | --- |
| de kledingwinkel | clothing shop |
| de maat | size |
| passen | to try on |
| de paskamer | fitting room |
| te groot / te klein | too big / small |
| de broek / de trui / het shirt | trousers / jumper / shirt |
| de kleur | colour |
| het kassatje | till |

## Key phrases

| # | Staff | You |
| --- | --- | --- |
| 1 | **Kan ik u helpen?** | **Ik zoek een blauwe trui, maat M.** |
| 2 | **De paskamer is daar.** | **Dank u. Mag ik deze passen?** |
| 3 | **Hoe zit het?** | **Te klein. Heeft u maat L?** |
| 4 | **Die staat u goed.** | **Oké, ik neem hem.** |
| 5 | **Dat is €39,95.** | **Pin, alstublieft.** |

## Example Conversation

<ShadCnCard className="my-4">
  <CardHeader><CardTitle className="text-lg">Bij H&M / kledingwinkel</CardTitle></CardHeader>
  <CardContent>
    **In Dutch:**

    > **Medewerker:** Goedemiddag! Kan ik u helpen?
    >
    > **Klant:** Ja, ik zoek een zwarte broek, maat 32.
    >
    > **Medewerker:** Hier is een model. De paskamer is achterin.
    >
    > **Klant:** *(later)* Hij is te lang. Heeft u een kortere maat?
    >
    > **Medewerker:** Probeer deze. Korter in het been.
    >
    > **Klant:** Perfect. Ik neem hem. Dank u wel!

    **In English:**

    > **Staff:** Good afternoon! Can I help you?
    >
    > **Customer:** Yes, I am looking for black trousers, size 32.
    >
    > **Staff:** Here is a model. The fitting room is at the back.
    >
    > **Customer:** *(later)* They are too long. Do you have a shorter size?
    >
    > **Staff:** Try these. Shorter in the leg.
    >
    > **Customer:** Perfect. I will take them. Thank you!
  </CardContent>
</ShadCnCard>
