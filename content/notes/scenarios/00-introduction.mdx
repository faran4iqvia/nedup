## Welcome to Scenarios 🎭

These pages are **not** numbered lessons. They are focused conversation drills you can open when you need Dutch for a **specific situation** — before a trip, before an appointment, or before a real conversation you are nervous about.

Each scenario file is labelled by CEFR level (**a1-**, **a2-**, **b1-**) so you know which grammar and vocabulary it assumes. Study the linked lessons first, then rehearse the scenario aloud.

<Callout title="How to use a scenario page" type="info">
  1. Skim **Sentence structure** so you know the patterns.
  2. Read all **15 questions** — officers rarely ask more than a handful.
  3. Practise **solo** and **with family** answers out loud.
  4. Run the **dialogues** twice: once reading, once from memory.
  5. Do the **exercises** without peeking, then check the answer key.
</Callout>
