Now you learn the **rule** behind all of them: Dutch **scheidbare werkwoorden** (separable verbs) split in main sentences. The verb part sits in position 2; the **voorvoegsel** (prefix) jumps to the end. After this lesson you can use **meenemen**, **uitgaan**, **afspreken**, and dozens of other everyday verbs with confidence.

## Grammar and Language Explanation

### What is a separable verb?

Many Dutch verbs are built from a **prefix** + **base verb**. In a **main clause**, the prefix **separates** and moves to the **end**:

| Infinitive | Prefix | Base verb | Main clause |
| --- | --- | --- | --- |
| **opstaan** | op | staan | Ik **sta** om zeven uur **op**. |
| **meenemen** | mee | nemen | Ik **neem** een paraplu **mee**. |
| **uitgaan** | uit | gaan | Wij **gaan** vanavond **uit**. |
| **afspreken** | af | spreken | Zij **spreekt** met Lisa **af**. |

| Dutch | English |
| --- | --- |
| **Opstaan** = to get up. | Prefix **op** + **staan**. |
| In the sentence: **sta** (position 2) … **op** (end). | I get up at seven. |
| **Meenemen** = to take along. | **neem** … **mee**. |
| **Uitgaan** = to go out. | **ga** … **uit**. |

Think of separable verbs as **two pieces** that bookend the sentence: **sta** at the front, **op** at the back.

  For inseparable verbs, word order in subordinate clauses, and a full prefix list, see [Separable Verbs: Prefixes and Word Order](/notes/grammar/15-separable-verbs) in the Grammar Track.
</Callout>

### Separable verbs at a glance

| Rule | What happens | Example |
| --- | --- | --- |
| **Main clause** | Prefix at the **end** | Ik **sta** vroeg **op**. |
| **Question** | Conjugated verb first, prefix at end | **Sta** je vroeg **op**? |
| **Inversion** | Verb still position 2, prefix at end | **Vandaag** sta ik laat **op**. |
| **Modal + infinitive** | Prefix **stays on** the infinitive | Ik **wil** vroeg **opstaan**. |
| **With niet** | **Niet** before the prefix | Ik sta **niet** op. |
| **Reflexive** | Reflexive between verb and prefix | Ik kleed **me** aan. |

| Dutch | English |
| --- | --- |
| Ik **neem** mijn tas **mee**. | I take my bag along. |
| **Neem** je de trein **mee**? | Are you taking the train? (as in: coming along by train) |
| Ik **wil** vanavond **uitgaan**. | I want to go out tonight. |
| Wij **gaan** **niet** uit. | We are not going out. |
| **Morgen** kom ik laat **thuis**. | Tomorrow I come home late. |

### The V2 slot and the end slot

From [Lesson 2](/notes/phase-1-a0-a1/a1/02-word-order-svo), Dutch has **verb second** in main clauses. Separable verbs use **two slots**:

| Slot | What goes there | Example |
| --- | --- | --- |
| **Position 2** | Conjugated verb stem | **sta**, **neem**, **ga** |
| **End of sentence** | Separable prefix | **op**, **mee**, **uit** |
| **Middle** | Objects, time, place, **niet**, reflexive | om zeven uur / **me** / **niet** |

| Dutch | Breakdown | English |
| --- | --- | --- |
| Ik **sta** om zeven uur **op**. | sta (pos. 2) + time + **op** (end) | I get up at seven. |
| Ik **kleed** **me** 's ochtends **aan**. | kleed + **me** + **aan** | I get dressed in the morning. |
| Ik **stap** in Amsterdam **over**. | stap + place + **over** | I transfer in Amsterdam. |
| **Vandaag** **ga** ik **niet** uit. | inversion + **niet** + **uit** | Today I am not going out. |

### With modal verbs: prefix stays attached

When a **modal** (**willen**, **kunnen**, **moeten**, **mogen**) sits in position 2, the separable verb stays **one word** at the end:

| Dutch | English |
| --- | --- |
| Ik **wil** vroeg **opstaan**. | I want to get up early. |
| **Kun** je me **helpen**? … Ik **kan** niet **meekomen**. | Can you help? … I cannot come along. |
| Wij **moeten** nu **weggaan**. | We have to leave now. |
| Hij **wil** vanavond **uitgaan**. | He wants to go out tonight. |

The modal is in position 2. The full infinitive (**opstaan**, **meekomen**, **weggaan**) closes the sentence.

### Past tense with separable verbs (preview)

In the simple past, the conjugated part changes; the prefix still goes to the end:

| Present | Past | English |
| --- | --- | --- |
| Ik **sta** om zeven uur **op**. | Ik **stond** om zeven uur **op**. | I got up at seven. |
| Wij **nemen** de tas **mee**. | Wij **namen** de tas **mee**. | We took the bag along. |
| Hij **gaat** **uit**. | Hij **ging** **uit**. | He went out. |

You learn more past forms in [Lesson 18](/notes/phase-1-a0-a1/a1/18-simple-past-regular) and [Lesson 19](/notes/phase-1-a0-a1/a1/19-simple-past-irregular). The **split** stays the same.

### Inseparable look alikes (warning)

Not every verb with **op**, **aan**, or **uit** is separable. Some prefixes are **inseparable** and never move:

| Separable ✅ | Inseparable ❌ | English |
| --- | --- | --- |
| **op**staan | **onder**nemen | get up / undertake |
| **uit**gaan | **be**zoeken | go out / visit |
| **aan**zetten | **ver**tellen | turn on / tell |

At A1, learn common separable verbs **as chunks**. If you know **opstaan** splits, do not assume **ondernemen** does too.

### Reflexive + separable together

From [Lesson 20](/notes/phase-1-a0-a1/a1/20-reflexive-verbs), some verbs are **both** reflexive and separable:

| Dutch | English |
| --- | --- |
| Ik **kleed** **me** **aan**. | I get dressed. |
| Ik **kleed** **me** **uit**. | I get undressed. |
| Ik **schrijf** **me** **in**. | I register. |
| Ik **was** **af**. | I wash up. (dishes) |

Order: **conjugated verb** + **reflexive** + … middle … + **prefix** at end.

<Callout title="Pronunciation Focus" type="info">
  In fast speech, the prefix at the end is light: _ik-STA-op_, _ik-NEEM-mee_. Stress the verb stem, not the prefix. Say **Ik sta om zeven uur op** as one rhythm. See [Sentence Intonation](/notes/pronunciation/04-intonation-and-rhythm).
</Callout>

## Common Mistakes for English Speakers

<ShadCnCard className="my-4">
  <CardHeader>
    <CardTitle className="text-lg">Keeping the prefix next to the verb</CardTitle>
  </CardHeader>

    ✅ Ik **sta** om zeven uur **op**. / Ik **ga** vanavond **uit**.

    English keeps _get up_ together. Dutch puts **op** and **uit** at the **end** of the main clause.
  </CardContent>

<ShadCnCard className="my-4">
  <CardHeader>
    <CardTitle className="text-lg">Splitting with modal verbs</CardTitle>
  </CardHeader>

  <CardContent>
    ❌ Ik wil **om zeven uur op** **staan**. / Ik kan **mee** **komen**.

    ✅ Ik wil om zeven uur **opstaan**. / Ik kan **meekomen**.

    With **willen**, **kunnen**, **moeten**, the infinitive stays **one word** at the end: **opstaan**, **meekomen**, **uitgaan**.
  </CardContent>
</ShadCnCard>

<ShadCnCard className="my-4">
  <CardHeader>
    <CardTitle className="text-lg">Niet after the prefix</CardTitle>
  </CardHeader>

  <CardContent>
    ❌ Ik sta op **niet**.

    ✅ Ik sta **niet** op.

    **Niet** comes **before** the separable prefix at the end. Same rule as [Lesson 3](/notes/phase-1-a0-a1/a1/03-negation).
  </CardContent>
</ShadCnCard>

<ShadCnCard className="my-4">
  <CardHeader>
    <CardTitle className="text-lg">Wrong reflexive position</CardTitle>
  </CardHeader>

  <CardContent>
    ❌ Ik **me** kleed aan. / Ik kleed aan **me**.

    ✅ Ik kleed **me** **aan**.

    Reflexive **me** sits **after** the conjugated verb, **before** the prefix at the end.
  </CardContent>
</ShadCnCard>
