## Introduction

The Dutch talk about **het weer** constantly. It is the safest small talk topic in the Netherlands: **Het regent weer**, **Lekker weer vandaag**, or **Wat koud, hè?** After this lesson you can describe rain, sun, and wind, ask **Hoe is het weer?**, and combine weather with seasons you met in [A0 Lesson 8](/notes/phase-1-a0-a1/a0/08-days-months-and-dates).

## Grammar and Language Explanation

### Seasons review

You learned the four seasons in A0. Use **in** + season:

| Dutch | English |
| --- | --- |
| de lente | spring |
| de zomer | summer |
| de herfst | autumn / fall |
| de winter | winter |
| het seizoen | season |

| Dutch | English |
| --- | --- |
| **In de zomer** is het warm. | In summer it is warm. |
| **In de winter** sneeuwt het. | In winter it snows. |
| **In de lente** regent het vaak. | In spring it often rains. |
| **In de herfst** is het windig. | In autumn it is windy. |

### het weer and basic questions

| Dutch | English |
| --- | --- |
| het weer | the weather |
| Hoe is het weer? | How is the weather? |
| Wat voor weer is het? | What kind of weather is it? |
| Lekker weer! | Nice weather! |
| Slecht weer. | Bad weather. |

| Dutch | English |
| --- | --- |
| **Hoe is het weer** vandaag? | How is the weather today? |
| **Wat voor weer** is het morgen? | What will the weather be like tomorrow? |
| Het is **lekker weer**! | The weather is nice! |

### Impersonal het: het is, het regent, het sneeuwt

Dutch weather often uses **het** without a real subject, like English _it rains_:

| Pattern | Example | English |
| --- | --- | --- |
| **Het is** + adjective | Het is warm. | It is warm. |
| **Het is** + number + graden | Het is tien graden. | It is ten degrees. |
| **Het** + verb (3rd sg) | Het regent. | It is raining. |
| **Het** + verb | Het sneeuwt. | It is snowing. |

| Dutch | English |
| --- | --- |
| **Het is** mooi weer. | The weather is beautiful. |
| **Het is** koud vandaag. | It is cold today. |
| **Het regent**. | It is raining. |
| **Het sneeuwt** in de winter. | It snows in winter. |
| **Het is** vijf **graden**. | It is five degrees. |

This **het** is not **the**. It is a dummy subject, like English _it_ in _it rains_.

### Weather verbs — Tier B grouped table

| Pronoun | regenen | sneeuwen | waaien | vriezen |
| --- | --- | --- | --- | --- |
| ik | regen | sneeuw | waai | vries |
| jij / je | regent | sneeuwt | waait | vriest |
| hij / zij / het | regent | sneeuwt | waait | vriest |
| wij / jullie / zij | regenen | sneeuwen | waaien | vriezen |

With weather, you usually hear the **het** form: **Het regent**, **Het sneeuwt**, **Het waait**, **Het vriest**.

| Verb | Example 1 | Example 2 |
| --- | --- | --- |
| **regenen** | **Het regent** hard. <br /> _It is raining hard._ | **Regent** het morgen? <br /> _Will it rain tomorrow?_ |
| **sneeuwen** | **Het sneeuwt** in januari. <br /> _It snows in January._ | Soms **sneeuwt** het in Nederland. <br /> _Sometimes it snows in the Netherlands._ |
| **waaien** | **Het waait** vandaag. <br /> _It is windy today._ | **Waait** het veel? <br /> _Is it very windy?_ |
| **vriezen** | **Het vriest** 's nachts. <br /> _It freezes at night._ | **Vriest** het op de weg? <br /> _Does it freeze on the road?_ |

### schijnen: de zon schijnt

The sun **shines**: **de zon schijnt**.

| Pronoun | schijnen |
| --- | --- |
| ik | schijn |
| jij / je | schijnt |
| hij / zij / het | schijnt |
| wij / jullie / zij | schijnen |

| Dutch | English |
| --- | --- |
| **De zon schijnt**. | The sun is shining. |
| **Schijnt** de zon vandaag? | Is the sun shining today? |
| **De zon schijnt niet**. Het is bewolkt. | The sun is not shining. It is cloudy. |

| De straat is **nat**. | The street is wet. |

### What to wear in Dutch weather

| Dutch | English |
| --- | --- |
| de paraplu | umbrella |
| de regenjas | raincoat |
| de laars | rain boot / boot |
| de sjaal | scarf |
| de handschoen | glove |

| Dutch | English |
| --- | --- |
| Neem een **paraplu** mee. | Take an umbrella with you. |
| Ik draag een **regenjas**. | I wear a raincoat. |
| **Het regent**. Waar is mijn **paraplu**? | It is raining. Where is my umbrella? |

**Draag** (to wear) is new here: ik draag, jij draagt, wij dragen.

### Combining weather with daily life

From [Lesson 1](/notes/phase-1-a0-a1/a1/01-daily-routines), [Lesson 14](/notes/phase-1-a0-a1/a1/14-hobbies), and [Lesson 15](/notes/phase-1-a0-a1/a1/15-transport):

| Dutch | English |
| --- | --- |
| **Omdat** het regent, neem ik de bus. | Because it rains, I take the bus. (preview) |
| Bij **mooi weer** fiets ik. | In nice weather I cycle. |
| **Het is te koud** om te wandelen. | It is too cold to walk. |
| **In de zomer** ga ik naar het park. | In summer I go to the park. |

At A1, simple separate sentences are fine: **Het regent. Ik neem de bus.**

### Temperature with numbers

Use **graden** (degrees) with numbers from A0:

| Dutch | English |
| --- | --- |
| Het is **nul** graden. | It is zero degrees. |
| Het is **tien** graden. | It is ten degrees. |
| Het is **min twee** graden. | It is minus two degrees. |
| De **temperatuur** is **twintig** graden. | The temperature is twenty degrees. |

<Callout title="Pronunciation Focus" type="info">
  **Regen**, **regent**, and **regenen** share that open Dutch **e**. **Graden** has a schwa on the final **en**. Say **Het regent**, **Het is koud**, **De zon schijnt** as one smooth phrase. See [Vowel Sounds](/notes/pronunciation/01-vowel-sounds).
</Callout>

## Example Conversation

<ShadCnCard className="my-4">
  <CardHeader>
    <CardTitle className="text-lg">Weerpraatje bij de bushalte</CardTitle>
  </CardHeader>

  <CardContent>
    **In Dutch:**

    > **Lisa:** Goedemorgen! **Wat voor weer**, hè?
    >
    > **Tom:** Ja, **het regent** weer. **Het is** maar **acht graden**.
    >
    > **Lisa:** **Bewolkt** en **nat**. Typisch weer voor de herfst.
    >
    > **Tom:** Precies. **In de zomer** was het **zonnig**. Nu **waait** het ook.
    >
    > **Lisa:** Neem je een **paraplu** mee?
    >
    > **Tom:** Ja. En jij?
    >
    > **Lisa:** Ik draag een **regenjas**. **Het weerbericht** zegt: **regenachtig** tot vanavond.
    >
    > **Tom:** Dan neem ik de bus. **Bij mooi weer** fiets ik.
    >
    > **Lisa:** Ja. **Lekker weer** komt wel weer, hoor.

    **In English:**

    > **Lisa:** Good morning! What weather, eh?
    >
    > **Tom:** Yes, it is raining again. It is only eight degrees.
    >
    > **Lisa:** Cloudy and wet. Typical weather for autumn.
    >
    > **Tom:** Exactly. In summer it was sunny. Now it is windy too.
    >
    > **Lisa:** Are you taking an umbrella?
    >
    > **Tom:** Yes. And you?
    >
    > **Lisa:** I wear a raincoat. The weather report says rainy until tonight.
    >
    > **Tom:** Then I take the bus. In nice weather I cycle.
    >
    > **Lisa:** Yes. Nice weather will come back, you know.
  </CardContent>
</ShadCnCard>
