## Grammar and Language Explanation

### The basic rule: adjective + er

Most Dutch comparatives add **er** to the adjective base:

| Base | Comparative | English |
| --- | --- | --- |
| groot | groter | bigger |
| klein | kleiner | smaller |
| mooi | mooier | more beautiful |
| duur | duurder | more expensive |
| goedkoop | goedkoper | cheaper |
| interessant | interessanter | more interesting |
| lang | langer | longer |

| Dutch | English |
| --- | --- |
| Utrecht is **groter** **dan** Amersfoort. | Utrecht is bigger than Amersfoort. |
| Deze jas is **duurder** **dan** die jas. | This coat is more expensive than that coat. |
| De les is **interessanter** **dan** gisteren. | The lesson is more interesting than yesterday. |
| **Vind** je Nederlands **moeilijker** **dan** Engels? | Do you find Dutch harder than English? |

**Dan** means _than_ after comparatives.

<Callout title="Grammar Focus" type="info">
  Full comparative and superlative rules live in the Grammar Track: [Comparatives and Superlatives](/notes/grammar/21-comparatives-superlatives). This lesson covers comparatives. Superlatives (**het grootst**, **het best**) come in [Lesson 7](/notes/phase-2-a1-a2/a2/07-superlatives).
</Callout>

### Spelling changes before er

| Situation | Rule | Example |
| --- | --- | --- |
| Short vowel + single consonant at end | double consonant + **er** | dun Ôćĺ **dunner**, plat Ôćĺ **platter** |
| Long vowel already in stem | add **er** only | groot Ôćĺ **groter**, duur Ôćĺ **duurder** |
| Adjective ends in **r** | add **der** | duur Ôćĺ **duurder**, proper Ôćĺ **properder** (rare) |

| Base | Comparative | Note |
| --- | --- | --- |
| groot | groter | long **oo** |
| dun | dunner | double **n** |
| plat | platter | double **t** |
| warm | warmer | no doubling |
| oud | ouder | |
| laat | later | |

| Dutch | English |
| --- | --- |
| Mijn appartement is **groter** **dan** het jouwe. | My flat is bigger than yours. |
| Het is **warmer** **dan** gisteren. | It is warmer than yesterday. |
| De weg is **dunner** **dan** ik dacht. | The road is narrower than I thought. |

### Irregular comparatives (learn by heart)

| Base | Comparative | English |
| --- | --- | --- |
| goed | **beter** | better |
| veel | **meer** | more |
| weinig | **minder** | less / fewer |
| graag | **liever** | rather / prefer |

| Dutch | English |
| --- | --- |
| Ik voel me **beter** vandaag. | I feel better today. |
| Ik heb **meer** tijd **dan** gisteren. | I have more time than yesterday. |
| Er zijn **minder** mensen **dan** verwacht. | There are fewer people than expected. |
| Ik drink **liever** thee **dan** koffie. | I prefer tea to coffee. |

**Beter**, **meer**, **minder**, and **liever** never take **er**.

### meer + adjective for long words

With longer adjectives, Dutch often uses **meer** + base adjective instead of **er**:

| With **er** | With **meer** | Both correct? |
| --- | --- | --- |
| interessanter | **meer** interessant | often both |
| moeilijker | **meer** moeilijk | **meer** common for long forms |
| **belangrijker** | **meer** belangrijk | both used |

| Dutch | English |
| --- | --- |
| Dit boek is **meer** interessant **dan** dat boek. | This book is more interesting than that book. |
| De opdracht is **moeilijker** **dan** de test. | The assignment is harder than the test. |
| Ik heb **meer** verantwoordelijkheid **dan** vorig jaar. | I have more responsibility than last year. |

At A2, if unsure with a long adjective, **meer** + base form is a safe choice.

### Comparatives before nouns: still add e

From [A1 Lesson 11](/notes/phase-1-a0-a1/a1/11-adjectives): comparative adjectives before nouns follow the same **e** rule:

| Dutch | English |
| --- | --- |
| **de grotere** stad | the bigger city |
| **een groter** huis | a bigger house (het word: no e) |
| **de duurdere** jas | the more expensive coat |
| **het betere** idee | the better idea |

| Dutch | English |
| --- | --- |
| Wij zoeken **een groter** appartement. | We are looking for a bigger flat. |
| **De goedkopere** winkel is **dichtbij**. | The cheaper shop is nearby. |
| Dat is **het betere** voorstel. | That is the better proposal. |

### Equality: net zo ÔÇŽ als and even ÔÇŽ als

| Pattern | English | Example |
| --- | --- | --- |
| **net zo** + adjective + **als** | just as ÔÇŽ as | **net zo** groot **als** |
| **even** + adjective + **als** | equally / as ÔÇŽ as | **even** duur **als** |
| **niet zo** + adjective + **als** | not as ÔÇŽ as | **niet zo** warm **als** |

| Dutch | English |
| --- | --- |
| Amsterdam is **net zo** gezellig **als** Utrecht. | Amsterdam is just as cosy as Utrecht. |
| Mijn auto is **even** duur **als** de jouwe. | My car is as expensive as yours. |
| Nederlands is **niet zo** moeilijk **als** ik dacht. | Dutch is not as difficult as I thought. |
| **Zij** is **even** lang **als** ik. | She is as tall as I am. |

Use the **base** adjective after **net zo** / **even** / **niet zo**, not the comparative form.

### Comparing verbs and nouns with meer and minder

| Dutch | English |
| --- | --- |
| Ik **werk** **meer** **dan** jij. | I work more than you. |
| Hij **eet** **minder** **dan** vroeger. | He eats less than before. |
| Er zijn **meer** studenten **dan** vorig jaar. | There are more students than last year. |
| Ik heb **minder** geld **dan** gisteren. | I have less money than yesterday. |

**Meer** and **minder** also compare quantity without an adjective.

### Comparatives in opinions

From [A2 Lesson 5](/notes/phase-2-a1-a2/a2/05-opinions):

| Dutch | English |
| --- | --- |
| Ik **vind** de **nieuwe** film **beter** **dan** de oude. | I think the new film is better than the old one. |
| Ik **denk dat** dit **plan** **goedkoper** **is**. | I think this plan is cheaper. |
| **Vind** jij **wonen** in de stad **fijner** **dan** op het platteland? | Do you find living in the city nicer than in the countryside? |

**Fijn** Ôćĺ **fijner** (comparative of pleasant / fine).

<Callout title="Pronunciation Focus" type="info">
  Comparative **er** is often a light schwa: **grot-er**, **mooi-er**. **Beter** and **meer** are stressed on the first syllable. Drill pairs: **groot / groter**, **goed / beter**, **veel / meer**. See [Connected Speech](/notes/pronunciation/05-connected-speech).
</Callout>

## Example Conversation

<ShadCnCard className="my-4">
  <CardHeader>
    <CardTitle className="text-lg">Een nieuw appartement zoeken</CardTitle>
    <CardDescription>Mila and Sam compare two flats online.</CardDescription>
  </CardHeader>

  <CardContent>
    **In Dutch:**

    > **Mila:** Kijk, twee appartementen in Rotterdam. **Welke vind jij beter**?
    >
    > **Sam:** Het eerste is **groter**, maar **duurder**. Het tweede is **kleiner en goedkoper**.
    >
    > **Mila:** Ja, maar de keuken is **veel kleiner dan** bij het eerste. **Vind jij dat niet**?
    >
    > **Sam:** **Dat klopt**. Maar het tweede is **rustiger**. De buurt is **niet zo druk als** bij het eerste.
    >
    > **Mila:** Dat is een goed punt. **Werk jij meer thuis dan** op kantoor?
    >
    > **Sam:** Ja, dus **een grotere woonkamer** is belangrijker voor mij. Jij?
    >
    > **Mila:** Ik ben vaak weg, dus **minder ruimte** is geen probleem. Ik **drink liever thee dan** koffie, maar dat is niet belangrijk voor het huis, hoor.
    >
    > **Sam:** Haha, inderdaad. Het eerste heeft een balkon. **Dat vind ik fijner dan** geen buitenruimte.
    >
    > **Mila:** **Even goed**: het tweede ligt dichterbij het station. De reis is **korter dan** naar het eerste.
    >
    > **Sam:** Dan is het tweede **misschien de betere keuze**. Wat denk jij?

    **In English:**

    > **Mila:** Look, two flats in Rotterdam. Which do you think is better?
    >
    > **Sam:** The first is bigger, but more expensive. The second is smaller and cheaper.
    >
    > **Mila:** Yes, but the kitchen is much smaller than in the first one. Do you not think so?
    >
    > **Sam:** True. But the second is quieter. The neighbourhood is not as busy as the first one.
    >
    > **Mila:** That is a good point. Do you work from home more than at the office?
    >
    > **Sam:** Yes, so a bigger living room is more important for me. You?
    >
    > **Mila:** I am often away, so less space is not a problem. I prefer tea to coffee, but that is not important for the flat.
    >
    > **Sam:** Ha, indeed. The first has a balcony. I find that nicer than no outdoor space.
    >
    > **Mila:** Equally good: the second is closer to the station. The journey is shorter than to the first.
    >
    > **Sam:** Then the second is perhaps the better choice. What do you think?
  </CardContent>
</ShadCnCard>
