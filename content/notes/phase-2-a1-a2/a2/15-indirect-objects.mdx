## Grammar and Language Explanation

### Direct object vs indirect object

| Role | Question | Example in **Ik geef Tom het boek** |
| --- | --- | --- |
| **Indirect object** (IO) | to / for whom? | **Tom** (receives) |
| **Direct object** (DO) | what? | **het boek** (thing given) |

| Dutch | English |
| --- | --- |
| Ik **geef** **Tom** **het boek**. | I give Tom the book. |
| Zij **vertelt** **ons** **het verhaal**. | She tells us the story. |
| **Stuur** je **haar** **het bericht**? | Are you sending her the message? |
| De docent **legt** **de studenten** **de regels** **uit**. | The teacher explains the rules to the students. |

The **indirect object** is the person (or entity) who **receives** something. The **direct object** is the **thing** transferred.

  Full pronoun order tables live in the Grammar Track: [Indirect Objects and Pronoun Order](/notes/grammar/27-indirect-objects-pronouns).
</Callout>

### Two nouns: indirect before direct

With **two nouns**, Dutch puts the **indirect object first**, then the **direct object**:

| Dutch | English |
| --- | --- |
| Ik **geef** **de man** **het boek**. | I give the man the book. |
| Hij **stuurt** **zijn moeder** **een brief**. | He sends his mother a letter. |
| **Vertel** je **de lerares** **het probleem**? | Are you telling the teacher the problem? |
| Wij **kopen** **onze dochter** **een cadeau**. | We buy our daughter a present. |

Pattern: **subject + verb + IO + DO**

### Two pronouns: direct before indirect

With **two pronouns**, Dutch reverses the order: **direct object pronoun first**, **indirect second**:

| Two nouns | Two pronouns |
| --- | --- |
| Ik **geef** **hem** **het boek**. | Ik **geef** **het** **hem**. |
| Zij **stuurt** **mij** **het bericht**. | Zij **stuurt** **het** **mij**. |

| Dutch | English |
| --- | --- |
| **Geef** je **het** **me**? | Will you give it to me? |
| Ik **geef** **het** **je** morgen. | I will give it to you tomorrow. |
| **Vertel** je **het** **ons**? | Will you tell us? |
| Hij **stuurt** **het** **haar** vandaag. | He is sending it to her today. |

**Memory trick:** with pronouns, the **thing** (**het**, **hem**, **ze**) comes **before** the **person** (**me**, **je**, **hem**, **haar**).

❌ **Geef** **me** **het** → ✅ **Geef** **het** **me**

### Unstressed vs stressed pronouns

| Unstressed | Stressed | Use |
| --- | --- | --- |
| me | mij | me = default after verb |
| je | jou | jou = emphasis or after preposition |
| hem | hem | same form |
| haar | haar | same form |
| ons | ons | same form |

| Dutch | English |
| --- | --- |
| **Geef** **het** **me**. | Give it to me. |
| **Geef** **het** **aan mij**, niet aan hem. | Give it to **me**, not to him. |
| Ik **geef** **het** **je**. | I give it to you. |
| Dit is **voor jou**, niet voor haar. | This is for **you**, not for her. |

After **aan** / **voor**, use stressed forms: **aan mij**, **aan jou**, **voor haar**.

### aan + person (alternative pattern)

Dutch often uses **aan** + person instead of a bare indirect object:

| Without **aan** | With **aan** |
| --- | --- |
| Ik **geef** **hem** **het boek**. | Ik **geef** **het boek aan** **hem**. |
| **Vertel** je **ons** **het**? | **Vertel** je **het aan** **ons**? |

| Dutch | English |
| --- | --- |
| Ik **schrijf** **een e-mail aan** **de klant**. | I write an email to the client. |
| **Stuur** je **de foto aan** **mij**? | Will you send the photo to me? |
| Het **bericht** is **voor jou**. | The message is for you. |

Both patterns are correct. **Aan** makes the recipient very clear.

### Common verbs with indirect objects

| Verb | Pattern | Example |
| --- | --- | --- |
| **geven** | iemand iets | **Geef** **me** **de sleutel**. |
| **vertellen** | iemand iets | **Vertel** **haar** **het nieuws**. |
| **vragen** | iemand iets | Ik **vraag** **hem** **een** **favor**. |
| **sturen** | iemand iets | **Stuur** **ons** **het** **document**. |
| **brengen** | iemand iets | **Breng** **je** **hem** **koffie**? |
| **kopen** | iemand iets | Ik **koop** **jou** **een** **cadeau**. |
| **bellen** | iemand | Ik **bel** **je** **vanavond**. |
| **helpen** | iemand | **Help** **je** **me**? |

| Dutch | English |
| --- | --- |
| **Kun** je **me** **het adres** **geven**? | Can you give me the address? |
| Ik **vertel** **het** **je** **later**. | I will tell you later. |
| **Vraag** **hem** **om** **hulp**. | Ask him for help. |
| **Stuur** **het** **me** **alsjeblieft** **door**. | Please forward it to me. |

### hen, hun, and ze for they/them

| Form | Role | Example |
| --- | --- | --- |
| **hen** | direct object (them) | Ik **zie** **hen**. |
| **hun** | indirect object (to them) | Ik **geef** **hun** **het boek**. |
| **ze** | both (spoken) | Ik **geef** **het** **ze**. |

| Dutch | English |
| --- | --- |
| Ik **ken** **hen**, **maar** ik **geef** **hun** **geen** **cadeau**. | I know them, but I do not give them a present. |
| **Stuur** je **het** **ze**? | Will you send it to them? |
| In daily speech, **ze** replaces **hen** and **hun** often. | |

At A2, understand **hen** vs **hun**. In conversation, **ze** is very common.

### Questions and imperatives

| Type | Example |
| --- | --- |
| Question | **Geef** je **het** **me**? |
| Imperative | **Geef** **het** **me**! |
| Modal + objects | **Kun** je **me** **het** **geven**? |

| Dutch | English |
| --- | --- |
| **Vertel** **ons** **de** **waarheid**! | Tell us the truth! |
| **Kun** je **hem** **het** **uitleggen**? | Can you explain it to him? |
| **Mag** ik **je** **iets** **vragen**? | May I ask you something? |

In questions, the verb stays first; object pronouns follow the usual order: **het** before **me**.

### In subordinate clauses

From [A2 Lesson 14](/notes/phase-2-a1-a2/a2/14-subordinate-want-omdat): object pronouns come **before** the verb (cluster) at the end:

| Main clause | Subordinate clause |
| --- | --- |
| **Geef** **het** **me**. | … **omdat** hij **het** **me** **geeft**. |
| Ik **vertel** **het** **je**. | … **als** je **tijd** **hebt**, **vertel** ik **het** **je**. |

| Dutch | English |
| --- | --- |
| Ik **bel** je, **omdat** ik **je** **iets** **wil** **vragen**. | I am calling you because I want to ask you something. |
| **Omdat** hij **het** **me** **niet** **geeft**, **koop** ik **een** **nieuwe**. | Because he does not give it to me, I buy a new one. |

<Callout title="Pronunciation Focus" type="info">
  **Geef het me** flows as one phrase: **geef-et-me**. Do not pause between pronouns. Drill: **Kun je het me geven?** · **Vertel het ons.** See [Connected Speech](/notes/pronunciation/05-connected-speech).
</Callout>

    >
