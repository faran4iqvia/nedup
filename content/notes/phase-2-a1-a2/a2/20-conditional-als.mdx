## Introduction

You know **omdat** and **want** from [A2 Lesson 14](/notes/phase-2-a1-a2/a2/14-subordinate-want-omdat), **Zou u … kunnen?** from [A2 Lesson 19](/notes/phase-2-a1-a2/a2/19-appointments), and **zou / zouden** from [A2 Lesson 4](/notes/phase-2-a1-a2/a2/04-modals-in-depth). A2 adds **als** for **if** and real conditions: **Als ik tijd heb, kom ik**, **Als het regent, neem ik een paraplu**. After this lesson you can talk about plans that depend on something else, in natural Dutch word order.

## Grammar and Language Explanation

### als = if (real condition)

**Als** introduces a **condition**: something that may happen, and a result that follows.

| Dutch | English |
| --- | --- |
| **Als** ik tijd **heb**, **kom** ik. | If I have time, I will come. |
| **Als** het **regent**, **blijf** ik thuis. | If it rains, I stay home. |
| **Als** je honger **hebt**, **eet** je iets. | If you are hungry, eat something. |
| **Kom** **als** je **kunt**. | Come if you can. |

These are **real conditions**: the situation is possible and likely. You are not dreaming or imagining an impossible world.

### Verb-final in the als clause

Like **omdat** and **dat**, the **als** clause is **subordinate**. The conjugated verb goes to the **end**:

| Main clause first | **als** clause first |
| --- | --- |
| Ik **kom**, **als** ik tijd **heb**. | **Als** ik tijd **heb**, **kom** ik. |
| Wij **gaan** fietsen, **als** het mooi **is**. | **Als** het mooi **is**, **gaan** wij fietsen. |
| **Bel** me, **als** je **klaar** **bent**. | **Als** je **klaar** **bent**, **bel** me. |

| Pattern | Rule |
| --- | --- |
| **als** clause | **als** + … + verb **last** |
| main clause | verb in **position 2** (V2) |

When the **als** clause comes **first**, the main clause still starts with the verb: **Als ik tijd heb, kom ik.**

### als vs wanneer vs toen

| Word | Use | Example |
| --- | --- | --- |
| **als** | if / when (condition, repeated or uncertain) | **Als** ik **kan**, **help** ik. |
| **wanneer** | when (time, more certain) | **Wanneer** **kom** je? |
| **toen** | when (past, one-time) | **Toen** ik jong **was**, **woonde** ik in Utrecht. |

| Dutch | English |
| --- | --- |
| **Als** het **regent**, **ga** ik niet. | If it rains, I do not go. |
| **Wanneer** **ben** je **klaar**? | When will you be ready? |
| **Toen** hij **belde**, **was** ik **onderweg**. | When he called, I was on my way. |

**Als** can mean _when_ for **habit or general situations**: **Als ik koffie drink, word ik wakker.** Use **als** when the result depends on a condition.

### Present + present / future result

The most common A2 pattern: **als** + **present tense**, main clause + **present** or **future** meaning:

| Dutch | English |
| --- | --- |
| **Als** ik morgen **kan**, **bel** ik je. | If I can tomorrow, I will call you. |
| **Als** we **tijd** **hebben**, **gaan** we **naar** de markt. | If we have time, we go to the market. |
| **Als** je **naar** links **gaat**, **zie** je de kerk. | If you go left, you see the church. |
| **Als** het **mooi** **is**, **fietsen** we **naar** het strand. | If the weather is nice, we cycle to the beach. |

Dutch often uses **present** in both clauses for future meaning: **Als ik morgen tijd heb, kom ik** = If I have time tomorrow, I will come.

### zou in polite and softer results

From [A2 Lesson 4](/notes/phase-2-a1-a2/a2/04-modals-in-depth) and [Lesson 19](/notes/phase-2-a1-a2/a2/19-appointments): **zou** softens a statement.

| Direct | Softer with **zou** |
| --- | --- |
| **Als** je **tijd** **hebt**, **help** je me. | **Als** je tijd **hebt**, **zou** je me **kunnen** **helpen**? |
| **Als** het **kan**, **kom** ik. | **Als** het **kan**, **zou** ik **graag** **komen**. |

| Dutch | English |
| --- | --- |
| **Als** u **vragen** **hebt**, **zou** u **kunnen** **bellen**. | If you have questions, you could call. |
| **Zou** je **langskomen** **als** je **tijd** **hebt**? | Would you come by if you have time? |

At A2, read **zou** in **als** sentences as _would_ or a polite _could_. Full unreal **als** + past (**Als ik rijk was, zou ik reizen**) comes at B1.

### alleen als, pas als, zolang, mits, tenzij

| Word | Meaning | Example |
| --- | --- | --- |
| **alleen als** | only if | **Alleen als** je **belt**, **kom** ik. |
| **pas als** | only when / not until | **Pas als** ik **klaar** **ben**, **ga** ik. |
| **zolang** | as long as | **Zolang** het **droog** **is**, **blijven** we **buiten**. |
| **mits** | provided that | **Ik** **kom**, **mits** ik **op tijd** **ben**. |
| **tenzij** | unless | Ik **ga**, **tenzij** het **regent**. |

| Dutch | English |
| --- | --- |
| **Alleen als** je **vandaag** **betaalt**, **is** het **goed**. | Only if you pay today is it okay. |
| **Zolang** je **stil** **bent**, **mag** je **blijven**. | As long as you are quiet, you may stay. |
| **Ik** **help**, **mits** je **op tijd** **bent**. | I will help, provided you are on time. |
| **Tenzij** er **een** **probleem** **is**, **kom** ik. | Unless there is a problem, I will come. |

From [A2 Lesson 14](/notes/phase-2-a1-a2/a2/14-subordinate-want-omdat): **tenzij** + verb last.

### In dat geval and anders

| Dutch | English |
| --- | --- |
| **Als** het **regent**, **blijf** ik thuis. **In dat geval** **bel** ik je. | If it rains, I stay home. In that case I will call you. |
| **Anders** **kom** ik **morgen**. | Otherwise I will come tomorrow. |
| **Als** je **niet** **kunt**, **zeg** het **dan**. **Anders** **wacht** ik **te** **lang**. | If you cannot, say so. Otherwise I wait too long. |

**In dat geval** refers back to the condition you just named. **Anders** = otherwise.

### Questions with als

| Dutch | English |
| --- | --- |
| **Wat** **doe** je **als** het **regent**? | What do you do if it rains? |
| **Kom** je **als** je **tijd** **hebt**? | Will you come if you have time? |
| **Kun** je **helpen** **als** ik **bel**? | Can you help if I call? |
| **Wat** **zou** je **doen** **als** je **meer** **tijd** **had**? | What would you do if you had more time? (softer / hypothetical) |

<Callout title="Pronunciation Focus" type="info">
  **Als** is short: **als**, not **als** with English _a_. In **Als ik tijd heb, kom ik**, do not pause after **heb**. Drill three chains daily. See [Connected Speech](/notes/pronunciation/05-connected-speech).
</Callout>

  </CardContent>

    >
    > **Mila:** **Goed** **idee**. **Als** het **regent**, **stellen** we het **uit**, **toch**?
    >
    > **Sam:** **Ja**. **In dat geval** **blijven** we **thuis** **en** **borrelen** we **gezellig**.
    >
    > **Mila:** **Zolang** het **droog** **is**, **gaan** we **naar** het **strand**. **Ik** **ga** **uit** **van** **goed** **weer**.
    >
    > **Sam:** **Wat** **als** ik **zaterdag** **moet** **werken**?
    >
    > **Mila:** **Dan** **gaan** we **zondag**. **Alleen als** je **vrij** **bent**, **hoor**. **Anders** **wachten** we **een** **week**.
    >
    > **Sam:** **Prima**. **Als** ik **vrij** **ben**, **bel** ik je **zodra** ik **het** **weet**.
    >
    > **Mila:** **Top**. **Mits** je **niet** **te** **laat** **belt**, **hoor**!

    **In English:**

    > **Sam:** Mila, what are we doing this weekend? If the weather is nice, I would like to cycle.
    >
    > **Mila:** Good idea. If it rains, we postpone it, right?
    >
    > **Sam:** Yes. In that case we stay home and have a cosy drink.
    >
    > **Mila:** As long as it is dry, we go to the beach. I am assuming good weather.
    >
    > **Sam:** What if I have to work on Saturday?
    >
    > **Mila:** Then we go Sunday. Only if you are free, though. Otherwise we wait a week.
    >
    > **Sam:** Fine. If I am free, I will call you as soon as I know.
    >
    > **Mila:** Great. Provided you do not call too late!
  </CardContent>
