## Introduction

This Grammar Track lesson gives the full reference for **Demonstratives**. Main lessons introduce these patterns in context; use this file when you want tables, extra examples, and focused drills.

## Explanation

| | near | far |
| --- | --- | --- |
| **de** word | deze | die |
| **het** word | dit | dat |
| plural (always de) | deze | die |

| Dutch | English |
| --- | --- |
| **Deze** jas is mooi. | This coat is beautiful. |
| **Dat** huis is groot. | That house is big. |
| **Die** schoenen zijn duur. | Those shoes are expensive. |
| **Dit** brood is vers. | This bread is fresh. |

  </CardHeader>
