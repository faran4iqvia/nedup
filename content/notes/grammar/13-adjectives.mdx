## Explanation

Before a noun, adjectives usually add **-e** except: **een + het word** and **geen + het word**.

| Context | Form | Example |
| --- | --- | --- |
| de + adj + noun | -e | de grote stad |
| het + adj + noun | -e | het kleine huis |
| een + de noun | -e | een grote stad |
| een + het noun | no -e | een groot huis |
| after zijn | base | De stad is groot |

| Dutch | English |
| --- | --- |
| **De mooie** bloemen | the beautiful flowers |
| **Een nieuw** horloge | a new watch |
| Het huis is **groot**. | The house is big. |

### Stacking adjectives (A2)

Two or more adjectives before one noun: each takes the correct ending. Use commas in writing.

| Example | Note |
| --- | --- |
| **een lange, donkere** man | both adjectives take **e** |
| **een vriendelijke, jonge** lerares | order flexible; size/age before character is natural |
| **de kleine, blauwe** ogen | colour + size |

**Met** adds a feature without stacking: **een man met een baard**, **een vrouw met krullend haar**.

| --- | --- |
