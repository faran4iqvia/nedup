## Introduction

**Om … te** + infinitive expresses purpose (_in order to_). Main lesson: [A2 Lesson 13](/notes/a2/13-om-te).

## Explanation

### Basic pattern

**main clause + om te + infinitive**

Ik ga naar school **om te leren**.

### om (time) vs om te (purpose)

| Type | Dutch | English |
| --- | --- | --- |
| Time | **Om** acht uur | at eight o'clock |
| Purpose | **om te** werken | in order to work |

### te required

❌ om werken → ✅ **om te** werken

### Separable verbs

**te** between prefix and stem:

| Infinitive | om te form | English |
| --- | --- | --- |
| opstaan | om op **te** staan | in order to get up |
| meekomen | om mee **te** komen | in order to come along |
| uitgaan | om uit **te** gaan | in order to go out |

### te + adjective + om te

**te** moe **om te** gaan · **te** laat **om te** bellen

### iets / niets + om te

**iets om te eten** · **niets om te doen**

### Purpose vs reason

| Purpose | Reason |
| --- | --- |
| **om te** + infinitive | **omdat** + clause (L14) |
| om brood te kopen | omdat ik brood nodig heb |

  </CardHeader>

  </CardContent>

## Quick Reference

| Dutch pattern | English |
| --- | --- |
| Purpose | in order to work · **om te werken** |
| Too … to | too tired to go · **te moe om te gaan** |
| Separable | in order to get up · **om op te staan** |
| Function | something to eat · **iets om te eten** |
