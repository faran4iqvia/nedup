## Introduction

The Dutch perfect tense (**voltooide tegenwoordige tijd**) uses **hebben** or **zijn** + past participle. This file gives the full rules. Main lesson: [A2 Lesson 2](/notes/a2/02-perfect-tense-intro).

## Explanation

### Formula

**Subject + hebben/zijn (conjugated) + past participle (last word)**

| Dutch | English |
| --- | --- |
| Ik **heb** gewerkt. | I have worked. |
| **Ben** je **gegaan**? | Have you gone? |
| Wij **zijn** thuis**gekomen**. | We have come home. |

### hebben vs zijn

| Use **hebben** | Use **zijn** | English |
| --- | --- | --- |
| Most verbs | Movement to a place (**gaan**, **komen**, **vertrekken**, **aankomen**) | Most actions vs movement/arrival |
| **werken**, **eten**, **drinken**, **maken** | **fietsen**, **lopen**, **reizen**, **vallen** | work, eat vs cycle, walk, travel, fall |
| | Change of state (**worden**, **zijn** → **geweest**) | become / been |

### Zwak participle: ge + stem + t/d

| Infinitive | Participle | English |
| --- | --- | --- |
| werken | gewerkt | worked |
| maken | gemaakt | made |
| wonen | gewoond | lived |
| luisteren | geluisterd | listened |

Same **'t ex kofschip** rule as simple past **te/de**.

### Separable verbs

**ge** between prefix and stem: **opgestaan**, **meegenomen**, **thuisgekomen**.

### No ge with inseparable prefixes

**be-**, **ge-**, **ver-**, **ont-**, **her-**: **vertellen** → **verteld**, not **geverteld**.

### Sterk participles (preview)

| Infinitive | Participle | English |
| --- | --- | --- |
| drinken | gedronken | drunk |
| schrijven | geschreven | written |
| doen | gedaan | done |
| zien | gezien | seen |
| gaan | gegaan | gone |
| zijn | geweest | been |

See [A2 Lesson 3](/notes/a2/03-perfect-irregular) for the full list.

## Quick Reference

| Rule | Summary |
| --- | --- |
| Formula | hebben/zijn + participle at end |
| Zwak participle | ge + stem + t/d |
| Movement | zijn + participle |
| Separable | prefix + ge + stem |
