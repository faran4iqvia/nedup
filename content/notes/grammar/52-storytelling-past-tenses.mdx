## Introduction

You know **werkte**, **ging**, and **heb gewerkt** from earlier levels. B1 storytelling mixes **onvoltooid verleden tijd** (imperfect) and **voltooid verleden tijd** (perfect) with connectors like **toen**, **op een gegeven moment**, and **kort daarna**. Main lesson: [B1 Lesson 22](/notes/b1/22-storytelling-past-tenses).

## Explanation

### Story spine

| Part | Dutch | Role |
| --- | --- | --- |
| Start | **de** **inleiding** / **het** **begin** | set time and place |
| Middle | **de** **verhaallijn** | events in order |
| Turn | **de** **wending** / **de** **plotwending** | surprise |
| End | **de** **ontknoping** / **ten** **slotte** | resolution |

### Time connectors (past stories)

| Connector | Use |
| --- | --- |
| **toen** | when (one-time past) → often imperfect |
| **eerst** … **daarna** | first … then |
| **op** **een** **gegeven** **moment** | at a certain point |
| **kort** **daarvoor** / **kort** **daarna** | shortly before / after |
| **later** **die** **avond** | later that evening |
| **ten** **slotte** | finally |

| Dutch | English |
| --- | --- |
| **Toen** ik **kwam**, **regende** **het**. | When I came, it was raining. |
| **Eerst** **werkte** ik, **daarna** **ging** ik **naar** **huis**. | First I worked, then I went home. |
| **Op** **een** **gegeven** **moment** **stopte** **de** **trein**. | At a certain point the train stopped. |

### Imperfect vs perfect in narratives

| Tense | Typical use in stories |
| --- | --- |
| **Imperfect** (**werkte**, **ging**, **was**) | scene setting, background, sequence in written stories |
| **Perfect** (**heb** **gewerkt**, **ben** **gegaan**) | spoken recap, result, completed step before next event |
| **Plusquamperfect** (**was** **gegaan**, **had** **gebeld**) | flashback: something **already** happened |

| Written story | Spoken recap |
| --- | --- |
| **Gisteren** **werkte** ik **thuis**. | **Gisteren** **heb** ik **thuis** **gewerkt**. |
| **Toen** hij **belde**, **sliep** ik. | **Toen** hij **belde**, **was** ik **al** **in** **slaap** **gevallen**. |

### Flashback pattern

**Plusquamperfect** + **toen** + imperfect:

| Dutch | English |
| --- | --- |
| **Ik** **was** **al** **vertrokken** **toen** hij **belde**. | I had already left when he called. |
| **Wij** **hadden** **net** **gegeten** **toen** **de** **stroom** **uitviel**. | We had just eaten when the power went out. |

### Narrative framing chunks

| Chunk | Example |
| --- | --- |
| **Het** **punt** **is** … | **Het** **punt** **is** **dat** **wij** **te** **laat** **waren**. |
| **In** **het** **kort** | **In** **het** **kort**: **de** **trein** **had** **vertraging**. |
| **Al** **met** **al** | **Al** **met** **al** **was** **het** **een** **leuke** **avond**. |
| **De** **moraal** **van** **het** **verhaal** | **De** **moraal** **is**: **kom** **op** **tijd**. |

## Common Mistakes for English Speakers

❌ **Toen** ik **ben** **gekomen**, **regende** **het**. (main clause perfect)

✅ **Toen** ik **kwam**, **regende** **het**.

❌ Using only perfect in a long written story where Dutch uses imperfect.

✅ Mix: imperfect for flow (**Ik liep**, **Hij lachte**), perfect for completed beats in speech (**Ik ben gaan lopen**).

## Quick Reference

| Need | Pattern |
| --- | --- |
| When (past once) | Toen + imperfect |
| Sequence | Eerst … daarna … |
| Surprise | Op een gegeven moment / plotseling |
| Flashback | had/was + participle |
| Wrap up | Ten slotte / Al met al |
