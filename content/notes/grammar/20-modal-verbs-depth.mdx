## Explanation

### Present tense modals

| Modal | ik | meaning |
| --- | --- | --- |
| willen | wil | want |
| kunnen | kan | can |
| moeten | moet | must |
| mogen | mag | may |
| zullen | zal | will / shall |

**Pattern:** modal (position 2) + … + infinitive (end).

### Past tense

| Modal | ik (past) | wij (past) |
| --- | --- | --- |
| willen | wilde / wou | wilden |
| kunnen | kon | konden |
| moeten | moest | moesten |
| mogen | mocht | mochten |
| zullen | zou | zouden |

### Double infinitive in perfect

**hebben + modal infinitive + main infinitive**

| Dutch | English |
| --- | --- |
| Ik **heb** **moeten** **werken**. | I had to work. |
| Ik **heb** **willen** **uitgaan**. | I wanted to go out. |

❌ Ik heb moeten **gewerkt**. No participle when modal infinitive is present.

### Separable verbs with modals

One word at end: **Ik wil opstaan**, **Ik heb willen opstaan**.

## Quick Reference

| Rule | Summary |
| --- | --- |
| Main clause | modal + infinitive at end |
| Perfect | hebben + modal inf + main inf |
| Separable | one word: opstaan, uitgaan |
| Permission | mogen, not kunnen |
