## Introduction

A2 introduced **worden** + participle. B1 deepens placement of **door**, modals, future, and subordinate passives. Main lesson: [B1 Lesson 8](/notes/b1/08-passive-with-worden).

## Explanation

### Core formula (review)

**subject + worden + past participle**

| Active | Passive |
| --- | --- |
| Mila **schrijft** de brief. | De brief **wordt geschreven**. |

### door + agent

| Dutch pattern | English |
| --- | --- |
| **wordt** + participle + **door** + agent | The letter is being written by Mila. |
| **wordt door** + agent + participle | The letter is being written by Mila. |

Agent is optional. Use **door**, not **van**.

### Modal + passive

**modal + worden + participle** (or **worden** + participle + modal at end in subordinates)

| Dutch | English |
| --- | --- |
| Het rapport **moet** **worden** **goedgekeurd**. | The report must be approved. |
| De data **kunnen** **worden** **geëxporteerd**. | The data can be exported. |
| Dit **mag** **niet** **worden** **gedeeld**. | This may not be shared. |

### Future passive

| Dutch | English |
| --- | --- |
| De wijziging **zal** **worden** **doorgevoerd**. | The change will be implemented. |
| De regels **zullen** **worden** **aangepast**. | The rules will be adjusted. |

### Subordinate passive

Verb cluster at the end: **participle + worden** (and modal if present).

| Main | Subordinate |
| --- | --- |
| Ik meld … | … **dat** het rapport **goedgekeurd** **moet** **worden**. |
| … **omdat** de toegang **geblokkeerd** **wordt**. | |

### Er wordt gezegd / verwacht dat

| Dutch | English |
| --- | --- |
| **Er** **wordt** **gezegd** **dat** … | It is said that … |
| **Er** **wordt** **verwacht** **dat** … | It is expected that … |

## Quick Reference

| Structure | Example |
| --- | --- |
| Present | wordt / worden + participle |
| Past | werd / werden + participle |
| Modal | moet worden goedgekeurd |
| Future | zal worden doorgevoerd |
| Agent | door het team |
| Impersonal | Er wordt gewerkt |
