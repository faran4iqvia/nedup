## Explanation

Reflexive pronouns: **me/mij**, **je/jou**, **zich**, **ons**.

| Pronoun | Reflexive |
| --- | --- |
| ik | me / mij |
| jij | je / jou |
| hij / zij | zich |
| wij | ons |

| Dutch | English |
| --- | --- |
| Ik **was me**. | I wash (myself). |
| **Kleed** je **aan**. | Get dressed. |
| Hij **scheert zich**. | He shaves. |
| Wij **haasten ons**. | We hurry. |
| **Voel** je je **beter**? | Do you feel better? |
