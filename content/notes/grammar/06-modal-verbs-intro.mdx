## Introduction

This Grammar Track lesson gives the full reference for **Modal Verbs**. Main lessons introduce these patterns in context; use this file when you want tables, extra examples, and focused drills.

## Explanation

Modals + infinitive at sentence end.

| Modal | Meaning | ik form |
| --- | --- | --- |
| willen | want | wil |
| kunnen | can | kan |
| moeten | must | moet |
| mogen | may | mag |

| Dutch | English |
| --- | --- |
| Ik **wil** koffie **drinken**. | I want to drink coffee. |
| **Kun** je me **helpen**? | Can you help me? |
| Ik **moet** werken. | I have to work. |
| **Mag** ik **zitten**? | May I sit down? |
| Wij **willen** uit **gaan**. | We want to go out. |
