## Introduction

Introduction to **willen**, **kunnen**, **mogen**, and **moeten**. Main lesson: [A1 Lesson 12: More Present Verbs](/notes/a1/12-more-present-verbs).

## Explanation

Modals + infinitive at sentence end.

| Modal | Meaning | ik form |
| --- | --- | --- |
| willen | want | wil |
| kunnen | can | kan |
| moeten | must | moet |
| mogen | may | mag |

| Dutch | English |
| --- | --- |
| Ik **wil** koffie **drinken**. | I want to drink coffee. |
| **Kun** je me **helpen**? | Can you help me? |
| Ik **moet** werken. | I have to work. |
| **Mag** ik **zitten**? | May I sit down? |
| Wij **willen** uit **gaan**. | We want to go out. |
