## Introduction

**niet** vs **geen** and where each goes. Main lesson: [A1 Lesson 3: Negation](/notes/a1/03-negation).

## Explanation

**geen** = not a / no (with nouns without de/het/dit/dat)
**niet** = not (verbs, adjectives, definite nouns)

| Dutch | English |
| --- | --- |
| Ik heb **geen** auto. | I do not have a car. |
| Ik drink **geen** melk. | I do not drink milk. |
| Ik werk **niet** vandaag. | I am not working today. |
| De winkel is **niet** open. | The shop is not open. |
| Ik koop **geen** dure schoenen. | I do not buy expensive shoes. |

## Common Mistakes for English Speakers

  </CardHeader>

  <CardContent>
    ❌ **Ik heb niet melk.** · ✅ **Ik heb geen melk.** Use **geen** for indefinite nouns (no **de/het**).
  </CardContent>
</ShadCnCard>

<ShadCnCard className="my-4">
  <CardHeader>
    <CardTitle className="text-lg">Using geen for verbs</CardTitle>
  </CardHeader>

  <CardContent>
    ❌ **Ik geen werk.** · ✅ **Ik werk niet.** Verbs and adjectives take **niet**, not **geen**.
  </CardContent>
</ShadCnCard>

<ShadCnCard className="my-4">
  <CardHeader>
    <CardTitle className="text-lg">Niet before separable prefix</CardTitle>
  </CardHeader>

  <CardContent>
    ❌ **Ik niet sta op.** · ✅ **Ik sta niet op.** **Niet** comes before the part that would move (here, before **op**).
  </CardContent>
</ShadCnCard>

## Quick Reference

| Rule | Summary |
| --- | --- |
| Core pattern | geen + indefinite noun · niet + verb/adj/definite · geen before adjective+noun |

## Answer Key

<ShadCnCard className="my-4">
  <CardContent>
    1. **Ik heb geen tijd.**
    2. **Ik drink geen koffie.** / **Ik drink niet.** (context)
    3. **geen**
    4. **Ik werk vandaag niet.**
    5. **Ik heb geen auto.** / **Ik heb niet een auto** → **geen**
    6. **Ik werk vandaag niet thuis.** / **Ik werk niet vandaag thuis.** (both possible; teach default end position)
