## Introduction

Dutch has two common ways to say _because_: **want** (two main clauses) and **omdat** (subordinate clause). Main lesson: [A2 Lesson 14](/notes/a2/14-subordinate-want-omdat).

Not sure which connector? See [Connector Choice](/notes/grammar/connector-choice).

You should be comfortable with:

- [Grammar 25: om te](/notes/grammar/25-om-te-infinitive) (do not confuse **om te** with **omdat**)

**omdat** follows the same verb-last rule as **dat** clauses.

## Explanation

### omdat: verb at the end

| Dutch | English |
| --- | --- |
| main + **omdat** … verb | I stay home **because** I am ill. · **Ik blijf thuis, omdat ik ziek ben.** |
| **omdat** … verb , main V2 | **Because** I am ill, I stay home. · **Omdat ik ziek ben, blijf ik thuis.** |

### want: V2 in both clauses

| Dutch | English |
| --- | --- |
| Ik **blijf** thuis, **want** ik **ben** ziek. | I stay home **because** I am ill. (informal, two main clauses) |

### Comparison

| | **want** | **omdat** |
| --- | --- | --- |
| Clauses | main + main | main + subordinate |
| Verb | V2 + V2 | V2 + verb last |
| Use | spoken, quick reason | neutral, full clause |
| English feel | _because_ (two sentences) | _because_ (one complex sentence) |

### Related words

| Word | Type | Dutch example | English |
| --- | --- | --- | --- |
| **doordat** | subordinate | **Doordat** het regende, bleven we thuis. | **Because** it rained, we stayed home. |
| **aangezien** | subordinate | **Aangezien** je druk bent, bel ik later. | **Since** you are busy, I will call later. |
| **daarom** | main (therefore) | **Daarom** blijf ik thuis. | **Therefore** I stay home. |

### Not purpose

| Dutch | English |
| --- | --- |
| **om te** + infinitive | purpose (_in order to_) · **Ik ga naar school om te leren.** |
| **omdat** + clause | reason (_because_) · **Ik blijf thuis omdat ik ziek ben.** |

### With modals / perfect

| Dutch | English |
| --- | --- |
| **omdat** ik **moet** werken | because I have to work |
| **omdat** hij **heeft** gebeld | because he has called |

## Quick Reference

| Meaning | Connector | Verb |
| --- | --- | --- |
| because (informal) | want | V2 + V2 |
| because (subordinate) | omdat | V2 + end |
| therefore | daarom | V2 |
| since | aangezien | V2 + end |

## Answer Key

<ShadCnCard className="my-4">
  <CardContent>
    **Part A**

    **1. want → omdat (samples)**

    - **Ik blijf thuis, want ik ben ziek.** → **Ik blijf thuis, omdat ik ziek ben.**

    - **Hij eet thuis, want hij heeft geen tijd.** → **Hij eet thuis, omdat hij geen tijd heeft.**
      **2. Waarom? with Omdat … (samples)**

    - **Waarom blijf je thuis?** → **Omdat ik ziek ben.**

    - **Waarom kom je niet?** → **Omdat ik moet werken.**
      **3. Verb position fixes (samples)**

    - ❌ **… omdat ik ben ziek** → ✅ **… omdat ik ziek ben**

    - ❌ **Omdat ik ga morgen werken, …** → ✅ **Omdat ik morgen ga werken, blijf ik thuis.**

    - ❌ **… want ik ziek ben** → ✅ **… want ik ben ziek** (V2 with **want**)
      **4. om te vs omdat (samples)**

    - Purpose: **Ik ga naar de winkel om brood te kopen.** (**om te** + infinitive)

    - Reason: **Ik ga naar de winkel, omdat ik brood nodig heb.** (**omdat** + verb last)

    **5. Paragraph (sample)**

    - **Het regent hard. Daarom blijf ik binnen. Ik kan niet fietsen, doordat de weg glad is.**
