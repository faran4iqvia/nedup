## Introduction

Dutch has two common ways to say _because_: **want** (two main clauses) and **omdat** (subordinate clause). Main lesson: [A2 Lesson 14](/notes/a2/14-subordinate-want-omdat).

## Explanation

### omdat — verb at the end

| Pattern | Example |
| --- | --- |
| main + **omdat** … verb | Ik blijf thuis, **omdat** ik ziek **ben**. |
| **omdat** … verb , main V2 | **Omdat** ik ziek **ben**, **blijf** ik thuis. |

### want — V2 in both clauses

Ik **blijf** thuis, **want** ik **ben** ziek.

### Comparison

| | **want** | **omdat** |
| --- | --- | --- |
| Clauses | main + main | main + subordinate |
| Verb | V2 + V2 | V2 + verb last |
| Use | spoken, quick reason | neutral, full clause |

### Related words

| Word | Type | Example |
| --- | --- | --- |
| **doordat** | subordinate | **Doordat** het regende, … |
| **aangezien** | subordinate | **Aangezien** je druk bent, … |
| **daarom** | main (therefore) | **Daarom** blijf ik thuis. |

### Not purpose

**om te** + infinitive = purpose · **omdat** + clause = reason

### With modals / perfect

**omdat** ik **moet** werken · **omdat** hij **heeft** gehaald

## Quick Reference

| Meaning | Connector | Verb |
| --- | --- | --- |
| because (informal) | want | V2 + V2 |
| because (subordinate) | omdat | V2 + end |
| therefore | daarom | V2 |
| since | aangezien | V2 + end |
