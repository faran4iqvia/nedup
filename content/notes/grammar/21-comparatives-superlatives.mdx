## Introduction

Dutch comparatives add **er** (or use **meer**). Superlatives add **st** or use **meest**. Main lessons: [A2 Lesson 6](/notes/a2/06-comparatives) and [A2 Lesson 7](/notes/a2/07-superlatives).

## Explanation

### Regular comparative: + er

| Base | Comparative |
| --- | --- |
| groot | groter |
| mooi | mooier |
| duur | duurder |

### Irregular comparatives

| Base | Comparative |
| --- | --- |
| goed | beter |
| veel | meer |
| weinig | minder |
| graag | liever |

### meer + adjective

Long adjectives: **meer interessant**, **meer verantwoordelijk**.

### Equality

**net zo** + base adjective + **als** · **even** + base + **als** · **niet zo** + base + **als**

### dan = than

**Groter dan**, **beter dan**, **meer dan**.

### Comparative before nouns

**de grotere stad**, **een groter huis** (no **e** with een + het).

### Regular superlative: + st

| Base | Superlative (predicate) | Before noun |
| --- | --- | --- |
| groot | het grootst | de grootste |
| mooi | het mooist | de mooiste |
| duur | het duurst | de duurste |

Add **het** in the predicate: **Dit is het grootst**.

### Irregular superlatives

| Base | Comparative | Superlative |
| --- | --- | --- |
| goed | beter | **best** / **de beste** |
| veel | meer | **meest** / **de meeste** |
| weinig | minder | **minst** / **de minste** |
| graag | liever | **liefst** / **het liefst** |

### meest + adjective

Long adjectives: **het meest interessant**, **de meest interessante film**.

### Superlative before nouns

**de grootste stad**, **het mooiste huis** (add **e** with het noun).

### van alle / in

**De mooiste van alle** · **het hoogst in Nederland** · **het allermooiste** (intensified).

### de meeste vs het meest

**de meeste** mensen (quantity) · **het meest** belangrijk (degree).

## Quick Reference

| Type | Pattern | Example |
| --- | --- | --- |
| Regular comparative | + er | groter |
| Regular superlative | + st | grootst / de grootste |
| Irregular | fixed form | beter / best, meer / meest |
| Equality | net zo + adj + als | net zo groot als |
| Than | dan | groter dan |
| Most (quantity) | de meeste | de meeste mensen |
| Most (degree) | het meest | het meest belangrijk |
