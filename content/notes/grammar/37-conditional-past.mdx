## Introduction

B1 Lesson 6 taught **present unreal** conditions: **Als ik rijk was, zou ik reizen**. When both the **if** clause and the result refer to the **past**, Dutch uses **plusquamperfect** in **als** and **zou** + perfect in the main clause. Main lesson: [B1 Lesson 7](/notes/b1/07-conditional-past).

### Before this page

You should be comfortable with:

- [Grammar 36: Hypothetical Conditionals](/notes/grammar/36-hypothetical-conditionals) (unreal present)
- [Grammar 18: Perfect Tense](/notes/grammar/18-perfect-tense) (**hebben/zijn** + participle)

If **Als ik rijk was, zou ik …** is shaky, read [36](/notes/grammar/36-hypothetical-conditionals) first.

## Explanation

### Three conditional types

| Type | **als** clause | Main clause |
| --- | --- | --- |
| Real (A2) | present | present / future |
| Unreal present (B1 L6) | past (was/had) | **zou** + infinitive |
| Unreal **past** (B1 L7) | **had/was** + participle | **zou** + **hebben/zijn** + participle |

### Past unreal pattern

| **als** clause (plusquamperfect) | Main clause |
| --- | --- |
| **Als** ik het **had** **geweten** | **zou** ik **zijn** **gekomen** |
| **Als** hij **had** **gebeld** | **zou** ik **thuis** **gebleven** **zijn** |
| **Als** wij **meer** **tijd** **hadden** **gehad** | **zouden** wij **hebben** **kunnen** **helpen** |

Verb **last** in **als**: **Als ik het had geweten**.

### zou + perfect in the result

| Dutch pattern | English |
| --- | --- |
| **zou** + **zijn** + participle | I would have come. · **Ik zou zijn gekomen.** |
| **zou** + **hebben** + participle | I would have called. · **Ik zou hebben gebeld.** |
| **zou** + **hebben** + modal + infinitive | I should have called. · **Ik zou hebben moeten bellen.** |

### Regret chunks

| Dutch | English |
| --- | --- |
| **Had** ik dat **maar** **geweten**! | If only I had known! |
| **Het** **spijt** me **dat** ik **niet** **ben** **gekomen**. | I regret that I did not come. |
| **Achteraf** **gezien** **zou** ik **anders** **hebben** **gehandeld**. | In hindsight I would have acted differently. |

  </CardHeader>

  </CardContent>

<ShadCnCard className="my-4">

  </CardContent>
</ShadCnCard>
