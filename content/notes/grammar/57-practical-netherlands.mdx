## Introduction

Living in the Netherlands means dealing with **de gemeente**, **het BSN**, **DigiD**, **de zorgverzekering**, and **de belastingdienst**. This file collects the main systems and phrases. Main lesson: [B1 Lesson 27](/notes/b1/27-practical-netherlands).

## Explanation

### First steps after arrival

| Step | Dutch | English |
| --- | --- | --- |
| Register | **inschrijven** **bij** **de** **gemeente** | register at the municipality |
| BSN | **het** **burgerservicenummer** | citizen service number |
| Address | **de** **adresregistratie** / **BRP** | address registration |
| Move | **de** **verhuizing** **doorgeven** | report a move |

### Healthcare

| Dutch | English |
| --- | --- |
| **de** **zorgverzekering** | health insurance |
| **de** **basisverzekering** | basic insurance (mandatory) |
| **de** **premie** | premium |
| **het** **eigen** **risico** | deductible |
| **de** **huisarts** | GP |

### Taxes and benefits

| Dutch | English |
| --- | --- |
| **de** **belastingdienst** | tax authority |
| **de** **belastingaangifte** | tax return |
| **de** **huurtoeslag** | rent allowance |
| **de** **zorgtoeslag** | healthcare allowance |

### Useful chunks

| Situation | Chunk |
| --- | --- |
| Appointment | **Ik** **heb** **een** **afspraak** **bij** **de** **gemeente**. |
| Insurance | **Ik** **moet** **me** **verzekeren** **voor** **1** **januari**. |
| Tax | **Ik** **doe** **mijn** **belastingaangifte** **online**. |
| GP | **Ik** **moet** **naar** **de** **huisarts**. |

## Common Mistakes for English Speakers

❌ **BSN** = tax number only (it's broader ID for all government).

✅ **BSN** for work, healthcare, taxes, and **gemeente** registration.

❌ Skipping **verhuizing doorgeven** when you move.

✅ Report within the legal deadline at **het gemeentehuis**.
