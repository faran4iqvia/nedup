## Introduction

Dutch past tense verbs belong to two families: **zwak** (weak) and **sterk** (strong). Weak verbs add **te/de**. Strong verbs change the vowel in the stem. This file gives the full classification system, vowel change patterns, and extra examples. Main lesson: [A2 Lesson 1](/notes/a2/01-past-tense-depth).

## Explanation

### Zwak (weak) verbs

| Rule | Detail |
| --- | --- |
| Signal | **te / ten** or **de / den** on the stem |
| Test | **'t ex kofschip** → voiceless ending = **te**, else **de** |
| Coverage | Most new verbs you learn |

| Infinitive | ik (past) | wij (past) |
| --- | --- | --- |
| werken | werkte | werkten |
| maken | maakte | maakten |
| wonen | woonde | woonden |
| luisteren | luisterde | luisterden |

### Sterk (strong) verbs

| Signal | Vowel change in stem |
| --- | --- |
| Singular | often one vowel pattern |
| Plural | often second vowel pattern |

| Pattern | Examples (ik → wij) |
| --- | --- |
| i → o | drink → **dronk / dronken** |
| ij → ee | schrijf → **schreef / schreven** |
| ee → a | lees → **las / lazen**; neem → **nam / namen** |
| A1 core | was/waren, had/hadden, ging/gingen, kwam/kwamen, deed/deden, at/aten |

| Dutch | English |
| --- | --- |
| Ik **dronk** koffie. | I drank coffee. |
| Wij **schreven** brieven. | We wrote letters. |
| Hij **vond** het niet. | He did not find it. |
| Zij **bleven** thuis. | They stayed home. |

### Quick decision flow

1. Is it **zijn, hebben, gaan, komen, doen, eten**? → A1 core sterk forms.
2. Does **te/de** sound natural? → zwak.
3. Otherwise → sterk; check vowel change tables.

<ShadCnCard className="my-4">

  </CardHeader>

  <CardContent>
    Follow **'t ex kofschip**: **werk → werkte**, **woon → woonde**. English -ed does not map 1:1 to Dutch **-te/-de**.
  </CardContent>
</ShadCnCard>

<ShadCnCard className="my-4">
  <CardHeader>
    <CardTitle className="text-lg">Using simple past where perfect is natural</CardTitle>
  </CardHeader>

  <CardContent>
    Spoken Dutch often prefers perfect for completed events: **Gisteren heb ik gewerkt** is very common; **Gisteren werkte ik** sounds more narrative or written.
  </CardContent>
</ShadCnCard>

<ShadCnCard className="my-4">
  <CardHeader>
    <CardTitle className="text-lg">Mixing up past and participle forms</CardTitle>
  </CardHeader>

  <CardContent>
    ❌ **Ik heb ging** · ✅ **Ik ging** (simple past) · ✅ **Ik ben gegaan** (perfect). Never combine **hebben/zijn** with simple past stem.
  </CardContent>
</ShadCnCard>

## Quick Reference

| Type | Past signal | Example |
| --- | --- | --- |
| Zwak | te/de | werkte, woonde |
| Sterk | vowel change | dronk, schreef, nam |
| A1 core sterk | fixed forms | was, had, ging |
