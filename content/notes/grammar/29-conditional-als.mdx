## Introduction

**Als** introduces **if** in real conditions. Main lesson: [A2 Lesson 20](/notes/a2/20-conditional-als).

## Explanation

### als + verb last

**Als** ik tijd **heb**, **kom** ik.

| als clause | main clause |
| --- | --- |
| verb **last** | V2 |

### Real conditions (A2)

Present + present/future result:

**Als het regent, neem ik een paraplu.**

### als vs wanneer vs toen

| Word | Use |
| --- | --- |
| **als** | if / when (condition) |
| **wanneer** | when (time) |
| **toen** | when (past, one-time) |

### zou in results

**Als je tijd hebt, zou je kunnen helpen?** (softer)

### Related conjunctions

| Word | Meaning |
| --- | --- |
| **alleen als** | only if |
| **pas als** | only when |
| **zolang** | as long as |
| **mits** | provided that |
| **tenzij** | unless |
| **zodra** | as soon as |

All take **verb-final** order (subordinate).

### in dat geval / anders

**Als het regent, blijf ik thuis. In dat geval bel ik je.**

**Anders** = otherwise.

## Quick Reference

| Pattern | Example |
| --- | --- |
| als + V-last | Als ik kan, help ik. |
| als first | Als ik tijd heb, kom ik. |
| zou result | Zou je komen als je tijd hebt? |
