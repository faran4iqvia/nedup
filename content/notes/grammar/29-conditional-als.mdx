## Introduction

**Als** introduces **if** in real conditions. Do not confuse **als** (if/when) with **toen** (when in the past) or **wanneer** (whenever) — see [Connector Choice](/notes/grammar/connector-choice). Main lesson: [A2 Lesson 20](/notes/a2/20-conditional-als).

Not sure which connector? See [Connector Choice](/notes/grammar/connector-choice).

## Explanation

### als + verb last

**Als** ik tijd **heb**, **kom** ik.

| als clause | main clause |
| --- | --- |
| verb **last** | V2 |

### Real conditions (A2)

Present + present/future result:

**Als het regent, neem ik een paraplu.**

### als vs wanneer vs toen

| Word | Use | English |
| --- | --- | --- |
| **als** | if / when (condition) | **Als** het regent, neem ik een paraplu. |
| **wanneer** | when (time) | **Wanneer** kom je? |
| **toen** | when (past, one-time) | **Toen** ik jong was, … |

### zou in results

**Als je tijd hebt, zou je kunnen helpen?** (softer)

### Related conjunctions

| Word | Meaning |
| --- | --- |
| **alleen als** | only if |
| **pas als** | only when |
| **zolang** | as long as |
| **mits** | provided that |
| **tenzij** | unless |
| **zodra** | as soon as |

All take **verb-final** order (subordinate).

### in dat geval / anders

**Als het regent, blijf ik thuis. In dat geval bel ik je.**

**Anders** = otherwise.

## Quick Reference

| Dutch pattern | English |
| --- | --- |
| als + V-last | If I can, I will help. · **Als ik kan, help ik.** |
| als first | If I have time, I will come. · **Als ik tijd heb, kom ik.** |
| zou result | Would you come if you have time? · **Zou je komen als je tijd hebt?** |
