## Introduction

B1 Lesson 10 teaches **indirect speech**: reporting what someone said without quoting. Main lesson: [B1 Lesson 10](/notes/b1/10-indirect-speech).

## Explanation

### Direct vs indirect

| Direct | Indirect |
| --- | --- |
| Mila **zegt**: "**Ik** **kom** **morgen**." | Mila **zegt** **dat** **zij** **morgen** **komt**. |
| Sam **vraagt**: "**Ben** **jij** **klaar**?" | Sam **vraagt** **of** **ik** **klaar** **ben**. |

### dat + verb last

**Hij zegt dat** + subordinate clause (verb at end).

### Tense: usually no backshift

Unlike English, Dutch often **keeps** the same tense:

| Direct | Indirect |
| --- | --- |
| "**Ik** **ben** **ziek**." | Hij **zegt** **dat** hij **ziek** **is**. |
| "**Ik** **kwam** **gisteren**." | Hij **zei** **dat** hij **gisteren** **kwam**. |

### zou for reported future

| Direct | Indirect |
| --- | --- |
| "**Ik** **zal** **komen**." | Hij **zei** **dat** hij **zou** **komen**. |
| "**Wij** **zullen** **helpen**." | Zij **zei** **dat** zij **zouden** **helpen**. |

### Yes/no questions: of

**Hij vraagt of** + statement word order (verb last).

### WH-questions

Keep the question word: **Hij vroeg waar ik woon.**

### Commands

**Hij zei dat ik moest wachten.** · **Hij vroeg me om te wachten.**

## Quick Reference

| Type | Pattern |
| --- | --- |
| Statement | **zegt dat** … verb last |
| Yes/no Q | **vraagt of** … verb last |
| WH-Q | **vraagt waar/wanneer** … |
| Reported future | **zei dat** … **zou** … |
| Command | **zei dat** ik **moest** … |
