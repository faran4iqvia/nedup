## Introduction

When a relative clause needs a **preposition** about a **thing**, formal Dutch merges **waar** + preposition: **waarop**, **waarmee**, **waarvan**. Spoken Dutch often keeps **die/dat** and puts the preposition at the end. Main lesson: [B1 Lesson 3](/notes/b1/03-waar-plus-preposition).

## Explanation

### The er rule applied to relatives

You know **erop**, **ermee**, **ervan** from everyday Dutch. In relatives, replace **er** with **waar**:

| er + prep. | waar + prep. | Example |
| --- | --- | --- |
| **erop** | **waarop** | het punt **waarop** wij **focussen** |
| **ermee** | **waarmee** | het gereedschap **waarmee** ik **werk** |
| **ervan** | **waarvan** | de reden **waarvan** ik **overtuigd** **ben** |
| **erover** | **waarover** | het onderwerp **waarover** wij **praten** |
| **ernaar** | **waarnaar** | het doel **waarnaar** wij **streven** |
| **ervoor** | **waarvoor** | de prijs **waarvoor** hij **dankbaar** **is** |
| **ertegen** | **waartegen** | het bezwaar **waartegen** hij **protesteerde** |
| **erin** | **waarin** | het jaar **waarin** ik **ben** **geboren** |
| **eraan** | **waaraan** | het project **waaraan** ik **werk** |
| **erbij** | **waarbij** | de situatie **waarbij** hulp **nodig** **is** |
| **eruit** | **waaruit** | de data **waaruit** dit **blijkt** |
| **ertoe** | **waartoe** | het doel **waartoe** dit **dient** |
| **eronder** | **waaronder** | de punten **waaronder** dit **valt** |

### Spoken vs written

| Register | Dutch pattern | English |
| --- | --- | --- |
| Spoken | **die/dat** + prep. at end | the point **that** we are talking **about** |
| Written | **waar** + prep. | the point **about which** we speak · **waarover** |

Both are correct in their register. News, reports, and formal emails prefer **waar** + preposition.

### Places: waar without a merged preposition

For **locations**, Dutch usually uses plain **waar**, not **waarin** for cities:

| Natural | Less natural |
| --- | --- |
| **De stad waar** ik **woon** | De stad **waarin** ik woon |

**Waarin** fits containers and periods: **het gebouw waarin ik werk**, **het jaar waarin ik ben geboren**.

### waarna: after which (time)

| Dutch | English |
| --- | --- |
| **De dag waarna** alles **veranderde** | The day after which everything changed |
| **Het moment waarna** wij **vertrokken** | The moment after which we left |

## Quick Reference

| Need | Form |
| --- | --- |
| Thing + preposition (formal) | **waar** + prep. |
| Person + preposition | prep. + **wie** |
| Place (no prep.) | **waar** |
| Spoken thing relative | **die/dat** + prep. at end |
