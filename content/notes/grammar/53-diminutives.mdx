## Introduction

Dutch **verkleinwoorden** add **je**, **tje**, **pje**, or **kje** to nouns: **huis** → **huisje**, **man** → **mannetje**, **straat** → **straatje**. Diminutives are **het** words and often sound warm or informal. Main lesson: [B1 Lesson 23](/notes/b1/23-diminutives).

## Explanation

### Basic endings

| Ending | When | Example |
| --- | --- | --- |
| **-je** | default after vowel / diphthong | **auto** → **autootje** |
| **-tje** | after **n**, many consonants | **bonen** → **boontje** |
| **-pje** | after **l**, **r**, **w**, long vowel + **m** | **appel** → **appeltje** |
| **-kje** | after long vowel + **ng** | **ding** → **dingetje** (variant) |

### Spelling changes

| Change | Example |
| --- | --- |
| **n** → **t** + **je** | **kraan** → **kraantje** |
| double consonant | **man** → **mannetje**, **kop** → **kopje** |
| vowel lengthening | **pad** → **paadje** |
| final **sch** / **s** | **huis** → **huisje**, **bus** → **busje** |

### Article and plural

| Rule | Example |
| --- | --- |
| always **het** | **het** **huisje** |
| plural **-s** or **-jes** | **de** **huisjes**, **de** **autootjes** |

### Affective meaning

| Nuance | Example |
| --- | --- |
| small size | **een** **klein** **huisje** |
| affection | **kom** **op**, **schattig** **kindje** |
| softening | **geen** **probleem**, **een** **klein** **foutje** |
| informal tone | **zullen** **we** **een** **biertje** **doen**? |

Not every **je** word is tiny: **het** **meisje** is the normal word for girl.

## Quick Reference

| Base ends in | Try |
| --- | --- |
| vowel | **-je** / lengthen vowel + **-je** |
| short vowel + one consonant | double + **-je** |
| **n** | **-tje** |
| **l, r, w** | **-pje** / **-tje** |
| **sch, s** | **-je** |
