## Explanation

### Function words vs content words

| Type | Examples | In fast speech |
| --- | --- | --- |
| **inhoudswoorden** | **werk**, **trein**, **morgen** | stay clear |
| **functiewoorden** | **de**, **het**, **een**, **dat** | shrink |

### Common reductions

| Written | Casual | Example |
| --- | --- | --- |
| **niet** | **nie** | **Ik kom nie.** |
| **dat** | **'t** / **da** | **'t is goed** |
| **even** | **eve** | **Wacht eve.** |
| **misschien** | **missche** | **Missche morgen.** |

### Recovery phrases

| Dutch | English |
| --- | --- |
| **Kun** **je** **dat** **herhalen**? | Can you repeat that? |
| **Wat** **zei** **je**? | What did you say? |
| **Spreek** **alsjeblieft** **langzamer**. | Please speak more slowly. |
| **Ik** **snap** **het** **niet** **helemaal**. | I do not quite get it. |

### Listening strategies

| Strategy | Action |
| --- | --- |
| **gist** | catch main topic, not every word |
| **context** | use situation to guess missing word |
| **kernwoord** | listen for stressed content words |
| **shadowen** | repeat half a beat behind speaker |
