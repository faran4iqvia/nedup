## Introduction

You know **Ik heb het koud** and **de relatie** from earlier lessons. B1 adds **richer emotions** and **relationship talk**: **verliefd**, **teleurgesteld**, **ruzie**, **verzoenen**, and **grenzen stellen**. Main lesson: [B1 Lesson 20](/notes/b1/20-feelings-and-relationships).

## Explanation

### zijn, hebben, and voelen

| Pattern | Example | English |
| --- | --- | --- |
| **zijn** + adj | **Ik** **ben** **boos**. | I am angry. |
| **hebben** + noun | **Ik** **heb** **spijt**. | I am sorry. / I regret it. |
| **zich** **voelen** + adj | **Ik** **voel** me **gespannen**. | I feel tense. |
| **worden** + adj | **Ik** **word** **boos**. | I am getting angry. |

### Common emotions

| Dutch | English |
| --- | --- |
| **blij** | happy / glad |
| **verdrietig** | sad |
| **boos** | angry |
| **teleurgesteld** | disappointed |
| **verliefd** | in love |
| **eenzaam** | lonely |
| **trots** | proud |

### Relationships

| Dutch | English |
| --- | --- |
| **de** **vriendschap** | friendship |
| **de** **liefdesrelatie** | romantic relationship |
| **vertrouwen** | trust |
| **de** **ruzie** | quarrel |
| **verzoenen** | to reconcile |
| **grenzen** **stellen** | to set boundaries |

### Talking about feelings

| Chunk | Example |
| --- | --- |
| **Ik** **maak** me **zorgen** **om** … | I worry about … |
| **Ik** **heb** **begrip** **voor** … | I understand / have sympathy for … |
| **Het** **spijt** me **dat** … | I am sorry that … |
| **Ik** **ben** **verliefd** **op** … | I am in love with … |

## Common Mistakes for English Speakers

❌ **Ik** **ben** **koud** **over** **het** **nieuws**. (emotion)

✅ **Ik** **ben** **boos** **over** **het** **nieuws**. / **Ik** **heb** **het** **koud**. (body)

❌ **Ik** **ben** **verliefd** **met** **Sam**.

✅ **Ik** **ben** **verliefd** **op** **Sam**.
