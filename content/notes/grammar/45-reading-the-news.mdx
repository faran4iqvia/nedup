## Introduction

B1 reading moves from classroom sentences to **real Dutch news**: headlines, leads, and short articles. You already know **volgens**, **melden**, and passive from earlier B1 lessons. Main lesson: [B1 Lesson 15](/notes/b1/15-reading-the-news).

## Explanation

### Headline grammar

Headlines are **short** and often drop articles:

| Headline | Full sentence feel |
| --- | --- |
| **Regering** **kondigt** **plan** **aan** | De regering kondigt een plan aan. |
| **Files** **treffen** **Nederland** | Files treffen Nederland. |
| **Minister** **ontkent** **fout** | De minister ontkent de fout. |

**Historical present:** Dutch news often uses **present tense** for past events:

| In the article | Meaning |
| --- | --- |
| **De** **minister** **zegt** **dat** … | The minister **said** that … |
| **Het** **bedrijf** **meldt** **dat** … | The company **reported** that … |

### volgens and sources

| Pattern | Example |
| --- | --- |
| **Volgens** **de** **krant** … | According to the newspaper … |
| **Volgens** **berichten** … | According to reports … |
| **Volgens** **de** **woordvoerder** … | According to the spokesperson … |

From [B1 Lesson 10](/notes/b1/10-indirect-speech): **Volgens Mila is het afgerond** + **Mila zegt dat het is afgerond**.

### Reporting patterns

| Dutch | English |
| --- | --- |
| **Er** **wordt** **gemeld** **dat** … | It is reported that … |
| **Er** **is** **bekendgemaakt** **dat** … | It has been announced that … |
| **X** **meldt** **dat** … | X reports that … |
| **X** **bevestigt** **dat** … | X confirms that … |
| **X** **ontkent** **dat** … | X denies that … |

All use **dat** + verb last in the reported clause.

### Passive in news

From [B1 Lesson 8](/notes/b1/08-passive-with-worden) and [Lesson 9](/notes/b1/09-passive-with-zijn):

| Dutch | English |
| --- | --- |
| **De** **wet** **wordt** **vandaag** **besproken**. | The law is being discussed today. |
| **Het** **rapport** **is** **gisteren** **gepubliceerd**. | The report was published yesterday. |
| **Er** **worden** **maatregelen** **genomen**. | Measures are being taken. |

### Reading order

1. **Kop** / **nieuwskop**: what happened?
2. **Eerste** **alinea** / **lead**: who, where, when?
3. **Achtergrond** and **reacties**: context and quotes
4. **Samenvatting**: can you say it in one sentence?

## Common Mistakes for English Speakers

❌ Reading **De minister zegt** as only present time.

✅ In news, **zegt** often reports a **past** statement. Check the time words.

❌ **Volgens** **de** **krant** **zegt** **dat** …

✅ **Volgens** **de** **krant** **is** **het** **plan** **goedgekeurd**. / **De** **krant** **meldt** **dat** …

**Volgens** is not a verb. It introduces a source; another verb carries the message.

❌ Translating headlines word for word with all articles.

✅ Expect **dropped** **de**/**het** and **compact** word order.

## Quick Reference

| Step | Focus |
| --- | --- |
| Headline | Short, often present tense |
| Source | Volgens + bron |
| Report | meldt / bevestigt / ontkent + dat |
| Passive | wordt / is + participle |
| Read | kop → lead → achtergrond |
