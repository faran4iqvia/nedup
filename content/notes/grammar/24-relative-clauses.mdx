## Introduction

Relative clauses add information about a noun: **de man die hier woont**. Main lesson: [A2 Lesson 12](/notes/a2/12-relative-clauses). Advanced forms (**waar**, **wie**, **hetgeen**) come in B1.

Not sure which connector? See [Connector Choice](/notes/grammar/connector-choice).

## Explanation

### die vs dat

| Antecedent | Relative pronoun | Example |
| --- | --- | --- |
| de word (sg. + pl.) | **die** | de man die …, de mensen die … |
| het word (sg.) | **dat** | het boek dat … |

### Verb to the end

| Main | Relative clause |
| --- | --- |
| Hij is aardig. | de man die aardig **is** |
| Zij woont hier. | de vrouw die hier **woont** |

Same pattern as **dat** clauses from [A2 Lesson 5](/notes/a2/05-opinions).

### die/dat required

❌ de man ik ken → ✅ **de man die** ik ken

### Not demonstratives

| Type | Example |
| --- | --- |
| Demonstrative | **Die** jas is duur. |
| Relative | De jas **die** duur is … |
| Conjunction | Ik denk **dat** hij komt. |
| Relative | Het idee **dat** hij noemde … |

### A2 preview: waar / wie

**De stad waar ik woon.** · **De collega met wie ik werk.**

## Quick Reference

| Rule | Summary |
| --- | --- |
| de noun | **die** |
| het noun | **dat** |
| Word order | subject + … + verb (last) |
| Omit pronoun? | Never in Dutch |
