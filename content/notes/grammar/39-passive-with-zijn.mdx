## Introduction

**worden** marks process; **zijn** marks result. B1 Lesson 9 completes the passive pair. Main lesson: [B1 Lesson 9](/notes/b1/09-passive-with-zijn).

Not sure which past, passive, or future form? See [Tense and Register Choice](/notes/grammar/tense-and-register-choice).

## Quick Review

From [Grammar 38: Passive with worden](/notes/grammar/38-passive-with-worden): **wordt** = in progress. **is** + participle = completed state (**is gebouwd** = is built).

  Remember from [Grammar 38](/notes/grammar/38-passive-with-worden) and [30](/notes/grammar/30-passive-voice): **wordt gebouwd** = being built · **is gebouwd** = is (already) built.
</Callout>

## Explanation

### worden vs zijn

| Type | Helper | Meaning | Dutch | English |
| --- | --- | --- | --- | --- |
| Process | **worden** | action underway | De soep **wordt** gemaakt. | The soup is being made. |
| Result | **zijn** | action done / state | De soep **is** gemaakt. | The soup has been made. |

### Spoken vs written

| Type | Register | Example | English |
| --- | --- | --- | --- |
| **worden** (process) | news, live updates | **De wet wordt aangenomen.** | The law is being passed. |
| **zijn** (result) | reports, status updates | **De wet is aangenomen.** | The law has been passed. |
| **Active** | spoken default | **Ze hebben de wet aangenomen.** | They passed the law. |

**zijn** + participle often appears in **written** status lines (**Het is afgerond**). In speech, people may say **Het is klaar** instead of a full passive.

### zijn + participle

| Tense | Example | English |
| --- | --- | --- |
| Present | Het rapport **is** **afgerond**. | The report has been completed. |
| Past | Het rapport **was** **afgerond**. | The report had been completed. |
| Plural | De data **zijn** **geëxporteerd**. | The data have been exported. |

### Er is / Er zijn + participle

| Dutch | English |
| --- | --- |
| **Er** **zijn** **fouten** **gemaakt**. | Mistakes have been made. |
| **Er** **is** **een** **besluit** **genomen**. | A decision has been taken. |

### Subordinate order

| Dutch | English |
| --- | --- |
| … **omdat** het rapport **is** **goedgekeurd**. | … because the report has been approved. |

**zijn** comes **before** the participle (unlike **worden** modal passives).

### moet zijn + participle

| Dutch | English |
| --- | --- |
| Het rapport **moet** **goedgekeurd** **zijn**. | The report must be approved (by now). |
| Alles **moet** **afgerond** **zijn** **voor** **vrijdag**. | Everything must be finished by Friday. |

## Quick Reference

| Meaning | Form |
| --- | --- |
| In progress | wordt + participle |
| Done / state | is + participle |
| Past result | was + participle |
| Impersonal | Er zijn fouten gemaakt |
