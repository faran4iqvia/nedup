## Introduction

**worden** marks process; **zijn** marks result. B1 Lesson 9 completes the passive pair. Main lesson: [B1 Lesson 9](/notes/b1/09-passive-with-zijn).

## Explanation

### worden vs zijn

| Type | Helper | Meaning | Example |
| --- | --- | --- | --- |
| Process | **worden** | action underway | De soep **wordt** gemaakt. |
| Result | **zijn** | action done / state | De soep **is** gemaakt. |

### zijn + participle

| Tense | Example |
| --- | --- |
| Present | Het rapport **is** **afgerond**. |
| Past | Het rapport **was** **afgerond**. |
| Plural | De data **zijn** **geëxporteerd**. |

### Er is / Er zijn + participle

| Dutch | English |
| --- | --- |
| **Er** **zijn** **fouten** **gemaakt**. | Mistakes have been made. |
| **Er** **is** **een** **besluit** **genomen**. | A decision has been taken. |

### Subordinate order

… **omdat** het rapport **is** **goedgekeurd**.

**zijn** comes **before** the participle (unlike **worden** modal passives).

### moet zijn + participle

| Dutch | English |
| --- | --- |
| Het rapport **moet** **goedgekeurd** **zijn**. | The report must be approved (by now). |
| Alles **moet** **afgerond** **zijn** **voor** **vrijdag**. | Everything must be finished by Friday. |

## Common Mistakes for English Speakers

❌ De soep **wordt** gemaakt. (already done, on the table)

✅ De soep **is** gemaakt.

Use **zijn** when the action is **complete**.

❌ … **dat** het rapport **goedgekeurd** **is**. (wrong order in some learner varieties)

✅ … **dat** het rapport **is** **goedgekeurd**.

## Quick Reference

| Meaning | Form |
| --- | --- |
| In progress | wordt + participle |
| Done / state | is + participle |
| Past result | was + participle |
| Impersonal | Er zijn fouten gemaakt |
