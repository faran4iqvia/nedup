## Introduction

Many Dutch mistakes are **connector choice**, not vocabulary. This hub compares pairs that English speakers mix up. Each row links to the full grammar file for drills.

## Quick Review

**want** = two main clauses (V2 both sides) · **omdat** = reason subordinate (verb last). **om te** = purpose · **omdat** = because. **als** = if (real) · **wanneer** = whenever · **toen** = when (past). **die/dat** can be demonstrative, relative, or conjunction — three different jobs.

<Callout title="Remember from…" type="info">
  Remember from [Grammar 07: Word Order](/notes/grammar/07-word-order-svo) and [26: want/omdat](/notes/grammar/26-subordinate-want-omdat): subordinate connectors push the verb to the end · **want** does not.
</Callout>

## Explanation

### want vs omdat

| | **want** | **omdat** |
| --- | --- | --- |
| Meaning | because (coordinate) | because (subordinate) |
| Word order | V2 in **both** clauses | verb **last** in **omdat** clause |
| Register | spoken, informal OK | neutral / written |
| Example | **Ik blijf thuis, want ik ben ziek.** | **Ik blijf thuis, omdat ik ziek ben.** |
| Full file | [26: want and omdat](/notes/grammar/26-subordinate-want-omdat) | same |

**Rule of thumb:** need verb at end in the reason clause? → **omdat**. Two full sentences glued? → **want**.

### om te vs omdat

| | **om te** | **omdat** |
| --- | --- | --- |
| Meaning | in order to (purpose) | because |
| Structure | **om** … **te** + infinitive | **omdat** + subordinate (verb last) |
| Example | **Ik ga naar de winkel om brood te kopen.** | **Ik ga niet, omdat ik ziek ben.** |
| Confusion | ❌ **omdat te kopen** | ❌ **om ik ziek ben** |
| Full file | [25: om te](/notes/grammar/25-om-te-infinitive) | [26: want/omdat](/notes/grammar/26-subordinate-want-omdat) |

### als vs wanneer vs toen

| Word | Use | Example | English |
| --- | --- | --- | --- |
| **als** | real **if** / general **when** | **Als ik tijd heb, kom ik.** | If I have time, I come. |
| **wanneer** | whenever / at times when | **Wanneer het regent, blijf ik thuis.** | Whenever it rains … |
| **toen** | **when** (single past moment) | **Toen ik jong was, woonde ik in Utrecht.** | When I was young … |
| Unreal if | **als** + past + **zou** | **Als ik rijk was, zou ik reizen.** | If I were rich … |
| Full files | [29: Conditional als](/notes/grammar/29-conditional-als) | [31: Extended subordinates](/notes/grammar/31-extended-subordinate-clauses) ( **toen** in stories → [52](/notes/grammar/52-storytelling-past-tenses)) |

### die / dat / dit / deze — four jobs

| Job | Form | Example | File |
| --- | --- | --- | --- |
| Demonstrative (this/that) | **deze/die**, **dit/dat** | **deze fiets**, **dat huis** | [11: Demonstratives](/notes/grammar/11-demonstratives) |
| Relative pronoun | **die** (de noun), **dat** (het noun) | **de man die hier woont** | [24](/notes/grammar/24-relative-clauses), [32](/notes/grammar/32-advanced-relative-clauses) |
| Conjunction **that** | **dat** + verb last | **Ik denk dat hij gelijk heeft.** | [26](/notes/grammar/26-subordinate-want-omdat) |
| After **alles/iets** | **wat** | **alles wat hij zegt** | [32](/notes/grammar/32-advanced-relative-clauses) |

**Test:** Can you replace with **this/that**? → demonstrative. Extra info about a noun? → relative. Introduces a whole clause after **denk/vind**? → conjunction **dat**.

### er — four jobs

| Job | Pattern | Example | File |
| --- | --- | --- | --- |
| Existential | **er is / er zijn** | **Er is een probleem.** | [28: Er](/notes/grammar/28-er-constructions) |
| Locative | **er** + **liggen/staan/…** | **Er ligt een boek op tafel.** | same |
| Partitive | verb + **er** + number | **Ik heb er drie.** | same |
| **er** + preposition | **erop**, **ermee**, … | **Ik denk erover na.** | same · relatives: [33](/notes/grammar/33-waar-plus-preposition) |

## Common Mistakes for English Speakers

    ❌ **… omdat ik ben ziek** · ✅ **… omdat ik ziek ben.** Same rule as **Ik denk dat …** — verb last.
  </CardContent>

<ShadCnCard className="my-4">
  <CardHeader>
    <CardTitle className="text-lg">toen for future if</CardTitle>
  </CardHeader>

  <CardContent>
    Real condition → **als** (**Als het morgen regent, …**). Single past moment → **toen** (**Toen ik aankwam, …**). Do not use **toen** for hypotheticals — use **als** + past + **zou** ([36](/notes/grammar/36-hypothetical-conditionals)).
  </CardContent>
</ShadCnCard>
