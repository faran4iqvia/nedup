## Introduction

Dutch **uitdrukkingen** are fixed phrases whose meaning is not literal word by word: **het maakt niet uit**, **geen zorgen**, **de draad kwijt**. B1 groups the phrases you hear every day. Main lesson: [B1 Lesson 25](/notes/b1/25-idioms-and-expressions).

## Explanation

### Types

| Term | Example |
| --- | --- |
| **de** **uitdrukking** | **het** **maakt** **niet** **uit** |
| **het** **gezegde** | **zo** **is** **het** |
| **het** **spreekwoord** | proverb (full sentence) |
| **de** **idioom** | idiomatic chunk |

### Figurative vs literal

| Type | Example | Meaning |
| --- | --- | --- |
| **letterlijk** | **Ik** **ben** **te** **laat**. | I am late. |
| **figuurlijk** | **Ik** **ben** **de** **draad** **kwijt**. | I lost the thread (confused). |

### High frequency chunks

| Dutch | English |
| --- | --- |
| **het** **maakt** **niet** **uit** | it does not matter |
| **geen** **zorgen** | no worries |
| **dat** **klopt** | that's correct |
| **het** **komt** **goed** | it will be fine |
| **het** **valt** **mee** | it's not so bad |
| **graag** **gedaan** | you're welcome |

### Classic idioms

| Dutch | English |
| --- | --- |
| **de** **handen** **uit** **de** **mouwen** **steken** | to roll up one's sleeves / get stuck in |
| **de** **kat** **uit** **de** **boom** **kijken** | to wait and see |
| **het** **ijzer** **smiden** | to strike while the iron is hot |
| **in** **een** **mum** **van** **tijd** | in no time |
