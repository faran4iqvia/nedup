## Introduction

This Grammar Track lesson gives the full reference for **Separable Verbs**. Main lessons introduce these patterns in context; use this file when you want tables, extra examples, and focused drills.

## Explanation

Prefix separates in main clauses; stays attached with infinitives and modals.

| Infinitive | Main clause | With modal |
| --- | --- | --- |
| opstaan | Ik **sta** om zeven uur **op**. | Ik **moet** **opstaan**. |
| meenemen | **Neem** je een paraplu **mee**? | Je **kunt** een paraplu **meenemen**. |
| uitgaan | Wij **gaan** vanavond **uit**. | Ik **wil** **uitgaan**. |

| Dutch | English |
| --- | --- |
| **Zet** de tv **aan**. | Turn the TV on. |
| **Bel** me **op**! | Call me! |
| Ik **kleed** me **aan**. | I get dressed. |
