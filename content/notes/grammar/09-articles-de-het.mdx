## Introduction

Every Dutch noun is **de** or **het**. There is no English equivalent, so learners often skip the article or guess wrong. This page turns A0 chunks (**de koffie**, **het brood**) into a system with guess strategies and plural rules. Main lessons: [A0 Lesson 12: _de_ and _het_ Primer](/notes/a0/12-de-het-primer) · [A1 Lesson 6: Food and Drink](/notes/a1/06-food-and-drink).

## Explanation

### One rule: noun + article = one word

| ❌ Learn this | ✅ Learn this |
| --- | --- |
| brood | **het** brood |
| fiets | **de** fiets |
| koffie | **de** koffie |

**de** = common gender (most nouns) · **het** = neuter gender · **een** = a/an (no gender on **een** itself)

| Article | Example |
| --- | --- |
| de | de stad, de fiets, de koffie |
| het | het huis, het brood, het water |

Plural nouns **always** use **de**, even if singular was **het**:

| Singular | Plural |
| --- | --- |
| **het** brood | **de** broden |
| **het** kind | **de** kinderen |
| **de** appel | **de** appels |

### Guess strategies and memory cues

These patterns are **hints**, not laws. When in doubt, learn the chunk from a dictionary.

| Cue | Usually | Examples |
| --- | --- | --- |
| People, jobs, animals | **de** | **de** man, **de** docent, **de** hond |
| Words ending in **-ing** | **de** | **de** rekening, **de** bestelling |
| Diminutives (**-je**, **-tje**, **-pje**) | **het** | **het** brood**je**, **het** meis**je** |
| Languages and metals | **het** | **het** Nederlands, **het** goud |
| Two syllables starting with **ge-** | often **het** | **het** geld, **het** gebouw |
| Infinitives used as nouns | **het** | **het** werken, **het** reizen |
| Loanwords ending in **-isme**, **-ment** | often **het** | **het** museum, **het** regime |

**Default guess:** about **75%** of nouns are **de**. If you must guess, **de** is statistically safer · but **het** words are common enough that guessing wrong still sounds off. **Always** learn high-frequency nouns with their article.

| Dutch | English |
| --- | --- |
| **De** appel is rood. | The apple is red. |
| **Het** brood is vers. | The bread is fresh. |
| Ik drink **het** water. | I drink the water. |
| **De** kinderen spelen buiten. | The children play outside. |
| **Een** broodje, alstublieft. | A roll, please. (**een** = indefinite) |

## Quick Reference

| Rule | Summary |
| --- | --- |
| Core pattern | de=most nouns · het=neuter · plural=always de |
| Best habit | learn **de/het** + noun as one chunk |
| Guess cues | -ing→de · diminutive→het · people→de · ge-→often het |
