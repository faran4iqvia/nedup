## Introduction

This Grammar Track lesson gives the full reference for **Articles**. Main lessons introduce these patterns in context; use this file when you want tables, extra examples, and focused drills.

## Explanation

**de** = common gender (most nouns)
**het** = neuter gender

| Article | Example |
| --- | --- |
| de | de stad, de fiets, de koffie |
| het | het huis, het brood, het water |

| Dutch | English |
| --- | --- |
| **De** appel is rood. | The apple is red. |
| **Het** brood is vers. | The bread is fresh. |
| Ik drink **het** water. | I drink the water. |
| **De** kinderen spelen buiten. | The children play outside. |
