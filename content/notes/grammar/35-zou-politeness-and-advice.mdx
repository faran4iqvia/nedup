## Introduction

**Zou** and **zouden** are the past forms of **zullen**, but at B1 they mainly add **distance**, **politeness**, and **soft advice**. Main lesson: [B1 Lesson 5](/notes/b1/05-zou-politeness-and-advice). Full hypothetical **als** + past comes in [B1 Lesson 6](/notes/b1/06-hypotheticals-with-als).

## Explanation

### Conjugation

| Pronoun | Form |
| --- | --- |
| ik | **zou** |
| jij / je / u / hij / zij / het | **zou** |
| wij / jullie / zij | **zouden** |

### Polite requests

| Pattern | Example |
| --- | --- |
| **Zou** je **kunnen** …? | **Zou** je me **kunnen** **helpen**? |
| **Zou** u **willen** …? | **Zou** u dit **willen** **bevestigen**? |
| **Zou** je **mogen** …? | **Zou** je even **mogen** **kijken**? |

**Zou** + second modal + infinitive at the end.

### Wishes: zou graag

| Dutch | English |
| --- | --- |
| Ik **zou** **graag** **willen** **reserveren**. | I would like to reserve. |
| Wij **zouden** **graag** **meewerken**. | We would like to collaborate. |

### Advice and warnings

| Dutch | English |
| --- | --- |
| Ik **zou** **dat** **niet** **doen**. | I would not do that. |
| **Zou** je **niet** **eerder** **beginnen**? | Would you not start earlier? |
| Ik **zou** **meer** **pauze** **nemen**. | I would take more breaks. |

### Hedged opinions

| Dutch | English |
| --- | --- |
| Dat **zou** **kunnen**. | That could be. |
| Dat **zou** **best** **kunnen**. | That could well be. |
| Ik **zou** **zeggen** **dat** … | I would say that … |

### zou in subordinate clauses

Verb at the end: **Ik vroeg of hij zou kunnen helpen**.

## Common Mistakes for English Speakers

❌ **Zou je kunnen helpen me?**

✅ **Zou je me kunnen helpen?**

Object before the modal chain when needed.

❌ Using **zal** for polite requests

✅ **Zou** je … **kunnen**?

**Zou** sounds softer than **zal**.

## Quick Reference

| Use | Pattern |
| --- | --- |
| Polite request | **Zou** je/u + modal + infinitive |
| Wish | **zou graag** + infinitive |
| Advice | Ik **zou** … |
| Hedge | Dat **zou** kunnen |
