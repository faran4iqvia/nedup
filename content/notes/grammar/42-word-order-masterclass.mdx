## Explanation

### Main clause: V2

**Subject ¬∑ Verb ¬∑ Rest** or **Fronted element ¬∑ Verb ¬∑ Subject ¬∑ Rest**

| Dutch | English |
| --- | --- |
| **Morgen** **werk** ik. | Tomorrow I work. |
| Ik **moet** morgen **werken**. | I must work tomorrow. |

### Subordinate: verb last

| Dutch | English |
| --- | --- |
| ‚Ä¶ **dat** ik morgen **moet** **werken**. | ‚Ä¶ that I must work tomorrow. |

### Cluster recipes (subordinate)

| Type | Dutch example | English |
| --- | --- | --- |
| Modal + infinitive | ‚Ä¶ **dat** ik **moet** **werken** | ‚Ä¶ that I must work |
| Perfect | ‚Ä¶ **dat** ik **heb** **gewerkt** | ‚Ä¶ that I worked |
| Modal + perfect | ‚Ä¶ **dat** ik **heb** **moeten** **werken** | ‚Ä¶ that I had to work |
| **worden** passive | ‚Ä¶ **dat** het **goedgekeurd** **moet** **worden** | ‚Ä¶ that it must be approved |
| **zijn** passive | ‚Ä¶ **dat** het **is** **goedgekeurd** | ‚Ä¶ that it has been approved |
| **te** | ‚Ä¶ **dat** het **moeilijk** **is** **te** **vinden** | ‚Ä¶ that it is hard to find |

### Pronoun order (main clause)

| Dutch | English |
| --- | --- |
| **Ik geef het hem.** | I give it to him. (DO before IO with pronouns) |

### niet placement

| Dutch | English |
| --- | --- |
| **Ik ga niet mee.** | I am not coming along. (before verb cluster / separable prefix) |

## Quick Reference

| Clause | Rule | English hint |
| --- | --- | --- |
| Main | V2 | verb in position 2 |
| Subordinate | verb(s) last | all verbs at end of clause |
| Fronted element | inversion | **Morgen ga ik** |
| Cluster | build from tables above | modal ¬∑ perfect ¬∑ passive ¬∑ **te** at end |
