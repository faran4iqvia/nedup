B1 Lesson 12 pulls Dutch word order together: V2, inversion, subordinate verb last, and verb clusters. Main lesson: [B1 Lesson 12](/notes/b1/12-word-order-masterclass).

## Explanation

### Main clause: V2

**Subject · Verb · Rest** or **Fronted element · Verb · Subject · Rest**

| Dutch | English |
| --- | --- |
| **Morgen** **werk** ik. | Tomorrow I work. |
| Ik **moet** morgen **werken**. | I must work tomorrow. |

### Subordinate: verb last

… **dat** ik morgen **moet** **werken**.

### Cluster recipes (subordinate)

| Type | Example |
| --- | --- |
| Modal + infinitive | … **dat** ik **moet** **werken** |
| Perfect | … **dat** ik **heb** **gewerkt** |
| Modal + perfect | … **dat** ik **heb** **moeten** **werken** |
| **worden** passive | … **dat** het **goedgekeurd** **moet** **worden** |
| **zijn** passive | … **dat** het **is** **goedgekeurd** |
| **te** | … **dat** het **moeilijk** **is** **te** **vinden** |

### Pronoun order (main clause)

**Ik geef het hem.** (DO before IO with pronouns)

### niet placement

Usually before the verb cluster / separable prefix: **Ik ga niet mee.**

## Common Mistakes for English Speakers

❌ **Morgen** ik **werk**.

✅ **Morgen** **werk** ik.

❌ … **dat** ik **moet** **niet** **werken**.

✅ … **dat** ik **niet** **moet** **werken**.
