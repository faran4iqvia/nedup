## Introduction

A2 taught basic **Geachte** and **Met vriendelijke groet** for appointments. B1 expands to full **zakelijke** correspondence: openings, closings, reference lines, and polite requests. Main lesson: [B1 Lesson 16](/notes/b1/16-formal-letters-and-emails).

## Explanation

### Structure of a formal message

| Part | Dutch | Role |
| --- | --- | --- |
| **Onderwerp** / **Betreft** | **Betreft:** **uw** **aanvraag** | subject line |
| **Aanhef** | **Geachte** **mevrouw** **Jansen**, | opening |
| **Body** | **Hierbij** **bevestig** ik … | message |
| **Afsluiting** | **Met** **vriendelijke** **groet**, | closing |
| **Naam** | Mila **de** **Vries** | signature |

### Openings

| Register | Dutch | Use |
| --- | --- | --- |
| Formal | **Geachte** **heer** / **mevrouw** … | unknown or official |
| Semi formal | **Beste** … | colleague you know |
| Very formal | **Geachte** **heer/mevrouw**, | letter when name unknown |
| Informal | **Hoi** / **Hey** … | friends only |

### Closings

| Register | Dutch |
| --- | --- |
| Standard formal | **Met** **vriendelijke** **groet**, |
| Very formal (letter) | **Hoogachtend**, / **Met** **hoogachting**, |
| Warm semi formal | **Hartelijke** **groet**, |
| Colleague | **Met** **collegiale** **groet**, |
| Informal | **Groetjes**, |

### Reference and reason chunks

| Dutch | English |
| --- | --- |
| **In** **verband** **met** … | In connection with … |
| **Met** **betrekking** **tot** … | With regard to … |
| **Naar** **aanleiding** **van** **uw** **schrijven** **van** … | Following your letter of … |
| **Met** **verwijzing** **naar** … | With reference to … |
| **Hierbij** … | Herewith … / hereby … |

### Polite requests in writing

From [B1 Lesson 5](/notes/b1/05-zou-politeness-and-advice):

| Dutch | English |
| --- | --- |
| **Zou** **u** … **kunnen** …? | Could you …? |
| **Graag** **verneem** ik … | I would like to hear … |
| **Ik** **zou** **het** **op** **prijs** **stellen** **als** … | I would appreciate it if … |
| **Alvast** **bedankt** **voor** … | Thanks in advance for … |

### Attachments

| Dutch | English |
| --- | --- |
| **Bijgaand** / **Bijgevoegd** vindt u … | Enclosed / attached you will find … |
| **In** **de** **bijlage** | In the attachment |
| **Zie** **bijlage** | See attachment |

## Common Mistakes for English Speakers

❌ **Geachte** **mevrouw** **Jansen** (no comma)

✅ **Geachte** **mevrouw** **Jansen**,

❌ **Met** **vriendelijke** **groeten**,

✅ **Met** **vriendelijke** **groet**, (singular is standard)

❌ **Dear** **mevrouw** (English opening)

✅ **Geachte** **mevrouw** …
