## Introduction

Dutch **partikels** are small words that change **tone**, not core grammar: **Kom maar binnen, hoor**, **Dat is toch logisch?**, **Even kijken**. You met **toch**, **hoor**, and **zeg** in earlier lessons. B1 groups the full spoken toolkit. Main lesson: [B1 Lesson 24](/notes/b1/24-particles-masterclass).

## Explanation

### What is a partikel?

| Term | Meaning |
| --- | --- |
| **het** **partikel** | particle |
| **de** **modal** **partikel** | modal particle (attitude) |
| **het** **vulwoord** | filler word |

Particles are often **unstressed**. They soften, insist, confirm, or contrast.

### Core particles

| Particle | Main feel | Example |
| --- | --- | --- |
| **hoor** | friendly, reassuring | **Kom** **binnen**, **hoor**. |
| **toch** | anyway / surely / right? | **Je** **komt** **toch** **wel**? |
| **maar** | just / go ahead (particle) | **Probeer** **het** **maar**. |
| **even** | briefly / just a sec | **Even** **wachten**. |
| **nou** | well / so | **Nou**, **wat** **vind** **je**? |
| **eens** | try / (imperative softener) | **Kijk** **eens**. |
| **wel** | indeed / emphasis | **Dat** **kan** **wel**. |
| **zeg** | surprise / emphasis | **Mooi**, **zeg**! |
| **hè** / **hé** | tag: right? / hey | **Leuk**, **hè**? |

### Common combinations

| Chunk | Feel |
| --- | --- |
| **toch** **wel** | surely / after all yes |
| **maar** **wel** | but still / definitely |
| **nou** **ja** | well yes / I suppose |
| **nou** **goed** | alright then |
| **eens** **even** | just a quick look |
| **maar** **eens** | just try |
| **ja** **hoor** | yes (reassuring) |
| **nee** **hoor** | no (soft) |

### Particle vs other word class

| Word | Conjunction / other | Particle |
| --- | --- | --- |
| **maar** | **Ik** **wil**, **maar** **ik** **kan** **niet** | **Doe** **maar** |
| **even** | **even** **duur** (equally) | **Even** **kijken** |
| **nou** | (rare noun) | **Nou**, **doei** |

## Quick Reference

| Goal | Particle |
| --- | --- |
| Friendly | hoor |
| Confirm | toch? / hè? |
| Soften imperative | maar / eens |
| Brief action | even |
| Filler | nou |
| Emphasis yes | wel / toch wel |
