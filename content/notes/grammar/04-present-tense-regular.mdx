## Introduction

This Grammar Track lesson gives the full reference for **Present Tense**. Main lessons introduce these patterns in context; use this file when you want tables, extra examples, and focused drills.

## Explanation

Regular verbs: remove **-en**, add endings.

| Person | Ending | werken example |
| --- | --- | --- |
| ik | stem | werk |
| jij / u | stem + t | werkt |
| hij / zij / het | stem + t | werkt |
| wij / jullie / zij | infinitive | werken |

**t-drop:** verb before **je** loses **t**: **Werk** je vandaag?

| Dutch | English |
| --- | --- |
| Ik **werk** in Rotterdam. | I work in Rotterdam. |
| **Maak** je koffie? | Do you make coffee? |
| Zij **spreekt** Nederlands. | She speaks Dutch. |
| Wij **wonen** in Utrecht. | We live in Utrecht. |
| Ik **loop** naar huis. | I walk home. |

    ❌ **Hij werks** · ✅ **Hij werkt.** Dutch regular verbs add **-t** for **hij/zij/het**, not English **-s**.

<ShadCnCard className="my-4">
  <CardHeader>
    <CardTitle className="text-lg">Forgetting the t in ik-form spelling</CardTitle>
  </CardHeader>

  <CardContent>
    Stem ends in **-t, -d, -f, -s, -ch, -x, -z** → add **-t** for **jij** form in writing: **jij werkt**, **jij gaat** (not **jij werk** in standard spelling).
  </CardContent>
</ShadCnCard>

<ShadCnCard className="my-4">
  <CardHeader>
    <CardTitle className="text-lg">Present for completed past</CardTitle>
  </CardHeader>

  <CardContent>
    ❌ **Gisteren ik werk thuis.** · ✅ **Gisteren werkte ik thuis.** / **Gisteren heb ik thuis gewerkt.**
  </CardContent>
</ShadCnCard>
