## Explanation

Dutch personal pronouns include stressed and unstressed forms (**jij/je**, **zij/ze**) and formal **u**.

| Pronoun | English |
| --- | --- |
| ik | I |
| jij / je | you (informal) |
| u | you (formal) |
| hij | he |
| zij / ze | she |
| het | it |
| wij / we | we |
| jullie | you (plural) |
| zij / ze | they |

| Dutch | English |
| --- | --- |
| **Ik** ben Anna. | I am Anna. |
| **Jij** bent mijn vriend. | You are my friend. |
| **Hij** is docent. | He is a teacher. |
| **Zij** woont in Utrecht. | She lives in Utrecht. |
| **Wij** zijn studenten. | We are students. |
| **Jullie** zijn op tijd. | You are on time. |
| **Zij** komen uit België. | They come from Belgium. |
