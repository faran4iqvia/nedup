## Introduction

A2 taught **Daar ben ik het mee eens** and **Dat zie ik anders**. B1 adds partial agreement, respectful disagreement, and debate structure for meetings and media. Main lesson: [B1 Lesson 14](/notes/b1/14-agreeing-and-debating).

## Explanation

### Agreement spectrum

| Dutch | English |
| --- | --- |
| **Daar** **ben** ik **het** **mee** **eens**. | I agree with that. |
| **Daar** **ben** ik **het** **deels** **mee** **eens**. | I partly agree with that. |
| **Daar** **ben** ik **het** **niet** **mee** **eens**. | I do not agree with that. |
| **Ik** **zie** **dat** **anders**. | I see that differently. |
| **Wij** **zijn** **het** **oneens**. | We disagree. |

### Soft disagreement

| Dutch | English |
| --- | --- |
| **Ik** **begrijp** je **punt**, **maar** … | I understand your point, but … |
| **Met** **alle** **respect**, **maar** … | With all due respect, but … |
| **Je** **hebt** **een** **goed** **punt**, **maar** … | You make a good point, but … |
| **Laten** **we** **het** **oneens** **zijn**. | Let's agree to disagree. |

### Debate structure

| Dutch | English |
| --- | --- |
| **Aan** **de** **ene** **kant** … **aan** **de** **andere** **kant** … | On one hand … on the other hand … |
| **Echter** … | However … |
| **Het** **tegenvoorstel** **is** **dat** … | The counterproposal is that … |
| **Ter** **verdediging** **van** … | In defence of … |

### Floor and wrap up

| Dutch | English |
| --- | --- |
| **Mag** ik **daar** **op** **reageren**? | May I respond to that? |
| **Ik** **wil** **daar** **op** **ingaan**. | I want to address that. |
| **Samenvattend** … / **Kortom** … / **Tot** **slot** … | In summary … / In short … / Finally … |

**Mee** in **het eens zijn mee** refers back to the previous statement. **Dat** in **Ik zie dat anders** is a demonstrative: _that_ (the idea), not a conjunction.

## Quick Reference

| Goal | Chunk |
| --- | --- |
| Full agree | Daar ben ik het mee eens |
| Partial | Daar ben ik het deels mee eens |
| Disagree | Daar ben ik het niet mee eens / Ik zie dat anders |
| Respectful clash | Ik begrijp je punt, maar … |
| Close debate | Laten we het oneens zijn |
| Two sides | Aan de ene kant … aan de andere kant … |
| Respond | Mag ik daarop reageren? / Ik wil daarop ingaan |
