## Introduction

This Grammar Track lesson gives the full reference for **Simple Past**. Main lessons introduce these patterns in context; use this file when you want tables, extra examples, and focused drills.

## Explanation

High-frequency irregular past forms.

| Infinitive | ik / hij | wij |
| --- | --- | --- |
| zijn | was | waren |
| hebben | had | hadden |
| gaan | ging | gingen |
| komen | kwam | kwamen |
| doen | deed | deden |
| eten | at | aten |
| kunnen | kon | konden |
| moeten | moest | moesten |
| willen | wilde / wou | wilden |

| Dutch | English |
| --- | --- |
| Ik **was** ziek. | I was ill. |
| **Ging** je naar huis? | Did you go home? |
| Wij **aten** pasta. | We ate pasta. |
| Hij **had** geen tijd. | He had no time. |
