## Introduction

B1 ends with **integration**: you combine grammar, vocabulary, pronunciation, and culture in **real situations**. This file maps the **grammar spine** from A0 through B1 and shows how **register** and **context** change your Dutch. Main lesson: [B1 Lesson 28](/notes/b1/28-bringing-it-together).

## Explanation

### Four skills at B1

| Skill | Dutch | B1 target |
| --- | --- | --- |
| Listening | **luisteren** | follow meetings, news, fast speech with strategies |
| Speaking | **spreken** | opinions, stories, formal chunks, recovery phrases |
| Reading | **lezen** | news, letters, forms, longer texts |
| Writing | **schrijven** | emails, applications, clear paragraphs |

**Receptief** (listening + reading) and **productief** (speaking + writing) reinforce each other.

### Grammar spine A0 → B1

| Level | Core patterns |
| --- | --- |
| **A0** | **zijn**, **hebben**, greetings, numbers, **de/het** |
| **A1** | V2, inversion, possessives, separables, perfect intro |
| **A2** | past tenses, modals, **omdat/want**, relatives, **er**, **als** |
| **B1** | **hoewel**, **zou**, passives, indirect speech, **te**, word order, conditionals |

### Register quick map

| Context | Tone | Examples |
| --- | --- | --- |
| Friend / colleague | informal | **Hoe gaat het?**, **Geen zorgen**, particles **hoor**, **even** |
| Work / email | neutral formal | **Met vriendelijke groet**, **Ik zou graag …** |
| Government / IND | formal | **Geachte**, **u**, **alstublieft**, fixed phrases |
| News / debate | structured | **hoewel**, **daarentegen**, passive, **volgens** |

### Integrated scenario chain

Typical B1 week might combine:

1. **E-mail** to employer (**B1 L16–17**)
2. **Meeting** opinion (**B1 L13–14**)
3. **Gemeente** visit (**B1 L27**)
4. **Story** about the week (**B1 L22**)
5. **Recovery** when speech is fast (**B1 L26**)

### Self check before the B1 test

| Area | Ask yourself |
| --- | --- |
| Word order | Can I build a **bijzin** + **hoofdzin** without freezing? |
| Past | Do I choose **imperfectum** vs **perfect** in stories? |
| Formal | Can I open and close a letter correctly? |
| Listening | Do I use **Kun je dat herhalen?** when needed? |
| Bureaucracy | Do I know **BSN**, **DigiD**, **huisarts** basics? |
