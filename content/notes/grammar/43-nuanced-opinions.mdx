## Introduction

A2 taught **Ik vind dat** and **Ik denk dat**. B1 adds nuance: soften, strengthen, and hedge. Main lesson: [B1 Lesson 13](/notes/b1/13-nuanced-opinions).

## Explanation

### Softening chunks

| Dutch | English |
| --- | --- |
| **Naar** **mijn** **mening** … | In my opinion … |
| **Het** **lijkt** me **dat** … | It seems to me that … |
| **Ik** **zou** **zeggen** **dat** … | I would say that … |
| **Het** **zou** **kunnen** **zijn** **dat** … | It could be that … |

### Uncertainty

| Dutch | English |
| --- | --- |
| **Ik** **ben** **er** **niet** **zeker** **van** **of** … | I am not sure whether … |
| **Voor** **zover** ik **weet** … | As far as I know … |
| **Misschien** … / **Waarschijnlijk** … | Maybe … / Probably … |

### Strengthening

| Dutch | English |
| --- | --- |
| **Ik** **ben** **ervan** **overtuigd** **dat** … | I am convinced that … |
| **Zonder** **twijfel** … | Without doubt … |
| **Ik** **sta** **er** **achter**. | I stand behind it. |

### Partial views

| Dutch | English |
| --- | --- |
| **Tot** **op** **zekere** **hoogte** … | To a certain extent … |
| **In** **zekere** **zin** … | In a certain sense … |
| **Ik** **heb** **gemengde** **gevoelens** **over** … | I have mixed feelings about … |

All opinion chunks take **dat** + verb last or **of** + statement order.

## Common Mistakes for English Speakers

❌ **Naar** **mijn** **mening** **is** **het** **goed** **is**.

✅ **Naar** **mijn** **mening** **is** **het** **goed**.

❌ **Het** **lijkt** me **dat** **is** **het** **moeilijk**.

✅ **Het** **lijkt** me **dat** **het** **moeilijk** **is**.

## Quick Reference

| Strength | Chunk |
| --- | --- |
| Soft | naar mijn mening, het lijkt me dat |
| Hedge | ik zou zeggen, het zou kunnen zijn |
| Unsure | ik ben er niet zeker van of |
| Strong | ervan overtuigd, zonder twijfel |
| Partial | tot op zekere hoogte, deels |
