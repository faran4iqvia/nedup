## Introduction

Dutch **er** is a small word with several jobs. Main lesson: [A2 Lesson 16](/notes/a2/16-er-constructions). Existential **er** was introduced in [A2 Lesson 9](/notes/a2/09-describing-places).

## Explanation

### Four uses at A2

| Use | Pattern | Example |
| --- | --- | --- |
| Existential | er is / er zijn | Er is een park. |
| Locative | er + liggen/staan/zitten/hangen | Er ligt een boek op tafel. |
| Partitive | verb + er + number/geen | Ik heb er drie. |
| Quantitative | hoeveel … zijn er? | Er zijn er vier. |

### Existential: er is / er zijn

| Pattern | Example |
| --- | --- |
| er is + singular | Er is een café. |
| er zijn + plural | Er zijn veel parken. |
| er is geen | Er is geen station. |
| question | Is er / Zijn er |

Add place with **in**, **op**, **aan**, **bij**: **Er is een markt in de buurt.**

### Locative: er + position verb

| Verb | Typical use |
| --- | --- |
| **liggen** | flat objects, on surfaces |
| **staan** | upright objects, people |
| **zitten** | inside, stains, seated |
| **hangen** | on walls, hooks |

**Er ligt** een sleutel **op** tafel. · **Er staat** een plant **naast** het raam.

Questions: **Ligt er …?** · **Staat er …?**

### Partitive er

Refers back to a noun: **of them / of it**.

| Pattern | Example |
| --- | --- |
| heb er + number | Ik heb er twee. |
| wil er geen | Ik wil er geen. |
| heb er genoeg | Ik heb er genoeg. |

**Er** comes **before** the number or **geen**.

### Double er

**Hoeveel stoelen zijn er? Er zijn er vier.**

| 1st **er** | 2nd **er** |
| --- | --- |
| there are | of them |

### er + preposition

| Form | Meaning |
| --- | --- |
| erop | on it |
| erin | in it |
| eronder | under it |
| ernaast | next to it |
| erachter | behind it |
| erboven | above it |
| ervoor | in front of it |

**Wat ligt erop?** · **Zit er nog iets erin?**

### Negation

**Er is geen melk.** · **Er zijn geen stoelen.** · **Ik heb er geen.**

## Quick Reference

| Question | Answer pattern |
| --- | --- |
| Is something there? | Er is / Er zijn |
| Where is it? | Er ligt/staat/zit/hangt … |
| How many? | Hoeveel … zijn er? Er zijn er … |
| Do you have any? | Ik heb er twee / Ik heb er geen |
