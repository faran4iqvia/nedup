## Explanation

### Four uses at A2

| Use | Dutch pattern | English |
| --- | --- | --- |
| Existential | er is / er zijn | there is / there are |
| Locative | er + liggen/staan/zitten/hangen | there lies / stands / sits / hangs |
| Partitive | verb + er + number/geen | … of them / of it |
| Quantitative | hoeveel … zijn er? | how many … are there? |

### Existential: er is / er zijn

| Dutch pattern | English |
| --- | --- |
| er is + singular | there is … · **Er is een café.** |
| er zijn + plural | there are … · **Er zijn veel parken.** |
| er is geen | there is no … · **Er is geen station.** |
| question | is there / are there · **Is er …?** / **Zijn er …?** |

Add place with **in**, **op**, **aan**, **bij**: **Er is een markt in de buurt.**

### Locative: er + position verb

| Verb | Typical use | English example |
| --- | --- | --- |
| **liggen** | flat objects, on surfaces | **Er ligt** a key **on** the table. |
| **staan** | upright objects, people | **Er staat** a plant **next to** the window. |
| **zitten** | inside, stains, seated | **Er zit** coffee on your shirt. |
| **hangen** | on walls, hooks | **Er hangt** a picture **on** the wall. |

**Er ligt** een sleutel **op** tafel. · **Er staat** een plant **naast** het raam.

Questions: **Ligt er …?** · **Staat er …?**

### Partitive er

Refers back to a noun: **of them / of it**.

| Dutch pattern | English |
| --- | --- |
| heb er + number | I have X of them · **Ik heb er twee.** |
| wil er geen | I want none of them · **Ik wil er geen.** |
| heb er genoeg | Ik heb er genoeg. |

**Er** comes **before** the number or **geen**.

### Double er

**Hoeveel stoelen zijn er? Er zijn er vier.**

| 1st **er** | 2nd **er** |
| --- | --- |
| there are | of them |

### er + preposition

| Form | Meaning |
| --- | --- |
| erop | on it |
| erin | in it |
| eronder | under it |
| ernaast | next to it |
| erachter | behind it |
| erboven | above it |
| ervoor | in front of it |

**Wat ligt erop?** · **Zit er nog iets erin?**

### Negation

**Er is geen melk.** · **Er zijn geen stoelen.** · **Ik heb er geen.**

## Quick Reference

| Question | Answer pattern |
| --- | --- |
| Is something there? | Er is / Er zijn |
| Where is it? | Er ligt/staat/zit/hangt … |
| How many? | Hoeveel … zijn er? Er zijn er … |
| Do you have any? | Ik heb er twee / Ik heb er geen |
