## Introduction

B1 Lesson 6 taught **present unreal** conditions: **Als ik rijk was, zou ik reizen**. When both the **if** situation and the result lie **in the past**, and the past **did not happen** that way, Dutch uses **plusquamperfect** in **als** and **zou** + perfect in the main clause: **Als ik het had geweten, zou ik zijn gekomen**, **Als wij meer tijd hadden gehad, zouden wij hebben kunnen helpen**, **Had ik dat maar geweten!** After this lesson you can express regret, hindsight, and _what would have happened if_ the way Dutch speakers do in apologies, reviews, and personal reflection.

## Quick Review

From [B1 Lesson 6](/notes/b1/06-hypotheticals-with-als): unreal **als** uses **past** (**was**, **had**, **kon**) + **zou** in the result. From [A2 Lesson 2](/notes/a2/02-perfect-tense-intro): **hebben/zijn** + past participle (**ik ben gekomen**, **ik heb gebeld**). From [B1 Lesson 5](/notes/b1/05-zou-politeness-and-advice): **zou** for softened statements. Today **als** takes **had/was** + participle and the result takes **zou** + **hebben/zijn** + participle for **past unreal**.

## Grammar and Language Explanation

### Three conditional levels

| Type | **als** clause | Main clause | Example |
| --- | --- | --- | --- |
| **Real** (A2) | present | present / future | **Als** ik tijd **heb**, **kom** ik. |
| **Unreal present** (B1 L6) | past imperfect | **zou** + infinitive | **Als** ik rijk **was**, **zou** ik **reizen**. |
| **Unreal past** (B1 L7) | **had/was** + participle | **zou** + **hebben/zijn** + participle | **Als** ik het **had** **geweten**, **zou** ik **zijn** **gekomen**. |

| Dutch | English |
| --- | --- |
| **Als** ik **het** **had** **geweten**, **zou** ik **zijn** **gekomen**. | If I had known, I would have come. |
| **Als** hij **had** **gebeld**, **zou** ik **thuis** **gebleven** **zijn**. | If he had called, I would have stayed home. |
| **Als** wij **meer** **budget** **hadden** **gehad**, **zouden** wij **hebben** **kunnen** **investeren**. | If we had had more budget, we could have invested. |

### Plusquamperfect in the als clause

The **als** clause uses **voltooid verleden tijd** (plusquamperfect): **had/was** + past participle, verb **last**.

| Auxiliary | Pattern | Example |
| --- | --- | --- |
| **hebben** | **had** + participle | **Als** ik **het** **had** **geweten** |
| **zijn** | **was** + participle | **Als** ik **eerder** **was** **gekomen** |
| plural | **hadden** / **waren** | **Als** wij **meer** **tijd** **hadden** **gehad** |

| Dutch | English |
| --- | --- |
| **Als** ik **niet** **zo** **overhaast** **had** **gehandeld**, **zou** alles **anders** **zijn** **afgelopen**. | If I had not acted so hastily, everything would have turned out differently. |
| **Als** zij **het** **verzoek** **had** **gezien**, **zou** zij **hebben** **gereageerd**. | If she had seen the request, she would have responded. |
| **Als** wij **dat** **hadden** **geweten**, **zouden** wij **een** **andere** **handelwijze** **hebben** **gekozen**. | If we had known that, we would have chosen a different course of action. |

### zou + perfect in the result clause

The main clause uses **zou/zouden** + **hebben** or **zijn** + past participle.

| Pattern | Example |
| --- | --- |
| **zou** + **zijn** + participle | Ik **zou** **zijn** **gekomen**. |
| **zou** + **hebben** + participle | Ik **zou** **hebben** **gebeld**. |
| **zou** + **hebben** + modal + infinitive | Ik **zou** **hebben** **moeten** **bellen**. |

| Dutch | English |
| --- | --- |
| **Als** ik **het** **had** **geweten**, **zou** ik **je** **hebben** **gebeld**. | If I had known, I would have called you. |
| **Als** hij **had** **kunnen** **komen**, **zou** hij **er** **zijn** **geweest**. | If he could have come, he would have been there. |
| **Als** wij **meer** **hadden** **voorbereid**, **zouden** wij **niet** **zoveel** **stress** **hebben** **gehad**. | If we had prepared more, we would not have had so much stress. |

**Zijn** + participle for movement/change (**zijn gekomen**, **zijn gebleven**). **Hebben** + participle for most other verbs (**hebben gebeld**, **hebben gewerkt**).

### Modals in past unreal chains

Modals can appear in **both** clauses:

| **als** clause | Main clause |
| --- | --- |
| **Als** ik **had** **moeten** **bellen** | **zou** ik **hebben** **gebeld** |
| **Als** ik **had** **kunnen** **helpen** | **zou** ik **hebben** **geholpen** |
| **Als** wij **hadden** **moeten** **wachten** | **zouden** wij **hebben** **gewacht** |

| Dutch | English |
| --- | --- |
| **Als** ik **had** **moeten** **stoppen**, **zou** ik **dat** **hebben** **gedaan**. | If I had had to stop, I would have done that. |
| **Als** jij **had** **kunnen** **komen**, **zou** je **ons** **hebben** **geholpen**. | If you could have come, you would have helped us. |
| **Als** zij **niet** **had** **willen** **komen**, **zou** zij **thuis** **zijn** **gebleven**. | If she had not wanted to come, she would have stayed home. |

In the **als** clause: **had** + modal + infinitive at the **end**. In the main clause: **zou** + **hebben** + modal + infinitive, or **zou** + **hebben** + past participle.

### Regret and hindsight language

| Form | Use | Example |
| --- | --- | --- |
| **Had** ik … **maar** …! | if only I had … | **Had** ik dat **maar** **geweten**! |
| **Het** **spijt** me **dat** … | I regret that … | **Het** **spijt** me **dat** ik **niet** **ben** **gekomen**. |
| **Achteraf** **gezien** | in hindsight | **Achteraf** **gezien** **zou** ik **anders** **hebben** **gehandeld**. |
| **Met** **de** **kennis** **van** **nu** | with what we know now | **Met** **de** **kennis** **van** **nu** **zouden** wij **niet** **tekenen**. |

| Dutch | English |
| --- | --- |
| **Had** ik **dat** **maar** **geweten**, **dan** **zou** ik **zijn** **gekomen**! | If only I had known, I would have come! |
| **Het** **spijt** me **dat** ik **zo** **voorbarig** **heb** **gereageerd**. | I regret that I reacted so hastily. |
| **Achteraf** **gezien** **was** het **een** **vergissing** **om** **te** **tekenen**. | In hindsight, signing was a mistake. |
| **Met** **de** **kennis** **van** **nu** **zou** ik **het** **verzoek** **hebben** **afgewezen**. | With what I know now, I would have declined the request. |

### Past unreal vs present unreal

| Present unreal (L6) | Past unreal (L7) |
| --- | --- |
| **Als** ik **meer** **tijd** **had**, **zou** ik **komen**. (now) | **Als** ik **meer** **tijd** **had** **gehad**, **zou** ik **zijn** **gekomen**. (then) |
| **Als** ik **rijk** **was**, **zou** ik **reizen**. | **Als** ik **rijk** **was** **geweest**, **zou** ik **hebben** **gereisd**. |

Ask: is the **unreal condition** about **now** or **already in the past**?

### Combining with earlier B1 grammar

| Topic | Past unreal example |
| --- | --- |
| B1 L1 **hoewel** | **Hoewel** ik **spijt** **heb**, **zou** ik **het** **niet** **anders** **hebben** **gedaan**. |
| B1 L2 **die/dat** | **De** **fout** **die** ik **had** **kunnen** **voorkomen**, **heb** ik **niet** **gezien**. |
| B1 L5 **zou** | **Zou** je **me** **hebben** **gebeld**, **als** je **het** **had** **geweten**? |
| B1 L6 **als** unreal | Upgrade: **Als** ik **jou** **was**, **zou** ik **hebben** **gewacht** → **Als** ik **jou** **was** **geweest**, **zou** ik **hebben** **gewacht**. |

### Preview: passive (Lesson 8)

Past unreal pairs naturally with **passive** regret: **Als** het **was** **afgesproken**, **zou** het **zijn** **uitgevoerd**. Full **worden** passive comes in [Lesson 8](/notes/b1/08-passive-with-worden).

## Common Mistakes for English Speakers

<ShadCnCard className="my-4">
  <CardHeader>
    <CardTitle className="text-lg">Simple past instead of plusquamperfect in als</CardTitle>
  </CardHeader>

  <CardContent>
    ❌ **Als** ik **wist** het, **zou** ik komen.

    ✅ **Als** ik het **had** **geweten**, **zou** ik **zijn** **gekomen**.

    Past unreal **als** needs **had/was** + participle, not simple past **wist/kwam**.
  </CardContent>
</ShadCnCard>

<ShadCnCard className="my-4">
  <CardHeader>
    <CardTitle className="text-lg">zou + infinitive for past result</CardTitle>
  </CardHeader>

  <CardContent>
    ❌ **Als** ik het had geweten, **zou** ik **komen**.

    ✅ **Als** ik het had geweten, **zou** ik **zijn** **gekomen**.

    Past result uses **zou** + **hebben/zijn** + participle, not bare infinitive.
  </CardContent>
</ShadCnCard>

<ShadCnCard className="my-4">
  <CardHeader>
    <CardTitle className="text-lg">Wrong auxiliary: hebben vs zijn</CardTitle>
  </CardHeader>

  <CardContent>
    ❌ **Als** ik had geweten, **zou** ik **hebben** **gekomen**.

    ✅ **Als** ik had geweten, **zou** ik **zijn** **gekomen**.

    Movement verbs (**komen**, **gaan**, **blijven**) take **zijn** in the perfect.
  </CardContent>
</ShadCnCard>

<ShadCnCard className="my-4">
  <CardHeader>
    <CardTitle className="text-lg">Using L6 pattern for past events</CardTitle>
  </CardHeader>

  <CardContent>
    ❌ **Als** ik **gisteren** **meer** **tijd** **had**, **zou** ik **komen**. (about yesterday, did not come)

    ✅ **Als** ik **gisteren** **meer** **tijd** **had** **gehad**, **zou** ik **zijn** **gekomen**.

    When the missed chance is **already past**, use plusquamperfect + **zou** perfect.
  </CardContent>
</ShadCnCard>

## Learning Tips

<ShadCnCard className="my-4">
  <CardHeader>
    <CardTitle className="text-lg">Had … zou … perfect drill</CardTitle>
  </CardHeader>

  <CardContent>
    Build five **Als ik … had …, zou ik … hebben/zijn …** chains about missed meetings, calls, and decisions. Say them aloud. The **had … zou … ge** rhythm should become automatic.
  </CardContent>
</ShadCnCard>

<ShadCnCard className="my-4">
  <CardHeader>
    <CardTitle className="text-lg">Had ik maar flashcards</CardTitle>
  </CardHeader>

  <CardContent>
    Turn three real regrets into **Had ik … maar …!** exclamations. This chunk is very common in spoken Dutch and sounds natural in apologies.
  </CardContent>
</ShadCnCard>

<ShadCnCard className="my-4">
  <CardHeader>
    <CardTitle className="text-lg">L6 to L7 upgrade</CardTitle>
  </CardHeader>

  <CardContent>
    Take three L6 sentences (**Als ik meer tijd had, zou ik …**) and push them into the past: **Als ik meer tijd had gehad, zou ik … hebben …**. Notice how **gehad** and **hebben/zijn** stack.
  </CardContent>
</ShadCnCard>

    > **Mila:** I regret that I did not come yesterday. If only I had known!
    >
    > **Sam:** No blame. What was going on?
    >
    > **Mila:** If I had seen the request, I would have come. I failed to respond.
    >

    >

    >
    > **Mila:** Good idea. I do not want to leave repairing trust aside. Would you help me put it right?
