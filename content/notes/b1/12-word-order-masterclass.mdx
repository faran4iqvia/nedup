## Introduction

From A1 you learned **V2**: **Ik werk vandaag**, **Vandaag werk ik**. From A2 and B1 you stacked **bijzinnen**, modals, perfect tenses, passives, **zou**, and **te**. Now you pull it together: where every piece goes in **main clauses**, **subordinate clauses**, and **verb clusters** at the end. After this lesson you can build long Dutch sentences with confidence: **Hoewel ik morgen had moeten werken, beloofde hij te helpen**, **Ik denk dat het rapport goedgekeurd moet worden voordat wij exporteren**.

## Grammar and Language Explanation

### The two clause types

| Clause | Dutch name | Verb rule | Example |
| --- | --- | --- | --- |
| Main | **hoofdzin** | **V2**: conjugated verb in position 2 | Ik **werk** vandaag. |
| Subordinate | **bijzin** | verb(s) at the **end** | … **omdat** ik vandaag **werk**. |

| Dutch | English |
| --- | --- |
| Ik **ga** **naar** huis, **omdat** ik moe **ben**. | I am going home because I am tired. |
| **Hoewel** het **regent**, **ga** ik **toch** **uit**. | Although it is raining, I still go out. |
| Ik **denk** **dat** hij **komt**. | I think he is coming. |

### Main clause: V2 and inversion

**Default:** Subject · Verb · Rest

| Dutch | English |
| --- | --- |
| Ik **werk** vandaag **thuis**. | I work at home today. |
| Mila **geeft** Sam **het** **rapport**. | Mila gives Sam the report. |

**Fronted element** (time, place, object): that element takes position 1; verb stays position 2; subject often follows the verb:

| Dutch | English |
| --- | --- |
| **Vandaag** **werk** ik **thuis**. | Today I work at home. |
| **Morgen** **moet** ik **indienen**. | Tomorrow I must submit. |
| **Het** **rapport** **geeft** Mila **aan** Sam. | The report, Mila gives to Sam. |

| Dutch | English |
| --- | --- |
| **Gisteren** **had** ik **moeten** **bellen**. | Yesterday I should have called. |
| **In** **dat** **geval** **zou** ik **wachten**. | In that case I would wait. |

From [A1 Lesson 3](/notes/a1/03-negation): **niet** usually sits before the item it negates, often before the verb cluster or separable prefix.

### Subordinate clause: verb last

After **dat**, **omdat**, **hoewel**, **als**, **of**, **wanneer**, **nadat**, **voordat**, **totdat**, **terwijl**, the conjugated verb goes to the **end**. If there is a **cluster**, the whole cluster goes to the end.

| Main | Subordinate |
| --- | --- |
| Ik **weet** … | … **dat** hij **morgen** **komt**. |
| Ik **blijf** thuis … | … **omdat** ik **ziek** **ben**. |
| **Hoewel** … | … **hoewel** ik **moe** **ben**, **ga** ik **toch**. |

| Dutch | English |
| --- | --- |
| Ik **mel** **dat** het **rapport** **is** **afgerond**. | I report that the report is completed. |
| **Voordat** wij **exporteren**, **moet** alles **worden** **gecontroleerd**. | Before we export, everything must be checked. |
| **Nadat** ik **heb** **gegeten**, **bel** ik je. | After I have eaten, I call you. |

### Bijzin first: inversion in the main clause

When the **bijzin** comes **first**, the **hoofdzin** still needs **V2**:

| Order | Example |
| --- | --- |
| bijzin + hoofdzin | **Omdat** ik **ziek** **ben**, **blijf** ik **thuis**. |
| hoofdzin + bijzin | Ik **blijf** thuis, **omdat** ik **ziek** **ben**. |

| Dutch | English |
| --- | --- |
| **Hoewel** ik **morgen** **moet** **werken**, **ga** ik **vanavond** **uit**. | Although I must work tomorrow, I am going out tonight. |
| **Als** het **regent**, **blijven** wij **binnen**. | If it rains, we stay inside. |

### Verb clusters: the B1 recipes

In **subordinate** clauses, non finite verbs pile up at the end. Learn these **recipes**:

| Type | Subordinate pattern | Example |
| --- | --- | --- |
| Single verb | … **dat** ik **werk** | … **dat** ik morgen **werk** |
| Modal + infinitive | … **dat** ik **moet** **werken** | … **dat** ik morgen **moet** **werken** |
| Perfect | … **dat** ik **heb** **gewerkt** | … **dat** ik gisteren **heb** **gewerkt** |
| Modal + perfect | … **dat** ik **heb** **moeten** **werken** | … **dat** ik gisteren **heb** **moeten** **werken** |
| **worden** passive | … **dat** het **goedgekeurd** **moet** **worden** | … **dat** het **moet** **worden** **goedgekeurd** |
| **zijn** passive | … **dat** het **is** **goedgekeurd** | … **dat** het **al** **is** **goedgekeurd** |
| **zou** + infinitive | … **dat** hij **zou** **komen** | … **dat** hij **morgen** **zou** **komen** |
| **zou** + perfect | … **dat** hij **zou** **hebben** **gebeld** | … **dat** hij **gisteren** **zou** **hebben** **gebeld** |
| **te** after adjective | … **dat** het **moeilijk** **is** **te** **vinden** | … **dat** het **lastig** **is** **te** **lezen** |
| Verb + **te** | … **dat** zij **belooft** **te** **helpen** | … **dat** hij **probeert** **te** **komen** |

| Dutch | English |
| --- | --- |
| Ik **denk** **dat** ik **morgen** **niet** **kan** **komen**. | I think I cannot come tomorrow. |
| **Hij** **zei** **dat** hij **gisteren** **had** **moeten** **bellen**. | He said he should have called yesterday. |
| **Zij** **meldt** **dat** het **rapport** **goedgekeurd** **moet** **worden**. | She reports the report must be approved. |

**Main clause** clusters sit at the **end** too, but the **conjugated** verb is in position 2:

| Dutch | English |
| --- | --- |
| Ik **moet** morgen **werken**. | I must work tomorrow. |
| Ik **heb** gisteren **moeten** **werken**. | I had to work yesterday. |
| Het **moet** **worden** **goedgekeurd**. | It must be approved. |

### worden vs zijn cluster order (subordinate)

| Passive | Subordinate order |
| --- | --- |
| **worden** + modal | … **goedgekeurd** **moet** **worden** |
| **zijn** + modal | … **afgerond** **moet** **zijn** |
| **zijn** result | … **is** **goedgekeurd** |

| Dutch | English |
| --- | --- |
| … **voordat** het **is** **gecontroleerd** | … before it has been checked |
| … **dat** het **niet** **mag** **worden** **gedeeld** | … that it may not be shared |
| … **dat** alles **afgerond** **moet** **zijn** | … that everything must be finished |

From [B1 Lesson 8](/notes/b1/08-passive-with-worden) and [B1 Lesson 9](/notes/b1/09-passive-with-zijn).

### te and om te in clusters

| Pattern | Example |
| --- | --- |
| Main + **om te** | Ik **ga** **naar** huis **om te** **rusten**. |
| Subordinate + **te** | … **dat** het **makkelijk** **is** **te** **gebruiken** |
| **zonder** / **door** + **te** | **Zonder** **te** **vragen** **nam** zij **het** **over**. |
| Verb + **te** | … **dat** zij **belooft** **te** **komen** |

| Dutch | English |
| --- | --- |
| … **dat** ik **vergeten** **ben** **te** **bellen** | … that I forgot to call |
| … **omdat** ik **probeer** **te** **helpen** | … because I try to help |
| **Door** **te** **oefenen** **word** je **beter**. | By practising you get better. |

From [B1 Lesson 11](/notes/b1/11-te-plus-infinitive).

### Separable verbs in clusters

The **prefix** stays with the **infinitive** or **participle** at the end; **te** splits separables:

| Type | Example |
| --- | --- |
| Main clause | Ik **sta** **om** **acht** uur **op**. |
| Subordinate | … **dat** ik **om** **acht** uur **opsta** |
| **om te** | … **om** **op** **te** **staan** |
| **zonder te** | … **zonder** **af** **te** **spreken** |

| Dutch | English |
| --- | --- |
| … **dat** wij **morgen** **afspreken** | … that we agree tomorrow |
| Ik **probeer** **op** **tijd** **te** **komen**. | I try to arrive on time. |
| … **dat** het **niet** **is** **doorgevoerd** | … that it has not been implemented |

From [A1 Lesson 21](/notes/a1/21-separable-verbs).

### Pronoun order

| Two nouns | Two pronouns |
| --- | --- |
| IO before DO: **Ik geef Mila het boek** | DO before IO: **Ik geef het haar** |

| Dutch | English |
| --- | --- |
| **Geef** **het** **me** **niet**. | Do not give it to me. |
| Ik **heb** **het** **haar** **gisteren** **beloofd** **te** **sturen**. | I promised her yesterday to send it. |

In **subordinate** clauses, pronoun objects usually sit in the **middengedeelte** (middle field), before the verb cluster:

| Dutch | English |
| --- | --- |
| … **dat** ik **het** **haar** **gisteren** **heb** **beloofd** **te** **sturen** | … that I promised her yesterday to send it |
| … **omdat** ik **hem** **niet** **kon** **bereiken** | … because I could not reach him |

From [A2 Lesson 15](/notes/a2/15-indirect-objects).

### niet in complex sentences

| Position | Example |
| --- | --- |
| Negate verb / cluster | Ik **werk** **niet**. / … **dat** ik **niet** **werk** |
| Negate adjective | Het **is** **niet** **moeilijk** **te** **vinden**. |
| Negate specific word | Ik **ga** **niet** **morgen** **maar** **overmorgen**. |
| Before separable prefix | Ik **ga** **niet** **me**e. |

| Dutch | English |
| --- | --- |
| … **dat** ik **gisteren** **niet** **heb** **kunnen** **komen** | … that I could not come yesterday |
| **Hoewel** ik **het** **niet** **eens** **ben**, **respecteer** ik **het**. | Although I do not agree, I respect it. |
| Ik **probeer** **niet** **te** **talmen**. | I try not to dawdle. |

### Long sentence building: step by step

Build from the **core**, then add **bijzinnen**:

| Step | Sentence |
| --- | --- |
| 1. Core | Ik **denk** … |
| 2. Add **dat** | Ik **denk** **dat** … |
| 3. Subordinate cluster | Ik **denk** **dat** het **rapport** **is** **afgerond**. |
| 4. Add second bijzin | **Voordat** ik **bel**, **denk** ik **dat** het **rapport** **is** **afgerond**. |

| Dutch | English |
| --- | --- |
| **Hoewel** **hij** **zei** **dat** hij **zou** **komen**, **is** hij **niet** **gekomen**. | Although he said he would come, he did not come. |
| Ik **mel** **dat** de **data** **versleuteld** **moeten** **zijn** **voordat** ze **worden** **geëxporteerd**. | I report that the data must be encrypted before they are exported. |

### Parataxe vs hypotaxe

| Type | Structure | Example |
| --- | --- | --- |
| **Parataxe** | clauses side by side | Ik **kom**, **ik** **zie**, ik **win**. |
| **Hypotaxe** | main + subordinate | Ik **kom**, **omdat** jij **vraagt**. |

B1 Dutch in writing and meetings is mostly **hypotactisch**: linked with **dat**, **omdat**, **hoewel**.

### Preview: nuanced opinions (Lesson 13)

Word order matters for **emphasis** in opinions: **Het is naar mijn mening niet nodig …**. Nuance chunks come in [Lesson 13](/notes/b1/13-nuanced-opinions).

<ShadCnCard className="my-4">
  <CardHeader>
    <CardTitle className="text-lg">Zinsanalyse in de training</CardTitle>
    <CardDescription>Mila and Sam analyse word order during an internal Dutch writing workshop.</CardDescription>
  </CardHeader>

  <CardContent>
    **In Dutch:**

    > **Sam:** **Kun** je **deze** **zin** **ontleden**? **Voordat** wij **exporteren**, **moet** alles **worden** **gecontroleerd**.
    >
    > **Mila:** **Ja**. **Voordat** wij **exporteren** is **de** **ingebedde** **bijzin**. **Moet** **worden** **gecontroleerd** is **het** **werkwoordcluster** **achteraan** in **de** **hoofdzin**.
    >
    > **Sam:** **En** **deze**? **Ik** **denk** **dat** het **rapport** **goedgekeurd** **moet** **worden** **voordat** **het** **wordt** **gepubliceerd**.
    >
    > **Mila:** **Hypotactisch**, **hoor**. **Twee** **bijzinnen**: **dat** het **goedgekeurd** **moet** **worden**, en **voordat** het **wordt** **gepubliceerd**. **Beide** **eindigen** **met** een **cluster**.
    >
    > **Sam:** **Waar** **plaats** je **niet**?
    >
    > **Mila:** **Voor** het **cluster**: **dat** ik **niet** **heb** **kunnen** **komen**. **Niet** **na** **moet**.
    >
    > **Sam:** **Gisteren** **had** ik **moeten** **bellen**, **hoewel** ik **wist** **dat** ik **te** **laat** **was**.
    >
    > **Mila:** **Mooi**. **Gisteren** **vooraan**, **dan** **inversie**. **Hoewel** **achteraan** **met** **wist** **aan** het **eind**. **Regelconform**.

    **In English:**

    > **Sam:** Can you parse this sentence? Before we export, everything must be checked.
    >
    > **Mila:** Yes. Before we export is the embedded clause. Must be checked is the verb cluster at the end of the main clause.
    >
    > **Sam:** And this one? I think the report must be approved before it is published.
    >
    > **Mila:** Hypotactic. Two subordinate clauses: that it must be approved, and before it is published. Both end with a cluster.
    >
    > **Sam:** Where do you place niet?
    >
    > **Mila:** Before the cluster: that I could not come. Not after must.
    >
    > **Sam:** Yesterday I should have called, although I knew I was too late.
    >
    > **Mila:** Nice. Yesterday at the front, then inversion. Although at the end with knew at the end. Rule-compliant.
  </CardContent>
</ShadCnCard>
