## Introduction

At A2 you learned the core passive: **De brief wordt geschreven**, **Er wordt gewerkt**, **werd verplaatst**. B1 takes **worden** further: naming the doer with **door**, combining passives with **moeten / kunnen / mogen**, forming **zal worden** futures, and placing passives correctly in **dat** and **omdat** clauses. After this lesson you can read compliance emails, IT updates, and policy documents in natural Dutch passive, and write them yourself.

## Grammar and Language Explanation

### Active vs passive: why B1 returns to worden

| Active | Passive (process) |
| --- | --- |
| **Het** **team** **implementeert** de wijziging. | De wijziging **wordt** **geïmplementeerd**. |
| **Wij** **keuren** het rapport **goed**. | Het rapport **wordt** **goedgekeurd**. |
| **IT** **configureert** het systeem. | Het systeem **wordt** **geconfigureerd**. |

| Role | Active | Passive |
| --- | --- | --- |
| Focus | who acts | what happens |
| Subject | doer | receiver of action |
| Verb | main verb | **worden** + participle |

Use **worden** passive when the **action is in progress** or when the doer is unknown, unimportant, or obvious from context: policies, procedures, system messages.

### worden conjugation (review + extension)

| Tense | singular | plural | Example |
| --- | --- | --- | --- |
| Present | **wordt** | **worden** | De regel **wordt** **ingevoerd**. |
| Past | **werd** | **werden** | De regels **werden** **aangepast**. |
| Future | **zal** **worden** | **zullen** **worden** | De data **zullen** **worden** **geëxporteerd**. |

| Dutch | English |
| --- | --- |
| De **wijziging** **wordt** **vandaag** **doorgevoerd**. | The change is being implemented today. |
| De **richtlijn** **werd** **vorige** **week** **vastgesteld**. | The guideline was established last week. |
| De **configuratie** **zal** **morgen** **worden** **afgerond**. | The configuration will be completed tomorrow. |

### door + agent: full B1 treatment

At A2 you recognised **door**. At B1 you **use** it confidently.

| Without agent | With **door** |
| --- | --- |
| Het rapport **wordt** **goedgekeurd**. | Het rapport **wordt** **door** de manager **goedgekeurd**. |
| De data **worden** **geëxporteerd**. | De data **worden** **door** IT **geëxporteerd**. |
| De audit **wordt** **uitgevoerd**. | De audit **wordt** **door** een extern bureau **uitgevoerd**. |

| Placement | Example |
| --- | --- |
| **wordt door** + agent + participle | Het systeem **wordt door** IT **geconfigureerd**. |
| **wordt** + participle + **door** + agent | Het systeem **wordt geconfigureerd door** IT. |

Both orders are correct. In formal writing, **wordt door** + agent + participle is very common.

| Dutch | English |
| --- | --- |
| De **validatie** **wordt** **door** het **kwaliteitsteam** **uitgevoerd**. | The validation is carried out by the quality team. |
| De **documentatie** **wordt** **gedocumenteerd** **door** elke afdeling. | The documentation is recorded by each department. |
| De **regelgeving** **wordt** **gehandhaafd** **door** de **verantwoordelijke**. | The regulations are enforced by the person responsible. |

**Door** + person or team. Not **van**, not **door de** + verb.

### Modal + passive

Modals sit **before** **worden** in main clauses:

| Modal | Pattern | Example |
| --- | --- | --- |
| **moeten** | **moet** **worden** + participle | Het rapport **moet** **worden** **goedgekeurd**. |
| **kunnen** | **kan** / **kunnen** **worden** + participle | De data **kunnen** **worden** **geëxporteerd**. |
| **mogen** | **mag** / **mogen** **niet** **worden** + participle | Dit **mag** **niet** **worden** **gedeeld**. |
| **willen** | **wil** / **willen** **worden** + participle | De wijziging **wil** **niet** **worden** **doorgevoerd**. (resists / will not go through) |

| Dutch | English |
| --- | --- |
| De **nieuwe** **richtlijn** **moet** **worden** **geïmplementeerd** **voor** **vrijdag**. | The new guideline must be implemented before Friday. |
| De **gegevens** **moeten** **worden** **geanonimiseerd** **voordat** ze **worden** **geëxporteerd**. | The data must be anonymised before they are exported. |
| **Mag** dit **worden** **gedeeld** **met** externe partners? | May this be shared with external partners? |

In **spoken** Dutch you also hear **moet goedgekeurd worden**. In **formal** writing, **moet worden goedgekeurd** is standard.

### Future passive with zullen

| Active future | Passive future |
| --- | --- |
| Wij **zullen** de regel **invoeren**. | De regel **zal** **worden** **ingevoerd**. |
| IT **zal** het systeem **configureren**. | Het systeem **zal** **worden** **geconfigureerd**. |

| Dutch | English |
| --- | --- |
| De **wijziging** **zal** **volgende** **week** **worden** **doorgevoerd**. | The change will be implemented next week. |
| De **resultaten** **zullen** **worden** **geëvalueerd** **door** het **team**. | The results will be evaluated by the team. |
| **Er** **zal** **morgen** **worden** **gewerkt** **aan** de **integratie**. | Work will be done on the integration tomorrow. |

From [B1 Lesson 4](/notes/b1/04-future-with-zullen): **zal** for **ik/hij/zij**, **zullen** for **wij/jullie/zij**.

### Passive in subordinate clauses

The verb cluster goes to the **end**: participle + **worden** (+ modal if present).

| Main clause | Subordinate |
| --- | --- |
| Ik meld **dat** … | … **dat** het rapport **goedgekeurd** **moet** **worden**. |
| … **omdat** … | … **omdat** de toegang **geblokkeerd** **wordt**. |
| … **hoewel** … | … **hoewel** de audit **nog** **niet** **is** **afgerond**. |

| Dutch | English |
| --- | --- |
| **Ik** **geef** **door** **dat** de **configuratie** **morgen** **wordt** **afgerond**. | I report that the configuration will be completed tomorrow. |
| **Omdat** de **regelgeving** **wordt** **aangepast**, **moet** alles **opnieuw** **worden** **gevalideerd**. | Because the regulations are being adjusted, everything must be revalidated. |
| **Hoewel** het **rapport** **al** **was** **goedgekeurd**, **werd** het **later** **gewijzigd**. | Although the report had already been approved, it was changed later. |

From [B1 Lesson 1](/notes/b1/01-conjunctions-overview): **dat**, **omdat**, **hoewel** + verb last.

### Er wordt gezegd / verwacht / gemeld dat

Impersonal passive with a **dat** clause is common in news and corporate Dutch:

| Pattern | Example |
| --- | --- |
| **Er** **wordt** **gezegd** **dat** … | **Er** **wordt** **gezegd** **dat** de deadline **verschuift**. |
| **Er** **wordt** **verwacht** **dat** … | **Er** **wordt** **verwacht** **dat** de audit **slaagt**. |
| **Er** **wordt** **gemeld** **dat** … | **Er** **wordt** **gemeld** **dat** er **een** **lek** **is**. |

| Dutch | English |
| --- | --- |
| **Er** **wordt** **verwacht** **dat** de **wijziging** **volgende** **maand** **wordt** **ingevoerd**. | It is expected that the change will be introduced next month. |
| **Er** **wordt** **gezegd** **dat** alle **data** **moeten** **worden** **versleuteld**. | It is said that all data must be encrypted. |

### Separable verbs in the passive

Separable prefixes stay with the participle:

| Active | Passive |
| --- | --- |
| Wij **voeren** de wijziging **door**. | De wijziging **wordt** **doorgevoerd**. |
| IT **voert** de update **uit**. | De update **wordt** **uitgevoerd**. |
| Zij **voeren** de regel **in**. | De regel **wordt** **ingevoerd**. |

| Dutch | English |
| --- | --- |
| De **standaardisatie** **wordt** **gefaseerd** **doorgevoerd**. | Standardisation is being phased in. |
| De **oude** **versie** **wordt** **volgende** **week** **uitgefaseerd**. | The old version is being phased out next week. |

### When Dutch prefers passive

| Context | Why passive |
| --- | --- |
| Policy / compliance | focus on rule, not person |
| IT / systems | process matters |
| Official announcements | neutral tone |
| Unknown agent | **Mijn** **account** **wordt** **geblokkeerd**. |

| Dutch | English |
| --- | --- |
| **Conform** de **richtlijn** **moet** elke **wijziging** **worden** **gedocumenteerd**. | In accordance with the guideline, every change must be documented. |
| **De** **naleving** **van** de **wetgeving** **wordt** **gehandhaafd**. | Compliance with legislation is enforced. |
| **Uw** **aanvraag** **wordt** **verwerkt** **en** **zal** **binnen** **vijf** **dagen** **worden** **beoordeeld**. | Your application is being processed and will be assessed within five days. |

### worden vs zijn: preview (Lesson 9)

| **worden** (process) | **zijn** (result / state) |
| --- | --- |
| Het rapport **wordt** **gemaakt**. | Het rapport **is** **gemaakt**. |
| De data **worden** **geëxporteerd**. | De data **zijn** **geëxporteerd**. |

At B1 you **produce** complex **worden** passives. [Lesson 9](/notes/b1/09-passive-with-zijn) covers **zijn** + participle for completed states.

<Callout title="Pronunciation Focus" type="info">
  In **moet worden goedgekeurd**, link **worden-goed**: _moet wor-den-goed-ge-keurd_. In **Er wordt gezegd dat**, keep **er wordt** light. See [Connected Speech](/notes/pronunciation/05-connected-speech).
</Callout>
