## Introduction

Dutch uses **verkleinwoorden** constantly: **het huisje**, **een kopje thee**, **even een hapje**, **wat een schattig hondje**. The ending **je** (or **tje**, **pje**, **kje**) can mean **small**, but often it means **friendly**, **informal**, or **soft**. After this lesson you can form diminutives with the main spelling rules, use **het** correctly, and sound warmer in everyday Dutch without treating every **je** word as _tiny_.

## Grammar and Language Explanation

### What is a verkleinwoord?

A **verkleinwoord** (diminutive) is a noun form with **je**, **tje**, **pje**, or **kje**. Dutch calls the process **verkleinen**.

| Dutch | English |
| --- | --- |
| **het** **verkleinwoord** | diminutive noun |
| **de** **verkleiningsvorm** | diminutive form |
| **verkleinen** | to form a diminutive / make smaller |
| **de** **verkleiningsregel** | diminutive rule |

| Dutch | English |
| --- | --- |
| **Huis** **wordt** **huisje**. | House becomes little house / cottage. |
| **Straat** **wordt** **straatje**. | Street becomes little street / lane. |
| **Wat** **een** **mooi** **huisje**! | What a lovely little house! |

### All diminutives are het

Every diminutive takes **het**, even if the base noun was **de**:

| Base | Diminutive |
| --- | --- |
| **de** **man** | **het** **mannetje** |
| **de** **straat** | **het** **straatje** |
| **het** **huis** | **het** **huisje** |
| **de** **bloem** | **het** **bloemetje** |

| Dutch | English |
| --- | --- |
| **Het** **mannetje** **op** **de** **foto** **is** **mijn** **oom**. | The little man in the photo is my uncle. |
| **Ik** **woon** **in** **een** **oud** **huisje**. | I live in an old cottage. |
| **Zij** **heeft** **een** **schattig** **katje**. | She has a cute kitten. |

Plural: usually **de** + stem + **s** or **jes**: **de huisjes**, **de straatjes**, **de autootjes**.

### The four endings: je, tje, pje, kje

Dutch chooses an ending based on the **last sound** of the base noun.

| Ending | Typical context | Example |
| --- | --- | --- |
| **-je** | after vowel or diphthong | **auto** Ôćĺ **autootje** |
| **-tje** | after **n**, many consonants | **boon** Ôćĺ **boontje** |
| **-pje** | after **l**, **r**, **w** | **appel** Ôćĺ **appeltje** |
| **-kje** | after **ng** (and some other cases) | **ding** Ôćĺ **dingetje** |

| Dutch | English |
| --- | --- |
| **Een** **autootje** **voor** **de** **deur**. | A little car in front of the door. |
| **Wil** **je** **een** **boontje**? | Do you want a little bean / jelly bean? |
| **Een** **appeltje** **per** **dag**. | An apple a day. |
| **Dat** **is** **een** **raar** **dingetje**. | That is a weird little thing. |

In practice, **je** and **tje** cover most words you need at B1. **Pje** appears after **l/r/w** in many common words (**appeltje**, **krantje** pattern).

### Spelling rules that matter

#### 1. Final n becomes t + je

| Base | Diminutive |
| --- | --- |
| **boon** | **boontje** |
| **kraan** | **kraantje** |
| **padden** (pad) | **padje** / **paddetje** (learn **pad** Ôćĺ **padje** as common) |

ÔŁî **bonenje** ┬Ě Ôťů **boontje**

#### 2. Short vowel + one consonant Ôćĺ double the consonant

| Base | Diminutive |
| --- | --- |
| **man** | **mannetje** |
| **kop** | **kopje** (you know from A1) |
| **kat** | **katje** |
| **glas** | **glaasje** |
| **tas** | **tasje** |

| Dutch | English |
| --- | --- |
| **Wat** **een** **lief** **mannetje**! | What a sweet little man! |
| **Het** **glaasje** **water**, **alsjeblieft**. | The glass of water, please. |

#### 3. Long vowel or diphthong Ôćĺ often just add je (sometimes lengthen)

| Base | Diminutive |
| --- | --- |
| **huis** | **huisje** |
| **vrouw** | **vrouwtje** |
| **auto** | **autootje** |
| **boot** | **bootje** |
| **fiets** | **fietsje** |

#### 4. Final sch, s, x Ôćĺ add je

| Base | Diminutive |
| --- | --- |
| **huis** | **huisje** |
| **bus** | **busje** |
| **doos** | **doosje** |

| Dutch | English |
| --- | --- |
| **Wij** **huren** **een** **huisje** **aan** **zee**. | We rent a cottage by the sea. |
| **Het** **bootje** **ligt** **aan** **de** **kade**. | The little boat is at the quay. |

### Common fixed forms to learn as wholes

Some diminutives are high frequency. Learn them as vocabulary, not only as rules:

| Diminutive | Meaning |
| --- | --- |
| **een** **beetje** | a little bit |
| **een** **stukje** | a small piece |
| **een** **momentje** | just a moment |
| **een** **woordje** | a quick word |
| **een** **hapje** | a bite / snack |
| **een** **biertje** | a beer (informal) |
| **een** **dagje** **uit** | a day out |
| **een** **uurtje** | about an hour |

| Dutch | English |
| --- | --- |
| **Heb** **je** **een** **momentje**? | Do you have a moment? |
| **Ik** **neem** **een** **stukje** **kaas**. | I'll take a little piece of cheese. |
| **Zullen** **we** **een** **biertje** **doen**? | Shall we have a beer? |
| **Wij** **gaan** **een** **dagje** **naar** **de** **stad**. | We're going to the city for the day. |

### Affective meaning: more than small

**De affectieve betekenis** (affective meaning) is central in Dutch. A diminutive can express:

| Nuance | Example | English feel |
| --- | --- | --- |
| **Size** | **een** **klein** **kamertje** | a small room |
| **Affection** | **schattig** **kindje** | sweet child |
| **Softening** | **een** **klein** **foutje** | a small mistake (less harsh) |
| **Informality** | **koffietje** **tussendoor** | casual coffee break |
| **Comfort** | **troostend** **theetje** | comforting cup of tea |

| Dutch | English |
| --- | --- |
| **Wat** **een** **schattig** **hondje**! | What a cute little dog! |
| **Geen** **zorgen**, **een** **klein** **foutje** **kan** **iedereen** **gebeuren**. | No worries, a small mistake can happen to anyone. |
| **Kom**, **ik** **zet** **een** **lekker** **theetje** **voor** **je** **klaar**. | Come, I'll make you a nice cup of tea. |

**Vertederen** = to soften emotionally. **De vertedering** = softening tone.

Words like **schattig**, **liefelijk**, and **troostend** often appear with diminutives.

### When diminutives sound natural

| Situation | Natural |
| --- | --- |
| Caf├ę, home, friends | **een** **wijntje**, **het** **terrasje** |
| Talking to children | **ga** **naar** **bedje** |
| Softening bad news | **een** **teleurstellend** **resultaat** Ôćĺ **tegenvalletje** |
| Formal report / legal text | avoid stacked diminutives |

| Dutch | English |
| --- | --- |
| **Zullen** **we** **op** **het** **terrasje** **zitten**? | Shall we sit on the little terrace? |
| **Het** **kindje** **sliep** **al**. | The little child was already asleep. |
| **In** **het** **verslag** **schrijf** ik **geen** **verkleinwoorden**. | In the report I do not write diminutives. |

### Special case: het meisje

**Het meisje** (girl) looks like a diminutive of **meid**, and historically it is. Today it is the **normal** word, not _a small girl_. The same pattern appears in a few fossilised forms. Do not assume every **je** word is _extra small_.

| Dutch | English |
| --- | --- |
| **Het** **meisje** **leest** **een** **boekje**. | The girl is reading a little book. |
| **Een** **meisje** **van** **twaalf** **jaar** | A twelve year old girl (standard size word) |

### Diminutives in longer sentences

Diminutives combine with B1 grammar you already know:

| Pattern | Example |
| --- | --- |
| **er** + diminutive | **Ik** **woon** **in** **een** **klein** **huisje** **in** **een** **rustig** **straatje**. |
| question | **Wil** **je** **nog** **een** **koekje**? |
| past story | **Wij** **dronken** **een** **biertje** **op** **het** **pleintje**. |
| opinion | **Ik** **vind** **het** **huisje** **erg** **gezellig**. |

From [B1 Lesson 22](/notes/b1/22-storytelling-past-tenses): **Op een gegeven moment zagen we een klein hondje bij het station.**

<Callout title="Pronunciation Focus" type="info">
  Diminutives are unstressed at the end: **HUIS-je**, **STRAAT-je**, **MAN-net-je**. The **je** is short, like **uh**. In fast speech **momentje** Ôćĺ **mo-MENT-juh**. Practise: **Zullen we een kopje thee doen?** See [Connected Speech](/notes/pronunciation/05-connected-speech).
</Callout>
