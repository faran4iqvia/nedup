## Introduction

You know **de vergadering**, **solliciteren**, and **het cv** from [A2 Lesson 17](/notes/a2/17-workplace). B1 adds what happens **inside** meetings and **during** job hunts: chairing language, **notulen**, **actiepunten**, a full **motivatiebrief**, and **sollicitatiegesprek** answers. After this lesson you can join a Dutch **teamoverleg**, write a professional application, and handle interview questions without switching to English in your head.

## Grammar and Language Explanation

### Types of work meetings

| Dutch | English | Typical use |
| --- | --- | --- |
| **de** **vergadering** | meeting | formal, scheduled |
| **het** **teamoverleg** | team meeting | regular sync |
| **het** **overleg** | consultation | short discussion |
| **de** **brainstorm** | brainstorm | ideas |
| **de** **workshop** | workshop | training session |

| Dutch | English |
| --- | --- |
| **De** **vergadertijd** **is** **om** **tien** **uur**. | The meeting time is at ten o'clock. |
| **De** **vergaderduur** **is** **maximaal** **één** **uur**. | The meeting lasts one hour at most. |
| **Kun** **je** **de** **vergadering** **bijwonen**? | Can you attend the meeting? |

### Agenda and flow

A Dutch meeting usually follows a visible **agenda**:

| Step | Dutch chunk | Example |
| --- | --- | --- |
| Open | **Laten** **we** **beginnen**. | **Laten** **we** **beginnen** **met** **punt** **één**. |
| Introduce topic | **Het** **eerste** **agendapunt** **is** … | **Het** **eerste** **agendapunt** **is** **de** **planning**. |
| Status | **De** **stand** **van** **zaken** **is** … | **De** **stand** **van** **zaken** **is** **positief**. |
| Return | **Terugkomen** **op** … | **Terugkomen** **op** **het** **vorige** **actiepunt**. |
| Propose | **Ik** **stel** **voor** **om** … | **Ik** **stel** **voor** **om** **te** **wachten**. |
| Decide | **Besluitvorming** | **De** **besluitvorming** **volgt** **na** **overleg**. |
| Close | **Laten** **we** **afronden**. | **Laten** **we** **de** **vergadering** **afronden**. |

| Dutch | English |
| --- | --- |
| **De** **voorzitter** **opent** **de** **vergadering**. | The chair opens the meeting. |
| **De** **notulist** **maakt** **de** **notulen**. | The minute taker writes the minutes. |
| **Zijn** **er** **nog** **vragen** **over** **dit** **punt**? | Are there any more questions about this point? |

From [B1 Lesson 4](/notes/b1/04-future-with-zullen): **Zullen** **we** **beginnen**? · **Zullen** **we** **stemmen**?

### Actiepunten and opvolging

Meetings end with **who does what**:

| Dutch | English |
| --- | --- |
| **Het** **actiepunt** **voor** **Mila** **is** … | Mila's action point is … |
| **Sam** **volgt** **dit** **op** **voor** **vrijdag**. | Sam follows this up by Friday. |
| **De** **opvolging** **staat** **in** **de** **notulen**. | The follow up is in the minutes. |
| **De** **opvolger** **neemt** **het** **over**. | The successor takes it over. |

| Dutch | English |
| --- | --- |
| **Ik** **agendeer** **het** **onderwerp** **voor** **volgende** **week**. | I put the topic on the agenda for next week. |
| **De** **agendering** **staat** **in** **de** **kalender**. | The scheduling is in the calendar. |
| **De** **taakverdeling** **is** **duidelijk**. | The division of tasks is clear. |

### Discussing progress and priorities

| Dutch | English |
| --- | --- |
| **De** **statusupdate** **is** **kort**. | The status update is brief. |
| **De** **voortgang** **is** **goed**, **maar** **de** **werkdruk** **is** **hoog**. | Progress is good, but workload is high. |
| **Dit** **heeft** **prioriteit**. | This has priority. |
| **De** **deadline** **blijft** **staan**. | The deadline remains. |

From [B1 Lesson 13](/notes/b1/13-nuanced-opinions): **Naar** **mijn** **mening** **moeten** **wij** **de** **prioriteit** **verschuiven**.

### Decisions and voting

| Dutch | English |
| --- | --- |
| **Wij** **nemen** **een** **besluit**. | We make a decision. |
| **De** **meerderheid** **is** **voor**. | The majority is in favour. |
| **Het** **besluit** **is** **unaniem**. | The decision is unanimous. |
| **De** **stemming** **is** **morgen**. | The vote is tomorrow. |

| Dutch | English |
| --- | --- |
| **Daar** **ben** ik **het** **mee** **eens**. | I agree with that. |
| **Ik** **zie** **dat** **anders**; **kunnen** **wij** **de** **middenweg** **kiezen**? | I see that differently; can we choose the middle ground? |

### Writing a motivatiebrief

Combine [B1 Lesson 16](/notes/b1/16-formal-letters-and-emails) with job vocabulary from A2:

| Block | Dutch |
| --- | --- |
| Opening | **Geachte** **mevrouw** **Jansen**, |
| Reason | **Hierbij** **solliciteer** ik **op** **de** **vacature** **als** **projectmedewerker**. |
| Motivation | **Ik** **ben** **geïnteresseerd** **in** **deze** **functie** **omdat** … |
| Experience | **Mijn** **werkervaring** **omvat** … |
| Skills | **Mijn** **sterke** **punt** **is** … |
| Close | **Graag** **nodig** ik **u** **uit** **voor** **een** **gesprek**. **Met** **vriendelijke** **groet**, |

| Dutch | English |
| --- | --- |
| **Bijgevoegd** **vindt** **u** **mijn** **cv** **en** **motivatiebrief**. | Attached you will find my CV and cover letter. |
| **De** **functieomschrijving** **sluit** **aan** **bij** **mijn** **ervaring**. | The job description matches my experience. |
| **Ik** **voldoe** **aan** **de** **functie-eisen**. | I meet the job requirements. |

**Solliciteren op** + **vacature**. **Solliciteren als** + profession (no **de**): **Ik solliciteer als accountant**.

### Sollicitatiegesprek: common questions

| Question | Strong answer frame |
| --- | --- |
| **Waarom** **wilt** **u** **bij** **ons** **werken**? | **Ik** **ben** **geïnteresseerd** **in** … **omdat** … |
| **Wat** **zijn** **uw** **sterke** **punten**? | **Mijn** **sterke** **punt** **is** … |
| **Wat** **is** **uw** **zwakte**? | **Een** **zwakte** **is** …; **ik** **werk** **eraan** **door** … |
| **Waar** **ziet** **u** **uzelf** **over** **vijf** **jaar**? | **Over** **vijf** **jaar** **wil** ik … |
| **Wat** **zijn** **uw** **salarisverwachtingen**? | **Ik** **sluit** **aan** **bij** **de** **arbeidsvoorwaarden**. |

| Dutch | English |
| --- | --- |
| **Het** **sollicitatiegesprek** **is** **volgende** **week** **dinsdag**. | The job interview is next Tuesday. |
| **De** **sollicitatieprocedure** **duurt** **twee** **weken**. | The application procedure takes two weeks. |
| **Bent** **u** **geschikt** **voor** **deze** **rol**? | Are you suitable for this role? |

Use **u** with HR and interview panels. **Ik zou het op prijs stellen als u …** from B1 Lesson 5.

### After the interview: outcomes

| Dutch | English |
| --- | --- |
| **U** **bent** **aangenomen**. / **U** **wordt** **aangenomen**. | You are hired. |
| **De** **aanwerving** **volgt** **na** **de** **proeftijd**. | Hiring follows after the probation period. |
| **U** **komt** **in** **aanmerking** **voor** **de** **shortlist**. | You are shortlisted. |
| **Helaas** **zijn** **wij** **een** **andere** **kandidaat** **gaan** **aanstellen**. | Unfortunately we hired another candidate. |

| Dutch | English |
| --- | --- |
| **De** **proeftijd** **is** **twee** **maanden**. | The probation period is two months. |
| **De** **arbeidsovereenkomst** **wordt** **vandaag** **getekend**. | The employment contract is signed today. |
| **Uw** **indiensttreding** **is** **1** **september**. | Your start date is 1 September. |
| **De** **arbeidsvoorwaarden** **staan** **in** **het** **contract**. | Employment terms are in the contract. |

### Sample motivatiebrief excerpt

**Geachte** **mevrouw** **De** **Vries**,

**Hierbij** **solliciteer** ik **op** **de** **vacature** **als** **projectmedewerker** **bij** **uw** **organisatie**. **Naar** **aanleiding** **van** **de** **functieomschrijving** **ben** ik **ervan** **overtuigd** **dat** **mijn** **werkervaring** **en** **competenties** **goed** **aansluiten**.

**Mijn** **sterke** **punt** **is** **gestructureerd** **werken** **onder** **druk**. **In** **mijn** **vorige** **functie** **was** **de** **werkdruk** **hoog**, **maar** **de** **voortgang** **bleef** **op** **schema**.

**Bijgevoegd** **vindt** **u** **mijn** **cv**. **Graag** **verneem** ik **of** **wij** **een** **sollicitatiegesprek** **kunnen** **plannen**.

**Met** **vriendelijke** **groet**,

Sam **Jansen**

<Callout title="Pronunciation Focus" type="info">
  **Sollicitatiegesprek** has five syllables: **so-li-ci-ta-tie-ges-prek**. Stress **VER-ga-de-ring** and **MO-ti-va-tie-brief**. Practise: **Hierbij solliciteer ik op de vacature als projectmedewerker.** See [Word Stress](/notes/pronunciation/03-word-stress).
</Callout>

## Example Conversation

<ShadCnCard className="my-4">
  <CardHeader>
    <CardTitle className="text-lg">Teamoverleg en sollicitatie</CardTitle>
    <CardDescription>Mila chairs a short meeting; Sam mentions a sollicitatiegesprek afterwards.</CardDescription>
  </CardHeader>

  <CardContent>
    **In Dutch:**

    > **Mila:** **Laten** **we** **beginnen**. **Het** **eerste** **agendapunt** **is** **de** **stand** **van** **zaken**. **Sam**, **kun** **je** **een** **korte** **statusupdate** **geven**?
    >
    > **Sam:** **De** **voortgang** **is** **goed**, **maar** **de** **werkdruk** **is** **hoog**. **Dit** **project** **heeft** **prioriteit**.
    >
    > **Mila:** **Daar** **ben** ik **het** **mee** **eens**. **Terugkomen** **op** **het** **vorige** **actiepunt**: **is** **het** **rapport** **klaar**?
    >
    > **Sam:** **Bijna**. **Het** **actiepunt** **voor** **mij** **is** **het** **vrijdag** **opleveren**. **Ik** **volg** **de** **opvolging** **op**.
    >
    > **Mila:** **Prima**. **Besluitvorming**: **wij** **wachten** **met** **de** **volgende** **fase**. **Zijn** **er** **nog** **vragen**? **Nee**? **Laten** **we** **afronden**.
    >
    > **Sam:** **Mag** ik **nog** **iets** **zeggen**? **Ik** **heb** **volgende** **week** **een** **sollicitatiegesprek**. **Ik** **heb** **mijn** **motivatiebrief** **en** **cv** **al** **gestuurd**.
    >
    > **Mila:** **Mooi**. **Waarom** **wil** **je** **daar** **werken**?
    >
    > **Sam:** **De** **functieomschrijving** **sluit** **aan** **bij** **mijn** **werkervaring**. **Mijn** **sterke** **punt** **is** **samenwerken** **onder** **druk**.

    **In English:**

    > **Mila:** Let's begin. The first agenda item is the state of affairs. Sam, can you give a short status update?
    >
    > **Sam:** Progress is good, but workload is high. This project has priority.
    >
    > **Mila:** I agree with that. Returning to the previous action point: is the report ready?
    >
    > **Sam:** Almost. My action point is to deliver it Friday. I will follow up.
    >
