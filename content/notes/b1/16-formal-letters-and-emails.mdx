## Introduction

You can already book an appointment by email with **Geachte** and **Met vriendelijke groet** from [A2 Lesson 19](/notes/a2/19-appointments). B1 expands that skeleton into **full professional correspondence**: complaints, confirmations, references to earlier messages, attachments, and the polite **u** register throughout. After this lesson you can write a formal email to a municipality, employer, or client without sounding like a translation from English.

## Quick Review

From [A2 Lesson 19](/notes/a2/19-appointments): **Geachte mevrouw Jansen**, **Met vriendelijke groet**, **Zou u … kunnen?**, **de onderwerpregel**. From [B1 Lesson 5](/notes/b1/05-zou-politeness-and-advice): **Zou u kunnen**, **Ik zou het op prijs stellen**. From [B1 Lesson 10](/notes/b1/10-indirect-speech): reporting and formal **melden**. From [B1 Lesson 3](/notes/b1/03-waar-plus-preposition): formal **waar** + preposition in written Dutch. Today you build **complete letters and emails**.

## Grammar and Language Explanation

### Formal vs informal: choose your register

Dutch correspondence has a clear **ladder**. Pick one rung and stay on it.

| Register | Opening | Closing | Request |
| --- | --- | --- | --- |
| **Formeel** (official) | **Geachte** **mevrouw** … | **Met** **vriendelijke** **groet**, | **Zou** **u** … **kunnen**? |
| **Zakelijk** (colleague) | **Beste** … | **Met** **collegiale** **groet**, | **Zou** **je** … **kunnen**? |
| **Informeel** | **Hoi** … | **Groetjes**, | **Kun** **je** …? |

| Dutch | English |
| --- | --- |
| **Geachte** **heer** **Bakker**, | Dear Mr Bakker, |
| **Geachte** **mevrouw** **Jansen**, | Dear Ms Jansen, |
| **Geachte** **heer/mevrouw**, | Dear Sir/Madam, |
| **Beste** **collega's**, | Dear colleagues, |

Very formal **letters** (not everyday email) may end with **Hoogachtend**, or **Met** **hoogachting**,. Email at work almost always uses **Met** **vriendelijke** **groet**,.

### The anatomy of a Dutch business email

| Block | Example | Note |
| --- | --- | --- |
| **Onderwerp** | **Betreft:** **uw** **aanvraag** **nr.** **2024-118** | **Betreft** or **Onderwerp** |
| **Aanhef** | **Geachte** **mevrouw** **De** **Vries**, | comma after name |
| **Reden** | **Naar** **aanleiding** **van** **uw** **e-mail** **van** **3** **juni** … | links to prior message |
| **Kern** | **Hierbij** **bevestig** ik **de** **ontvangst** **van** **uw** **aanvraag**. | main message |
| **Verzoek** | **Graag** **verneem** ik **of** **u** **akkoord** **gaat**. | polite ask |
| **Bijlage** | **Zie** **bijlage** **voor** **het** **formulier**. | attachment |
| **Afsluiting** | **Met** **vriendelijke** **groet**, | standard close |
| **Naam** | Mila **de** **Vries** | full name in formal mail |

| Dutch | English |
| --- | --- |
| **Betreft:** **schriftelijke** **bevestiging** | Re: written confirmation |
| **Ter** **attentie** **van** **de** **afdeling** **Financiën** | For the attention of the Finance department |
| **Hierbij** **stuur** ik **u** **de** **gevraagde** **informatie**. | Herewith I send you the requested information. |
| **Alvast** **bedankt** **voor** **uw** **medewerking**. | Thanks in advance for your cooperation. |

### Openings beyond Geachte

You know **Geachte** from A2. B1 adds nuance:

| Opening | When |
| --- | --- |
| **Geachte** **mevrouw** / **heer** + name | default formal |
| **Geachte** **heer/mevrouw**, | name unknown |
| **Beste** + first name | colleague, semi formal |
| **Beste** **mevrouw** **Jansen**, | warmer but still polite |
| **Hartelijke** **groet** as **closing**, not opening | do not open with **Hartelijke groet** |

| Dutch | English |
| --- | --- |
| **Beste** **Sam**, | Dear Sam, |
| **Beste** **collega's**, | Dear colleagues, |
| **Geachte** **commissie**, | Dear committee, |

### Closings: groet not groeten

| Closing | Register |
| --- | --- |
| **Met** **vriendelijke** **groet**, | standard formal email |
| **Met** **hoogachting**, | very formal letter |
| **Hoogachtend**, | very formal, traditional |
| **Hartelijke** **groet**, | warm semi formal |
| **Met** **collegiale** **groet**, | between colleagues |
| **Groetjes**, | informal only |

| Dutch | English |
| --- | --- |
| **Met** **vriendelijke** **groet**, **Mila** **de** **Vries** | Kind regards, Mila de Vries |
| **Alvast** **hartelijk** **dank** **en** **met** **vriendelijke** **groet**, | Many thanks in advance and kind regards, |
| **Met** **vriendelijke** **groet**, **namens** **het** **team** | Kind regards, on behalf of the team |

**Groet** (singular) is the norm in **Met vriendelijke groet**. **Groeten** (plural) appears sometimes but singular is safer.

### Referring to earlier contact

Professional mail almost always **anchors** to context:

| Chunk | Example |
| --- | --- |
| **In** **verband** **met** | **In** **verband** **met** **uw** **aanvraag** … |
| **Met** **betrekking** **tot** | **Met** **betrekking** **tot** **ons** **gesprek** … |
| **Betreffende** | **Betreffende** **het** **contract** … |
| **Inzake** | **Inzake** **uw** **klacht** … (formal) |
| **Ten** **aanzien** **van** | **Ten** **aanzien** **van** **de** **factuur** … |
| **Naar** **aanleiding** **van** **uw** **schrijven** | **Naar** **aanleiding** **van** **uw** **schrijven** **van** **12** **mei** … |
| **Met** **verwijzing** **naar** | **Met** **verwijzing** **naar** **referentienummer** **2024-118** … |

| Dutch | English |
| --- | --- |
| **Uw** **schrijven** **van** **3** **juni** **hebben** **wij** **ontvangen**. | We have received your letter of 3 June. |
| **Met** **verwijzing** **naar** **uw** **e-mail** **bevestig** ik **de** **afspraak**. | With reference to your email I confirm the appointment. |
| **Het** **betreft** **de** **schriftelijke** **melding** **van** **gisteren**. | It concerns the written notice from yesterday. |

**Uw schrijven** = your letter (formal noun). **Uw e-mail** is fine in modern correspondence.

### Polite requests in writing

Combine **u** with **zou** from [B1 Lesson 5](/notes/b1/05-zou-politeness-and-advice):

| Pattern | Example |
| --- | --- |
| **Zou** **u** … **kunnen**? | **Zou** **u** **het** **formulier** **kunnen** **invullen**? |
| **Graag** **verneem** ik | **Graag** **verneem** ik **uw** **reactie** **voor** **vrijdag**. |
| **Ik** **zou** **het** **op** **prijs** **stellen** **als** | **Ik** **zou** **het** **op** **prijs** **stellen** **als** **u** **vandaag** **reageert**. |
| **Ik** **verzoek** **u** **vriendelijk** **om** | **Ik** **verzoek** **u** **vriendelijk** **om** **de** **bijlage** **te** **lezen**. |

| Dutch | English |
| --- | --- |
| **Zou** **u** **de** **ontvangst** **kunnen** **bevestigen**? | Could you confirm receipt? |
| **Graag** **ontvang** ik **uw** **schriftelijke** **reactie**. | I would like to receive your written response. |
| **Ik** **zou** **het** **op** **prijs** **stellen** **als** **u** **contact** **opneemt**. | I would appreciate it if you would get in touch. |

### Attachments and enclosures

| Dutch | English |
| --- | --- |
| **Bijgaand** vindt u **het** **formulier**. | Enclosed you will find the form. |
| **Bijgevoegd** **vindt** u **de** **offerte**. | Attached you will find the quotation. |
| **In** **de** **bijlage** **vindt** u **het** **rapport**. | In the attachment you will find the report. |
| **Zie** **bijlage**. | See attachment. |
| **In** **kopie** **aan** **de** **afdeling** **HR** | Cc to the HR department |

| Dutch | English |
| --- | --- |
| **Hierbij** **stuur** ik **u** **per** **e-mail** **de** **gevraagde** **documenten**. | Herewith I email you the requested documents. |
| **Het** **aanvraagformulier** **is** **bijgevoegd**. | The application form is attached. |
| **Graag** **ontvang** ik **een** **schriftelijke** **bevestiging**. | I would like to receive written confirmation. |

### Letters vs email: post and signature

| Dutch | English |
| --- | --- |
| **Per** **post** | by post |
| **Per** **aangetekende** **post** | by registered post |
| **Per** **e-mail** | by email |
| **Getekend**, **Mila** **de** **Vries** | Signed, Mila de Vries |
| **De** **handtekening** **staat** **onderaan**. | The signature is at the bottom. |

| Dutch | English |
| --- | --- |
| **Ondertekend** **op** **3** **juni** **2024** | Signed on 3 June 2024 |
| **De** **klachtenbrief** **stuur** ik **per** **aangetekende** **post**. | I send the complaint letter by registered post. |
| **Het** **bezwaarschrift** **moet** **schriftelijk** **worden** **ingediend**. | The objection must be submitted in writing. |

### Sample formal email

**Onderwerp:** **Betreft:** **aanvraag** **vergunning**

**Geachte** **mevrouw** **De** **Vries**,

**Naar** **aanleiding** **van** **uw** **e-mail** **van** **gisteren** **bevestig** ik **hierbij** **de** **ontvangst** **van** **uw** **aanvraag**. **Met** **verwijzing** **naar** **referentienummer** **2024-118** **verzoek** ik **u** **het** **bijgevoegde** **formulier** **in** **te** **vullen**.

**Graag** **verneem** ik **voor** **1** **juli** **of** **u** **akkoord** **gaat** **met** **de** **voorgestelde** **planning**. **Alvast** **bedankt** **voor** **uw** **medewerking**.

**Met** **vriendelijke** **groet**,

Sam **Jansen**

### Types of formal writing at B1

| Type | Dutch | Typical use |
| --- | --- | --- |
| Request | **het** **verzoekschrift** | municipality, formal ask |
| Objection | **het** **bezwaarschrift** | appeal against decision |
| Complaint | **de** **klachtenbrief** | service complaint |
| Quote | **de** **offerte** | business proposal |
| Reminder | **de** **herinneringsmail** | unpaid invoice, deadline |
| Confirmation | **de** **schriftelijke** **bevestiging** | receipt, agreement |

<Callout title="Pronunciation Focus" type="info">
  **Geachte** stresses **GE-ach-te**. In **Met vriendelijke groet**, link **vriendelijke-groet**. Read your email aloud before sending: Dutch business writing sounds confident when rhythm is smooth. See [Connected Speech](/notes/pronunciation/05-connected-speech).
</Callout>
