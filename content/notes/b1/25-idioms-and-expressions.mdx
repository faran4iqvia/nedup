## Introduction

You know grammar, particles, and stories. Real Dutch also runs on **fixed phrases**: **Het maakt niet uit**, **Geen zorgen**, **Dat klopt**, **Het komt goed**, **Graag gedaan**. Many chunks are **figuurlijk**: **Ik ben de draad kwijt** does not mean you lost cotton thread. After this lesson you can recognise common **uitdrukkingen**, respond naturally in social talk, and use classic Dutch idioms at B1 level.

## Grammar and Language Explanation

### Uitdrukking, gezegde, spreekwoord, idioom

Dutch groups non literal language in several terms:

| Dutch | English | Example |
| --- | --- | --- |
| **de** **uitdrukking** | expression / phrase | **Het** **maakt** **niet** **uit** |
| **het** **gezegde** | saying | **Zo** **is** **het** |
| **het** **spreekwoord** | proverb | **Oefening** **baart** **kunst** |
| **de** **idioom** | idiom | **de** **draad** **kwijt** |
| **de** **vaste** **uitdrukking** | fixed expression | **aan** **tafel** |

| Dutch | English |
| --- | --- |
| **Leer** **uitdrukkingen** **als** **hele** **blokken**. | Learn expressions as whole blocks. |
| **De** **betekenis** **is** **vaak** **figuurlijk**. | The meaning is often figurative. |
| **Niet** **elk** **woord** **vertalen**. | Do not translate every word. |

### Letterlijk vs figuurlijk

| Type | Dutch | English |
| --- | --- | --- |
| **Letterlijk** | **Ik** **ben** **te** **laat**. | I am late. |
| **Figuurlijk** | **Ik** **ben** **de** **draad** **kwijt**. | I lost the thread. |
| **Letterlijk** | **Hij** **stak** **zijn** **handen** **uit** **de** **mouwen**. | He pulled his hands from his sleeves. |
| **Figuurlijk** | **Wij** **staken** **de** **handen** **uit** **de** **mouwen**. | We got stuck in / pitched in. |

When someone uses an idiom, ask: **Is** **dit** **letterlijk** **of** **figuurlijk**?

### Everyday reassurance and agreement

These phrases appear dozens of times per week:

| Dutch | English | When |
| --- | --- | --- |
| **het** **maakt** **niet** **uit** | it does not matter | minor problem |
| **geen** **zorgen** | no worries | calming someone |
| **maak** **je** **geen** **zorgen** | don't worry | direct comfort |
| **dat** **klopt** | that's correct | agreement |
| **het** **komt** **goed** | it will be fine | reassurance |
| **het** **valt** **mee** | it's not so bad | softening bad news |
| **het** **valt** **tegen** | it's disappointing | negative surprise |
| **niet** **erg** | not a big deal | downplaying |
| **geen** **punt** | no issue | informal OK |
| **in** **orde** | in order / fine | all good |
| **het** **zit** **goed** | it's fine | situation OK |
| **het** **is** **geregeld** | it's arranged | problem solved |
| **het** **schiet** **op** | it's progressing | work moving |

| Dutch | English |
| --- | --- |
| **Sorry**, **ik** **ben** **te** **laat**. **Het** **maakt** **niet** **uit**, **hoor**. | Sorry I'm late. It does not matter. |
| **Geen** **zorgen**, **het** **komt** **goed**. | No worries, it will be fine. |
| **Dat** **klopt**, **zo** **is** **het**. | That's right, that's how it is. |
| **Het** **valt** **mee**, **het** **had** **erger** **kunnen** **zijn**. | It's not so bad, it could have been worse. |

From [A1 Lesson 3](/notes/a1/03-negation): **geen probleem** is close to **geen punt** and **niet erg**.

### Indifference and low concern

| Dutch | English |
| --- | --- |
| **het** **is** **me** **om** **het** **even** | I do not mind / it's all the same to me |
| **het** **kan** **me** **niet** **schelen** | I do not care |
| **het** **trekt** **zich** **niets** **aan** | it does not bother (him/her) |
| **het** **lost** **zich** **op** | it will sort itself out |
| **het** **komt** **vanzelf** | it will come naturally |
| **het** **lukt** **wel** | it will work out |
| **het** **gaat** **wel** | I'm managing / it's OK |
| **het** **zal** **wel** **meevallen** | it will probably be fine |

| Dutch | English |
| --- | --- |
| **Welke** **film**? **Het** **is** **me** **om** **het** **even**. | Which film? I do not mind. |
| **Maak** **je** **geen** **zorgen**, **het** **lost** **zich** **op**. | Don't worry, it will sort itself out. |
| **Moeilijk** **examen**? **Het** **zal** **wel** **meevallen**. | Difficult exam? It will probably be OK. |

### Thanks, wishes, and social formulas

| Dutch | English |
| --- | --- |
| **graag** **gedaan** | you're welcome |
| **van** **harte** | wholeheartedly / sincere |
| **veel** **succes** | good luck |
| **veel** **plezier** | enjoy / have fun |
| **sterkte** | strength (sympathy) |
| **gefeliciteerd** | congratulations |
| **proficiat** | congratulations (formal) |
| **tot** **straks** | see you soon |
| **tot** **zo** | see you in a bit |
| **slaap** **lekker** | sleep well |
| **proost** | cheers |
| **gezondheid** | cheers / bless you |
| **aan** **tafel** | come to the table / dinner is served |
| **fijn** **weekend** | have a nice weekend |

| Dutch | English |
| --- | --- |
| **Dank** **je** **wel**! **Graag** **gedaan**. | Thank you! You're welcome. |
| **Van** **harte** **gefeliciteerd**! | Heartfelt congratulations! |
| **Veel** **succes** **morgen**! | Good luck tomorrow! |
| **Proost**! **Gezondheid**! | Cheers! |

You know **smakelijk** and **welterusten** from A0. **Aan tafel** calls the family to eat.

### Apology and regret chunks

| Dutch | English |
| --- | --- |
| **het** **spijt** **me** | I'm sorry |
| **het** **spijt** **me** **dat** … | I'm sorry that … (B1 L7) |

| Dutch | English |
| --- | --- |
| **Het** **spijt** **me**, **ik** **ben** **te** **laat**. | I'm sorry, I'm late. |
| **Het** **zit** **me** **niet** **lekker** | something bothers me (gut feeling) |
| **Het** **zit** **me** **dwars** | something annoys me / I resent it |
| **Ergens** **mee** **zitten** | to be bothered by something |

| Dutch | English |
| --- | --- |
| **Het** **zit** **me** **dwars** **dat** **hij** **niet** **belde**. | It annoys me that he did not call. |
| **Ik** **zit** **ergens** **mee**, **maar** **ik** **wil** **er** **niet** **over** **praten**. | Something is bothering me, but I do not want to talk about it. |

From [B1 Lesson 20](/notes/b1/20-feelings-and-relationships): **Ik maak me zorgen om je** is deeper than **geen zorgen** (reassurance to the other person).

### Confusion and stress idioms

| Dutch | English |
| --- | --- |
| **de** **draad** **kwijt** | to lose the thread (lose track) |
| **de** **kluts** **kwijt** | to be flustered / lose the plot |
| **in** **de** **war** **zijn** | to be confused |
| **uit** **je** **bol** **gaan** | to lose one's temper |
| **door** **het** **lint** **gaan** | to snap / go off the rails |
| **van** **de** **kaart** **zijn** | to be out of it / wiped out |
| **alles** **op** **een** **rijtje** **hebben** | to have it all clear / organised |

| Dutch | English |
| --- | --- |
| **Sorry**, **ik** **ben** **even** **de** **draad** **kwijt**. | Sorry, I lost the thread for a moment. |
| **Na** **het** **nieuws** **was** **ik** **de** **kluts** **kwijt**. | After the news I was all flustered. |
| **Ik** **moet** **alles** **op** **een** **rijtje** **hebben** **voor** **het** **gesprek**. | I need to get everything straight before the meeting. |

### Classic Dutch action idioms

| Dutch | English |
| --- | --- |
| **de** **handen** **uit** **de** **mouwen** **steken** | to pitch in / get stuck in |
| **de** **kat** **uit** **de** **boom** **kijken** | to wait and see what others do |
| **het** **ijzer** **smeden** | to strike while the iron is hot |
| **in** **een** **mum** **van** **tijd** | in no time |
| **in** **een** **oogwenk** | in the blink of an eye |
| **zo** **meteen** | very soon / right away |
| **uit** **het** **hoofd** **kennen** | to know by heart |
| **bij** **de** **pinken** **zijn** | to be on the ball |
| **aan** **de** **hand** **zijn** | to be going on / happening |

| Dutch | English |
| --- | --- |
| **Laten** **we** **de** **handen** **uit** **de** **mouwen** **steken**. | Let's get stuck in. |
| **Hij** **kijkt** **de** **kat** **uit** **de** **boom** **en** **wacht** **af**. | He waits and sees what happens. |
| **Nu** **het** **ijzer** **smeden**, **terwijl** **het** **heet** **is**. | Strike while the iron is hot. |
| **Het** **was** **in** **een** **mum** **van** **tijd** **geregeld**. | It was sorted in no time. |

From [B1 Lesson 21](/notes/b1/21-dutch-culture-and-directness): **de mouwen opstropen** overlaps in meaning with **de handen uit de mouwen steken**. Both mean _get to work_.

### Cause and dependency chunks

| Dutch | English |
| --- | --- |
| **het** **ligt** **aan** | it is due to / it depends on |
| **het** **hangt** **af** **van** | it depends on |
| **zo** **is** **het** | that's how it is |

| Dutch | English |
| --- | --- |
| **Het** **ligt** **aan** **het** **weer**. | It's because of the weather. |
| **Het** **hangt** **af** **van** **het** **budget**. | It depends on the budget. |
| **Zo** **is** **het** **nu** **eenmaal**. | That's just how it is. |

### How to learn fixed phrases

| Strategy | Action |
| --- | --- |
| Block learning | learn **het maakt niet uit** as one unit |
| Context tags | tag each phrase: reassurance / thanks / anger |
| Response pairs | **Dank je** → **Graag gedaan** · **Sorry** → **Geen zorgen** |
| Figuurlijk check | draw literal vs real meaning in your notebook |

| Dutch | English |
| --- | --- |
| **Als** **je** **twijfelt**, **vraag**: **Is** **dit** **figuurlijk**? | If you doubt, ask: is this figurative? |
| **Gebruik** **één** **nieuwe** **uitdrukking** **per** **dag**. | Use one new expression per day. |

<Callout title="Pronunciation Focus" type="info">
  Fixed phrases run as one rhythm: **het-MAAKT-niet-UIt**, **geen-ZOR-gen**, **het-VALT-mee**. **Proost** and **gezondheid** at the table are short and clear. Practise chains: **Dank je wel. Graag gedaan. Geen zorgen. Het komt goed.** See [Connected Speech](/notes/pronunciation/05-connected-speech).
</Callout>

## Common Mistakes for English Speakers

<ShadCnCard className="my-4">
  <CardHeader>
    <CardTitle className="text-lg">Word for word translation</CardTitle>
  </CardHeader>

  <CardContent>
    ❌ **It** **falls** **me** **along** (for **het valt mee**).

    ✅ **Het** **valt** **mee** = it's not so bad.

    Learn the **whole** chunk, not each word in isolation.
  </CardContent>
</ShadCnCard>

<ShadCnCard className="my-4">
  <CardHeader>
    <CardTitle className="text-lg">De draad kwijt as sewing</CardTitle>
  </CardHeader>

  <CardContent>
    ❌ Only imagining thread.

    ✅ **Ik** **ben** **de** **draad** **kwijt** in meetings, stories, and explanations = I lost track.

    Context tells you literal vs **figuurlijk**.
  </CardContent>
</ShadCnCard>

<ShadCnCard className="my-4">
  <CardHeader>
    <CardTitle className="text-lg">Geen zorgen vs zich zorgen maken</CardTitle>
  </CardHeader>

  <CardContent>
    ❌ Using them as exact opposites in the same sentence logic.

    ✅ **Geen** **zorgen** = reassurance to someone else. **Ik** **maak** **me** **zorgen** = I worry (B1 L20).

    **Maak** **je** **geen** **zorgen** tells the other person not to worry.
  </CardContent>
</ShadCnCard>

<ShadCnCard className="my-4">
  <CardHeader>
    <CardTitle className="text-lg">Het kan me niet schelen in polite talk</CardTitle>
  </CardHeader>

  <CardContent>
    ❌ **Het** **kan** **me** **niet** **schelen** when you mean _I don't mind_ in a friendly offer.

    ✅ **Het** **is** **me** **om** **het** **even** = neutral indifference. **Het** **kan** **me** **niet** **schelen** sounds colder / I don't care.
  </CardContent>
</ShadCnCard>
