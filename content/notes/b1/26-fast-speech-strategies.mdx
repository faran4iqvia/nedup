## Introduction

You know correct Dutch on paper. In cafés, meetings, and trains, people speak **fast**: words **link**, **de** and **het** shrink, **niet** becomes **nie**, and whole clauses run together. Missing one word can feel like missing everything. After this lesson you can use **listening strategies**, ask for **herhaling** without shame, and practise **shadowen** so rapid Dutch stops sounding like noise.

## Grammar and Language Explanation

### Why Dutch feels fast

| Factor | Effect |
| --- | --- |
| **de spreeksnelheid** | natives use fewer pauses between words |
| **het linken** | boundaries blur: **dat ik** → **da-tik** |
| **de reductie** | function words shorten |
| **het wegvallen** | some sounds drop in casual speech |
| **de versnelde spraak** | particles and pronouns shrink |

| Dutch | English |
| --- | --- |
| **In** **de** **spreektaal** **klinkt** **alles** **sneller**. | In spoken language everything sounds faster. |
| **Je** **hoeft** **niet** **elk** **woord** **te** **horen**. | You do not need to hear every word. |
| **Luister** **naar** **kernwoorden**. | Listen for key words. |

### Inhoudswoorden vs functiewoorden

Train your ear to separate **content** from **function** words:

| Type | Dutch term | Examples | Stress |
| --- | --- | --- | --- |
| Content | **de** **inhoudswoord** / **het** **hoofdwoord** | **werk**, **trein**, **morgen**, **regent** | strong |
| Function | **de** **functiewoord** | **de**, **het**, **een**, **ik**, **dat**, **niet** | weak |

| Dutch | English |
| --- | --- |
| **Inhoudswoorden** **blijven** **duidelijk**. | Content words stay clear. |
| **Functiewoorden** **verkorten** **vaak**. | Function words often shorten. |
| **Zoek** **de** **sleutelwoord** **in** **de** **zin**. | Find the keyword in the sentence. |

When you catch **trein**, **morgen**, and **vertraging**, you already understand much of **De trein heeft morgen vertraging**, even if **de** and **het** were unclear.

### Linking and samensmelting

From A2 pronunciation work, these links appear constantly:

| Written | Linked feel | Meaning |
| --- | --- | --- |
| **is het** | i-shet | is it |
| **op een** | o-peen | on a |
| **in de** | in-de | in the |
| **aan het** | aan-tet | busy doing |
| **dat ik** | da-tik | that I |
| **niet waar** | nie-waar | isn't it |

| Dutch | English |
| --- | --- |
| **de** **verbindingsklank** | linking sound between words |
| **de** **klinkerverbinding** | vowel/consonant link |
| **de** **samensmelting** | merging of sounds |
| **de** **assimilatie** | sound change at boundary |

| Dutch | English |
| --- | --- |
| **Ik** **ben** **aan** **het** **werken** **klinkt** **als** **één** **keten**. | I am working sounds like one chain. |
| **Omdat** **ik** **tijd** **heb**, **kom** **ik** **zonder** **pauze** **na** **de** **komma**. | Because I have time, I come without pausing after the comma. |

B1 sentences are longer. Practise chains from your grammar:

| Sentence | Link to practise |
| --- | --- |
| **Ik** **denk** **dat** **het** **morgen** **regent**. | **denk-dat** |
| **Omdat** **de** **trein** **vertraging** **had**, **kwam** **ik** **te** **laat**. | **omdat-de** |
| **Als** **ik** **tijd** **heb**, **bel** **ik** **je**. | **als-ik**, **bel-ik-je** |

### Reductions in informal Dutch

| Written | Reduced / casual | Example |
| --- | --- | --- |
| **niet** | **nie** | **Ik kom nie.** |
| **dat** | **'t** / **da** | **'t is goed** |
| **even** | **eve** | **Wacht eve.** |
| **misschien** | **missche** | **Missche morgen.** |
| **gewoon** | **gewoan** | **Gewoan doen.** |
| **waarschijnlijk** | **waarschijnlek** | **Waarschijnlek wel.** |

| Dutch | English |
| --- | --- |
| **de** **verkorte** **vorm** | shortened form |
| **de** **verkorting** | shortening |
| **de** **informele** **uitspraak** | informal pronunciation |
| **het** **Randstad-Nederlands** | Randstad Dutch (Amsterdam, Rotterdam, Utrecht, Den Haag) |

You do **not** have to speak this way immediately. **Recognise** reductions when listening.

### The schwa and weak vowels

Unstressed syllables often use **de schwa** (neutral **uh** sound):

| Word | Full careful | Fast |
| --- | --- | --- |
| **de** | duh | very short |
| **het** | ut / ht | very short |
| **een** | un | very short |
| **je** | juh | very short |

| Dutch | English |
| --- | --- |
| **de** **zwakke** **klinker** | weak vowel |
| **de** **schwa** | schwa (neutral vowel) |
| **de** **vloeiende** **uitspraak** | fluent pronunciation |

Stress stays on **inhoudswoorden**. From [Word Stress](/notes/pronunciation/03-word-stress): **VER-gadering**, not **ver-GAA-dering**.

### Listening strategies at B1

When speech is too fast, use **luisterstrategieën**:

| Strategy | Dutch term | What to do |
| --- | --- | --- |
| **Gist** | **het** **gist** | catch topic + attitude, not every word |
| **Context** | **de** **luistercontext** / **de** **contexthint** | use place, speaker, situation |
| **Keyword** | **de** **kernwoord** / **de** **sleutelwoord** | listen for stressed words |
| **Guess** | **de** **gokstrategie** / **het** **raden** | guess missing word, confirm later |
| **Fragment** | **het** **luisterfragment** | understand one chunk, then the next |
| **Half hear** | **half** **verstaan** / **deels** **verstaan** | accept partial understanding |

| Dutch | English |
| --- | --- |
| **Ik** **heb** **het** **gemiste** **woord** **niet** **gehoord**, **maar** **ik** **snap** **de** **zin** **wel**. | I did not hear the missing word, but I get the sentence. |
| **Gebruik** **de** **context** **om** **te** **raden**. | Use context to guess. |
| **Focus** **op** **woordaccent** **en** **cadans**. | Focus on word stress and rhythm. |

**De luistercompensatie** = compensating when audio is unclear. **De luisterinspanning** = the effort of active listening.

### Recovery phrases (herstelzinnen)

Polite **herstelzinnen** keep conversation going:

| Dutch | English | Formality |
| --- | --- | --- |
| **Kun** **je** **dat** **herhalen**? | Can you repeat that? | neutral |
| **Wat** **zei** **je**? | What did you say? | informal |
| **Nog** **eens** **alsjeblieft**? | Once more, please? | polite |
| **Spreek** **alsjeblieft** **langzamer**. | Please speak more slowly. | polite |
| **Langzamer** **alstublieft**? | Slower, please? | formal |
| **Even** **langzamer**, **alsjeblieft**. | A bit slower, please. | friendly |
| **Ik** **snap** **het** **niet** **helemaal**. | I do not quite get it. | honest |
| **Ik** **heb** **het** **niet** **verstaan**. | I did not catch that. | neutral |
| **Wat** **bedoel** **je**? | What do you mean? | clarification |
| **Hoe** **bedoel** **je** **dat**? | How do you mean that? | deeper clarification |
| **Ik** **ben** **even** **de** **draad** **kwijt**. | I lost the thread for a moment. | B1 L25 |

| Dutch | English |
| --- | --- |
| **Sorry**, **kun** **je** **dat** **herhalen**? **Ik** **hoorde** **het** **laatste** **woord** **niet**. | Sorry, can you repeat that? I did not hear the last word. |
| **Geen** **probleem** **als** **je** **even** **langzamer** **praat**, **hoor**. | No problem if you speak a bit slower. |

From [A1 Lesson 3](/notes/a1/03-negation): **geen probleem** pairs well with recovery requests.

### Practice techniques: shadowen and naspreken

| Technique | Dutch | How |
| --- | --- | --- |
| **Shadowing** | **het** **shadowen** | repeat half a beat behind audio |
| **Repeat after** | **het** **naspreken** | copy speaker phrase by phrase |
| **Listen along** | **het** **meeluisteren** | follow without speaking |
| **Listening exercise** | **de** **luisteroefening** | structured audio practice |

| Dutch | English |
| --- | --- |
| **Shadow** **twee** **minuten** **per** **dag**. | Shadow two minutes per day. |
| **Spreek** **de** **zin** **na** **zonder** **te** **pauzeren** **bij** **dat** **of** **omdat**. | Repeat the sentence without pausing at **dat** or **omdat**. |
| **Luister** **eerst** **op** **gist**, **dan** **opnieuw** **op** **detail**. | Listen first for gist, then again for detail. |

### Repairing misunderstandings

| Dutch | English |
| --- | --- |
| **het** **misverstand** | misunderstanding |
| **de** **verduidelijking** | clarification |
| **de** **verduidelijkingsvraag** | clarification question |
| **de** **luisterherhaling** | listening repeat / replay |

| Dutch | English |
| --- | --- |
| **Even** **checken**: **bedoel** **je** **morgen** **of** **vandaag**? | Quick check: do you mean tomorrow or today? |
| **Dus** **jij** **komt** **niet**, **toch**? | So you are not coming, right? |
| **Ik** **dacht** **dat** **je** **zei** … | I thought you said … |

Confirm guesses. Dutch directness accepts **Wat bedoel je?** and **Dus … toch?**

### Fast speech in real situations

| Situation | Tip |
| --- | --- |
| Train announcement | listen for **spoor**, **vertraging**, **uitval** |
| Colleague at **vrijmibo** | particles + reductions together |
| Podcast | **gist** first; subtitles second pass |
| Shop | short chunks: **Alstublieft**, **Dank u**, numbers |

| Dutch | English |
| --- | --- |
| **Op** **het** **station** **hoor** **je** **korte** **fragmenten**. | At the station you hear short fragments. |
| **In** **de** **kroeg** **is** **de** **luistertempo** **hoog**. | In the pub the listening pace is high. |

  Drill B1 chains aloud: **Ik denk dat het morgen regent** · **Omdat ik tijd heb, kom ik** · **Weet je waar de receptie is?** · **Kun je dat herhalen?** First slow and clear, then one flowing unit. See [Intonation and Rhythm](/notes/pronunciation/04-intonation-and-rhythm).
</Callout>
