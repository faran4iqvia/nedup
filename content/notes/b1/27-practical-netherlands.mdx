## Introduction

You can discuss opinions, work, culture, and feelings in Dutch. **Living** in the Netherlands also means **systems**: register at **de gemeente**, get **een BSN**, arrange **de zorgverzekering**, file **de belastingaangifte**, and know **de huisarts**. These tasks use fixed vocabulary and formal chunks. After this lesson you can navigate the main bureaucratic steps, ask clear questions at **het gemeentehuis**, and understand letters from **de belastingdienst** or **de zorgverzekeraar** at B1 level.

### Inschrijven bij de gemeente and BSN

When you live in the Netherlands, you **register** your address. **Het gemeentehuis** (town hall) or **het stadhuis** handles this.

| Dutch | English |
| --- | --- |
| **de** **adresregistratie** | address registration |
| **de** **BRP** | Personal Records Database |
| **het** **BSN** / **het** **burgerservicenummer** | citizen service number |
| **de** **verhuizing** | move (relocation) |
| **de** **adreswijziging** | address change |
| **de** **verhuismelding** | move notification |

| Dutch | English |
| --- | --- |
| **Ik** **moet** **me** **inschrijven** **bij** **de** **gemeente**. | I need to register at the municipality. |
| **Waar** **kan** **ik** **mijn** **verhuizing** **doorgeven**? | Where can I report my move? |
| **U** **krijgt** **een** **BSN** **na** **inschrijving**. | You receive a BSN after registration. |
| **Het** **BSN** **hebt** **u** **nodig** **voor** **werk** **en** **zorg**. | You need the BSN for work and healthcare. |

From [B1 Lesson 18](/notes/b1/18-education-and-learning): **inschrijven** also means enrol at school. Context decides: **gemeente** vs **universiteit**.

**Moeten** + infinitive is common: **U moet binnen vier weken melden** (you must report within four weeks).

### DigiD: online government

Many services are online. **DigiD** is your login for government websites.

| Dutch | English |
| --- | --- |
| **DigiD** | Dutch government login |
| **de** **inlogcode** | login code |
| **de** **belastingaangifte** | tax return (filing) |
| **online** **regelen** | arrange online |

| Dutch | English |
| --- | --- |
| **Ik** **log** **in** **met** **DigiD**. | I log in with DigiD. |
| **Kunt** **u** **mij** **helpen** **met** **DigiD**? | Can you help me with DigiD? |
| **De** **belastingaangifte** **doe** **ik** **online**. | I file my tax return online. |

### Zorgverzekering and healthcare access

Everyone living in the Netherlands must have **basisverzekering** (basic health insurance).

| Dutch | English |
| --- | --- |
| **de** **zorgverzekering** | health insurance |
| **de** **basisverzekering** | basic insurance (mandatory) |
| **de** **aanvullende** **verzekering** | supplementary insurance |
| **de** **zorgverzekeraar** | health insurer |
| **de** **zorgpolis** | health policy |
| **de** **premie** | premium (monthly payment) |
| **het** **eigen** **risico** | deductible |
| **de** **zorgkosten** | healthcare costs |
| **de** **zorgverplichting** | insurance obligation |
| **de** **eigen** **bijdrage** | personal contribution / co-pay |

| Dutch | English |
| --- | --- |
| **Ik** **zoek** **een** **zorgverzekering**. | I am looking for health insurance. |
| **De** **premie** **betaal** **ik** **per** **maand**. | I pay the premium monthly. |
| **Het** **eigen** **risico** **is** **€385** **per** **jaar**. | The deductible is €385 per year (amount changes). |
| **U** **moet** **zich** **voor** **1** **januari** **verzekeren**. | You must insure yourself before 1 January. |

### De huisarts: your first medical contact

| Dutch | English |
| --- | --- |
| **de** **huisarts** | GP / family doctor |
| **de** **huisartsenpraktijk** | GP practice |
| **de** **huisartsenpost** | out of hours GP centre |
| **de** **tandarts** | dentist |
| **de** **patiëntenregistratie** | patient registration (at GP) |

| Dutch | English |
| --- | --- |
| **Ik** **zoek** **een** **huisarts** **in** **de** **buurt**. | I am looking for a GP in the neighbourhood. |
| **U** **heeft** **een** **verwijzing** **nodig** **voor** **de** **specialist**. | You need a referral for the specialist. |
| **Buiten** **kantooruren** **bel** **je** **de** **huisartsenpost**. | Outside office hours you call the GP centre. |
| **De** **apotheek** **haalt** **u** **medicijnen** **op**. | You pick up medicine at the pharmacy (A1). |

You usually **register** with one GP practice. **De apotheek** dispenses prescriptions.

### Belastingdienst, toeslagen, and benefits

| Dutch | English |
| --- | --- |
| **de** **belastingdienst** | tax authority |
| **de** **inkomstenbelasting** | income tax |
| **de** **belastingteruggave** | tax refund |
| **de** **toeslag** | allowance / benefit top up |
| **de** **huurtoeslag** | rent allowance |
| **de** **zorgtoeslag** | healthcare allowance |
| **de** **kinderbijslag** | child benefit |

| Dutch | English |
| --- | --- |
| **Ik** **doe** **mijn** **belastingaangifte** **voor** **1** **mei**. | I file my tax return before 1 May. |
| **Kom** **ik** **in** **aanmerking** **voor** **huurtoeslag**? | Am I eligible for rent allowance? |

| **De** **belastingteruggave** **komt** **in** **juli**. | The tax refund comes in July. |

**In aanmerking komen voor** = to be eligible for. Useful at **het gemeentehuis** and on government sites.

| Dutch | English |
| --- | --- |
