## Subordinate connectors (verb last)

| Connector | Meaning | Example |
| --- | --- | --- |
| **hoewel** | although | **Hoewel** ik moe **ben**, ga ik uit. |
| **terwijl** | while | **Terwijl** ik kook, luister ik naar muziek. |
| **nadat / voordat** | after / before | **Nadat** ik heb gegeten, bel ik je. |
| **totdat** | until | **Totdat** hij komt, wacht ik. |

❌ **Hoewel ik ben ziek, ik blijf thuis.**\
✅ **Hoewel ik ziek ben, blijf ik thuis.**

## zou and hypotheticals

| Use | Example |
| --- | --- |
| Polite request | **Zou** je me kunnen helpen? |
| Soft advice | Ik **zou** wachten. |
| Unreal present | **Als** ik rijk **was**, **zou** ik reizen. |
| Unreal past | **Als** ik het **had geweten**, **zou** ik **zijn gekomen**. |

## Passive: worden vs zijn

| Type | Formula | Example |
| --- | --- | --- |
| Process (ongoing) | **worden** + participle | Het rapport **wordt** afgerond. |
| Result / state | **zijn** + participle | Het rapport **is** afgerond. |
| Agent | **door** + noun | … **door** het team. |

## Indirect speech (quick)

| Direct | Indirect |
| --- | --- |
| Hij zegt: "Ik kom morgen." | Hij zegt dat hij morgen komt. |
| Hij zei: "Ik kom morgen." | Hij zei dat hij **de volgende dag zou komen**. |
