## Introduction

At A2 you recognised **zijn** + participle as a **completed result**: **Het rapport is gemaakt**, **De deur is gesloten**. B1 Lesson 8 deepened **process** passive with **worden**: **Het rapport wordt gemaakt**. Now you master the **result** side: when to say **is** instead of **wordt**, how **Er zijn fouten gemaakt** works, and how **moet afgerond zijn** expresses deadlines. After this lesson you can switch naturally between _it is being done_ and _it is done_ in Dutch.

## Quick Review

From [B1 Lesson 8](/notes/b1/08-passive-with-worden): **worden** + participle for process; **door**, modals, **zal worden**, subordinate **goedgekeurd moet worden**. From [A2 Lesson 2](/notes/a2/02-perfect-tense-intro): **hebben/zijn** + participle in the perfect tense. From [A2 Lesson 21](/notes/a2/21-passive-intro): **worden** vs **zijn** preview. The **form** of **zijn** passive matches the perfect tense, but the **meaning** is passive result or state.

## Grammar and Language Explanation

### worden vs zijn: the B1 decision

| | **worden** (process) | **zijn** (result) |
| --- | --- | --- |
| Time | now / ongoing | already done |
| Focus | action happening | state after action |
| Example | De brief **wordt** **geschreven**. | De brief **is** **geschreven**. |
| English feel | is being written | has been written / is written |

| Dutch | English |
| --- | --- |
| Het rapport **wordt** **nu** **gemaakt**. | The report is being made now. |
| Het rapport **is** **klaar**. Het **is** **gisteren** **afgerond**. | The report is ready. It was completed yesterday. |
| De deur **wordt** **om** **vijf** uur **gesloten**. | The door is closed at five (process each day). |
| De deur **is** **gesloten**. | The door is closed (state now). |
| De wijziging **wordt** **doorgevoerd**. | The change is being implemented. |
| De wijziging **is** **doorgevoerd**. | The change has been implemented. |

### zijn + participle: the result formula

**subject + zijn + past participle**

| Tense | singular | plural | Example |
| --- | --- | --- | --- |
| Present | **is** | **zijn** | Het systeem **is** **geconfigureerd**. |
| Past | **was** | **waren** | De data **waren** **geëxporteerd**. |

| Dutch | English |
| --- | --- |
| De **configuratie** **is** **afgerond**. | The configuration is completed. |
| De **documentatie** **is** **bijgewerkt**. | The documentation has been updated. |
| De **gegevens** **zijn** **versleuteld**. | The data are encrypted. |
| De **audit** **was** **vorige** **week** **afgerond**. | The audit was completed last week. |

**Zijn** agrees with the subject: **het rapport is**, **de data zijn**.

### Timeline: from wordt to is

Picture a project moving through stages:

| Stage | Dutch |
| --- | --- |
| 1. In progress | Het rapport **wordt** **gemaakt**. |
| 2. Done | Het rapport **is** **gemaakt**. |
| 3. Ready for use | Het rapport **is** **klaar** / **gereed**. |

| Dutch | English |
| --- | --- |
| **Terwijl** de **validatie** **wordt** **uitgevoerd**, **is** de **export** **nog** **niet** **vrijgegeven**. | While validation is being carried out, the export has not yet been released. |
| **Zodra** alles **is** **geverifieerd**, **wordt** de **wijziging** **doorgevoerd**. | As soon as everything has been verified, the change is implemented. |
| **Nadat** het **rapport** **is** **goedgekeurd**, **wordt** het **gepubliceerd**. | After the report has been approved, it is published. |

### Result passives without a clear agent

Some **zijn** passives describe what **happened** to something, with no doer named or even relevant:

| Dutch | English |
| --- | --- |
| Mijn **fiets** **is** **gestolen**. | My bike has been stolen. |
| Het **raam** **is** **beschadigd**. | The window is damaged. |
| Mijn **sleutel** **is** **verloren**. | My key is lost. |
| De **e-mail** **is** **verzonden**. | The email has been sent. |

Here **zijn** is natural. **Wordt gestolen** would mean _is being stolen right now_.

### Er is / Er zijn + participle

When the subject is indefinite or introduced for the first time, Dutch uses **er**:

| Pattern | Example |
| --- | --- |
| **Er** **is** + singular | **Er** **is** **een** **fout** **gemaakt**. |
| **Er** **zijn** + plural | **Er** **zijn** **wijzigingen** **doorgevoerd**. |

| Dutch | English |
| --- | --- |
| **Er** **zijn** **drie** **fouten** **gemaakt** **in** het **rapport**. | Three mistakes have been made in the report. |
| **Er** **is** **een** **nieuwe** **richtlijn** **ingevoerd**. | A new guideline has been introduced. |
| **Er** **zijn** **geen** **gegevens** **verloren** **gegaan**. | No data have been lost. |

From [A2 Lesson 16](/notes/a2/16-er-constructions): **er** + **zijn** for existence; here **er** + **zijn** + participle for _there have been_.

### Subordinate clauses with zijn passive

With **zijn**, the auxiliary comes **before** the participle at the end (like the perfect tense):

| Main | Subordinate |
| --- | --- |
| Ik meld … | … **dat** het rapport **is** **goedgekeurd**. |
| … **omdat** … | … **omdat** de data **zijn** **geëxporteerd**. |
| … **nadat** … | … **nadat** alles **is** **geverifieerd**. |

| Dutch | English |
| --- | --- |
| **Ik** **zie** **dat** de **configuratie** **is** **afgerond**. | I see that the configuration is completed. |
| **Omdat** de **audit** **is** **afgerond**, **kunnen** wij **doorgaan**. | Because the audit is completed, we can continue. |
| **Voordat** de **export** **is** **vrijgegeven**, **moet** alles **worden** **gecontroleerd**. | Before the export is released, everything must be checked. |

Compare B1 L8 **worden** subordinate: **goedgekeurd moet worden** vs **is goedgekeurd**.

### moet / mag / kan zijn + participle

**Modal + zijn + participle** expresses that something **must / may / can be in a completed state**:

| Dutch | English |
| --- | --- |
| Het rapport **moet** **voor** **vrijdag** **afgerond** **zijn**. | The report must be finished by Friday. |
| De **data** **moeten** **versleuteld** **zijn** **voordat** ze **worden** **geëxporteerd**. | The data must be encrypted before they are exported. |
| **Mag** dit **al** **verzonden** **zijn**? | May this already have been sent? |
| Dit **kan** **niet** **juist** **zijn** **ingediend**. | This cannot have been submitted correctly. |

| Dutch | English |
| --- | --- |
| **Zodra** alles **is** **geregistreerd**, **krijgt** u **een** **bevestiging**. | As soon as everything has been registered, you receive a confirmation. |
| **Het** **project** **moet** **halverwege** **de** **maand** **opgeleverd** **zijn**. | The project must be delivered midway through the month. |

### door + agent with zijn (recognition)

With **zijn**, **door** is possible but less frequent than with **worden**:

| Dutch | English |
| --- | --- |
| Het **boek** **is** **door** hem **geschreven**. | The book was written by him. |
| De **fout** **is** **door** het **systeem** **veroorzaakt**. | The error was caused by the system. |

In everyday Dutch, **zijn** passives often **omit** the agent.

### worden vs zijn in the same conversation

| Question (process) | Answer (result) |
| --- | --- |
| **Wordt** het rapport **al** **gemaakt**? | Ja, het **is** **afgerond**. |
| **Worden** de **data** **geëxporteerd**? | Nee, ze **zijn** **nog** **niet** **vrijgegeven**. |
| **Wordt** de **deur** **gesloten**? | Ja, ze **is** **al** **gesloten**. |

### Combining with B1 grammar

| Topic | Example |
| --- | --- |
| B1 L2 **die/dat** | Het **rapport** **dat** **is** **goedgekeurd**, **ligt** **op** **SharePoint**. |
| B1 L7 **conditional past** | **Als** alles **was** **afgerond**, **zou** de **export** **zijn** **vrijgegeven**. |
| B1 L8 **worden** | **Voordat** het **is** **afgerond**, **wordt** het **nog** **gevalideerd**. |

### Preview: indirect speech (Lesson 10)

Result passives appear often in reported speech: **Hij zegt dat het rapport is afgerond**. Full **indirect speech** patterns come in [Lesson 10](/notes/b1/10-indirect-speech).

  </CardContent>

<ShadCnCard className="my-4">

  </CardHeader>

  <CardContent>

    **zijn** passive: … **dat** het rapport **is** **goedgekeurd**.

  </CardContent>
