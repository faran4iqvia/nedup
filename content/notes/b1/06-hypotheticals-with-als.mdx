## Introduction

At A2 you learned **real** conditions: **Als ik tijd heb, kom ik**. B1 opens **unreal** hypotheticals: situations that are **not true now**, but you imagine anyway. Dutch uses **past tense** in the **als** clause and **zou** in the result: **Als ik rijk was, zou ik reizen**, **Als ik meer tijd had, zou ik een cursus doen**, **Als wij in Amsterdam woonden, zouden wij vaker fietsen**. After this lesson you can discuss dreams, alternatives, and _what if_ scenarios the way Dutch speakers do in conversation and meetings.

## Quick Review

From [A2 Lesson 20](/notes/a2/20-conditional-als): **als** + verb last; real conditions with **present** (**Als het regent, blijf ik thuis**); **wat als**, **in dat geval**, **tenzij**, **zolang**. From [B1 Lesson 5](/notes/b1/05-zou-politeness-and-advice): **zou/zouden** for politeness and **Als ik jou was, zou ik …** From [B1 Lesson 4](/notes/b1/04-future-with-zullen): **zou** as past of **zullen**. Today **als** + **past** pairs with **zou** for **unreal** meaning.

## Grammar and Language Explanation

### Real vs unreal: the B1 split

| Type | **als** clause | Main clause | Example |
| --- | --- | --- | --- |
| **Real** (A2) | **present** | present / future | **Als** ik tijd **heb**, **kom** ik. |
| **Unreal** (B1) | **past** (imperfect) | **zou** + infinitive | **Als** ik rijk **was**, **zou** ik **reizen**. |

| Dutch | English |
| --- | --- |
| **Als** ik **morgen** **kan**, **bel** ik je. (real) | If I can tomorrow, I will call you. |
| **Als** ik **meer** **tijd** **had**, **zou** ik **bellen**. (unreal) | If I had more time, I would call. |
| **Als** het **regent**, **gaan** wij **niet**. (real) | If it rains, we do not go. |
| **Als** het **nu** **regende**, **zouden** wij **thuisblijven**. (unreal) | If it were raining now, we would stay home. |

The **past** in **als** does not always mean _back then_. It signals **unreal** or **imaginary** in the present.

### The core pattern: als + was/had/kon + zou

| **als** clause (past, verb last) | Main clause (**zou** + infinitive) |
| --- | --- |
| **Als** ik rijk **was** | **zou** ik **reizen** |
| **Als** ik meer geld **had** | **zou** ik **kopen** |
| **Als** ik Nederlands **kon** spreken | **zou** ik **in** **Nederland** **werken** |
| **Als** wij **in** **Utrecht** **woonden** | **zouden** wij **vaker** **fietsen** |

| Dutch | English |
| --- | --- |
| **Als** ik **een** **week** **langer** **had**, **zou** ik **het** **project** **afmaken**. | If I had one more week, I would finish the project. |
| **Als** hij **hier** **was**, **zou** hij **ons** **helpen**. | If he were here, he would help us. |
| **Als** wij **meer** **budget** **hadden**, **zouden** wij **een** **extra** **persoon** **aannemen**. | If we had more budget, we would hire an extra person. |
| **Als** ik **jou** **was**, **zou** ik **wachten**. | If I were you, I would wait. |

**Ik was / hij was** for _were_. **Wij hadden / zij hadden** for _had_ (plural). **Zou** for **ik/jij/hij/zij**; **zouden** for **wij/jullie/zij**.

### als clause first or second

Same V2 rule as A2:

| Order | Example |
| --- | --- |
| **als** first | **Als** ik rijk **was**, **zou** ik **reizen**. |
| main first | Ik **zou** **reizen**, **als** ik rijk **was**. |

| Dutch | English |
| --- | --- |
| **Zou** je **me** **helpen**, **als** je **tijd** **had**? | Would you help me if you had time? |
| **Als** de **baas** **niet** **zo** **streng** **was**, **zou** het **werk** **rustiger** **zijn**. | If the boss were not so strict, work would be calmer. |

### zou + modal in the result

The main clause can use **zou** + **kunnen / moeten / willen**:

| Dutch | English |
| --- | --- |
| **Als** ik **meer** **energie** **had**, **zou** ik **kunnen** **sporten**. | If I had more energy, I could exercise. |
| **Als** wij **meer** **tijd** **hadden**, **zouden** wij **moeten** **testen**. | If we had more time, we would have to test. |
| **Als** het **kon**, **zou** ik **graag** **verhuizen**. | If it were possible, I would like to move. |

### Stel dat and stel je eens voor

| Form | Use | Example |
| --- | --- | --- |
| **Stel dat** … | suppose that | **Stel dat** het **mislukt**, **wat** **dan**? |
| **Stel je eens voor dat** … | imagine that | **Stel** je **eens** **voor** **dat** wij **in** **Lissabon** **wonen**. |
| **Wat als** …? | what if | **Wat als** de **klant** **nee** **zegt**? |

| Dutch | English |
| --- | --- |
| **Stel dat** wij **dubbel** **zoveel** **budget** **hadden**. **Zouden** wij **dan** **sneller** **groeiën**? | Suppose we had double the budget. Would we grow faster then? |
| **Stel** je **voor** **dat** je **niet** **hoeft** **te** **werken**. **Wat** **zou** je **doen**? | Imagine you did not have to work. What would you do? |
| **Wat als** het **project** **uitloopt**? **Hebben** wij **een** **plan** **B**? | What if the project runs over? Do we have a plan B? |

**Stel dat** and **stel je voor dat** take **verb last** like **als**.

### Wat als: real worry or hypothetical

**Wat als** can be a **real** concern or a **hypothetical**:

| Real worry | Hypothetical |
| --- | --- |
| **Wat als** hij **morgen** **niet** **komt**? | **Wat als** wij **miljonair** **waren**? |
| (possible problem) | (imaginary scenario) |

Answer with **dan** … or **zou** … depending on context.

### In theorie vs in de praktijk

| Dutch | English |
| --- | --- |
| **In theorie** **zou** het **kunnen**, **maar** **in** **de** **praktijk** **is** het **lastig**. | In theory it could work, but in practice it is difficult. |
| **In** **een** **ideaal** **geval** **zouden** wij **vandaag** **klaar** **zijn**. | In an ideal case we would be done today. |
| **Theoretisch** **zou** dat **kunnen** **kloppen**. | Theoretically that could be correct. |

### Combining with B1 grammar

| Topic | Hypothetical example |
| --- | --- |
| B1 L1 **hoewel** | **Hoewel** ik **graag** **zou** **willen**, **kan** ik **niet**; **als** ik **meer** **tijd** **had**, **zou** ik **komen**. |
| B1 L2 **die/dat** | **Het** **plan** **dat** wij **zouden** **kiezen**, **hing** **af** **van** **het** **budget**. |
| B1 L3 **waarover** | **Het** **onderwerp** **waarover** wij **speculeerden**, **was** **onrealistisch**. |
| B1 L5 **zou** | **Zou** je **me** **kunnen** **helpen**, **als** je **ervaring** **had**? |

### Preview: past unreal (Lesson 7)

When the **if** clause refers to something **already past** and **counterfactual**, Dutch often uses **past perfect** in **als**:

| B1 L6 (present unreal) | B1 L7 preview (past unreal) |
| --- | --- |
| **Als** ik **meer** **tijd** **had**, **zou** ik **komen**. | **Als** ik **het** **had** **geweten**, **zou** ik **zijn** **gekomen**. |

Full **Als ik had geweten, …** patterns come in [Lesson 7](/notes/b1/07-conditional-past).

    >
    > **Mila:** **Goed** **idee**. **In** **het** **slechtste** **geval** **verschuift** **alles** **een** **kwartaal**. **De** **impact** **zou** **groot** **zijn**, **maar** **niet** **onomkeerbaar**.

    **In English:**

    > **Mila:** Suppose we had twenty percent less budget. What would you do then?
    >
    > **Sam:** In theory we could speed up, but in practice that is unrealistic.
    >
    > **Mila:** What if we have to cut one FTE? That is no longer fiction.
    >
    > **Sam:** If we had more time, we would do a scenario analysis. Now we have to be pragmatic.
    >
    > **Mila:** Just imagine that in the best case we already knew next month what is coming.
    >
    > **Sam:** Then we could plan more calmly. If I were you, I would already model a plan B now.
    >
    > **Mila:** Good idea. In the worst case everything shifts by a quarter. The impact would be large, but not irreversible.
    >
    > **Sam:** Exactly. Speculating is fine, but in the end we choose one achievable variant.
