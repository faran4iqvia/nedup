## Introduction

You know active sentences (**Mila schrijft de brief**) and past participles from [A2 Lesson 2](/notes/a2/02-perfect-tense-intro) (**geschreven**, **gemaakt**, **gewerkt**). Dutch often puts the **object** first and hides the doer: **De brief wordt geschreven**, **Er wordt gewerkt**, **De vergadering wordt verplaatst**. After this lesson you can read announcements, office updates, and signs in the passive, and form basic **worden** + participle sentences yourself.

## Quick Review

From [A2 Lesson 2](/notes/a2/02-perfect-tense-intro): participles (**ge** + stem + **t/d**, or irregular **geschreven**). From [A2 Lesson 3](/notes/a2/03-perfect-irregular): **worden** → **werd** / **geworden**. From [A2 Lesson 16](/notes/a2/16-er-constructions): **Er wordt** + activity. From [A2 Lesson 17](/notes/a2/17-workplace): **de vergadering**, **het rapport**. Today **worden** becomes the passive helper verb.

## Grammar and Language Explanation

### Active vs passive

| Active | Passive |
| --- | --- |
| **Mila** **schrijft** de brief. | De brief **wordt** **geschreven**. |
| **Wij** **maken** het rapport. | Het rapport **wordt** **gemaakt**. |
| **Zij** **verplaatst** de vergadering. | De vergadering **wordt** **verplaatst**. |

| Role | Active | Passive |
| --- | --- | --- |
| Focus | who does it | what happens to something |
| Subject | doer (**Mila**) | receiver (**de brief**) |
| Verb | main verb (**schrijft**) | **worden** + participle (**wordt geschreven**) |

Use the passive when the **action** matters more than **who** does it: signs, news, procedures, office updates.

  Full passive tables live in the Grammar Track: [Passive Voice: _worden_ + Past Participle](/notes/grammar/30-passive-voice). Agents with **door** (**door Mila**) come at B1.
</Callout>

### worden + participle: the A2 formula

| Tense | Pattern | Example |
| --- | --- | --- |
| Present | **wordt** / **worden** + participle | De brief **wordt** **geschreven**. |
| Past | **werd** / **werden** + participle | De brief **werd** **geschreven**. |
| Question | **Wordt** … ? | **Wordt** het rapport **vandaag** **gemaakt**? |

| Dutch | English |
| --- | --- |
| Het rapport **wordt** **nu** **gemaakt**. | The report is being made now. |
| De brieven **worden** **morgen** **verstuurd**. | The letters will be sent tomorrow. |
| De deur **wordt** **om** vijf uur **gesloten**. | The door is closed at five o'clock. |
| **Werd** de vergadering **verplaatst**? | Was the meeting rescheduled? |

**Worden** agrees with the **subject**: **de brief wordt**, **de brieven worden**.

### worden conjugation (present and past)

| Pronoun | present | past |
| --- | --- | --- |
| ik | **word** | **werd** |
| jij / je / u | **wordt** | **werd** |
| hij / zij / het | **wordt** | **werd** |
| wij / jullie / zij | **worden** | **werden** |

| Dutch | English |
| --- | --- |
| Ik **word** **gebeld**. | I am being called. / I get called. |
| **Word** je **uitgenodigd**? | Are you invited? |
| De taken **werden** **gisteren** **afgehandeld**. | The tasks were handled yesterday. |

### Past participle in the passive

The participle is the **same** form as in the perfect tense:

| Infinitive | Participle | Passive |
| --- | --- | --- |
| schrijven | geschreven | **wordt** geschreven |
| maken | gemaakt | **wordt** gemaakt |
| werken | gewerkt | **wordt** gewerkt |
| versturen | verstuurd | **worden** verstuurd |
| verplaatsen | verplaatst | **wordt** verplaatst |

| Dutch | English |
| --- | --- |
| De e-mail **wordt** **vandaag** **verstuurd**. | The email is sent today. |
| Het systeem **wordt** **gecontroleerd**. | The system is being checked. |
| De kantoren **worden** **gerenoveerd**. | The offices are being renovated. |

### Er wordt …: impersonal passive

When there is **no specific thing** as subject, Dutch uses **er**:

| Dutch | English |
| --- | --- |
| **Er** **wordt** **hier** **niet** **gerookt**. | No smoking here. / Smoking is not allowed here. |
| **Er** **wordt** **vandaag** **gewerkt** **aan** het **rapport**. | Work is being done on the report today. |
| **Er** **worden** **nieuwe** **regels** **ingevoerd**. | New rules are being introduced. |
| **Wordt** **er** **nu** **vergaderd**? | Is there a meeting now? |

From [A2 Lesson 16](/notes/a2/16-er-constructions): **er** + **worden** + participle is very common on signs and in announcements.

### door + agent (light preview)

You can name the doer with **door** + person. Full practice comes at B1; recognise the pattern now:

| Without agent | With **door** |
| --- | --- |
| De brief **wordt** **geschreven**. | De brief **wordt** **door** Mila **geschreven**. |
| Het rapport **wordt** **gemaakt**. | Het rapport **wordt** **door** het **team** **gemaakt**. |

**Door** comes **before** the agent, **between** **worden** and the participle or after **worden**: **wordt door Mila geschreven**.

### When to use passive in real Dutch

| Situation | Example |
| --- | --- |
| Signs / rules | **Er** **wordt** **niet** **gefotografeerd**. |
| Office updates | De vergadering **wordt** **verplaatst**. |
| Process steps | Uw aanvraag **wordt** **verwerkt**. |
| Unknown doer | Mijn fiets **is** **gestolen**. (result with **zijn**; see below) |

| Dutch | English |
| --- | --- |
| Uw **betaling** **wordt** **verwerkt**. | Your payment is being processed. |
| De **deur** **wordt** **om** acht uur **gesloten**. | The door is closed at eight o'clock. |
| **Hier** **wordt** **niet** **gepraat**. | No talking here. |

### Passive with zijn (result): recognition only

Dutch also uses **zijn** + participle for a **result** or **state**:

| **worden** (process) | **zijn** (result) |
| --- | --- |
| De soep **wordt** **gemaakt**. | De soep **is** **gemaakt**. |
| Het huis **wordt** **gebouwd**. | Het huis **is** **gebouwd**. |

| Dutch | English |
| --- | --- |
| Het rapport **is** **klaar**. Het **is** **gisteren** **gemaakt**. | The report is ready. It was made yesterday. |
| De **deur** **is** **gesloten**. | The door is closed. |

At A2, **produce** passives with **worden**. **Recognise** **is** + participle as _has been done_ / _is done_.

### In subordinate clauses

The participle stays near the end; **worden** follows:

| Main clause | Subordinate clause |
| --- | --- |
| De brief **wordt** **geschreven**. | … **omdat** de brief **geschreven** **wordt**. |
| **Ik** **hoop** **dat** het **snel** **gaat**. | … **dat** het rapport **vandaag** **wordt** **gemaakt**. |

| Dutch | English |
| --- | --- |
| **Ik** **meld** **dat** de vergadering **verplaatst** **wordt**. | I report that the meeting is being rescheduled. |
| **Omdat** **er** **gewerkt** **wordt**, **is** het **stil**. | Because work is being done, it is quiet. |

From [A2 Lesson 14](/notes/a2/14-subordinate-want-omdat): verb cluster at the end.

<Callout title="Pronunciation Focus" type="info">
  **Wordt** and **word** sound like **wort** / **wort** with Dutch **w**. In **er wordt gewerkt**, link **wordt-gewerkt**. See [Connected Speech](/notes/pronunciation/05-connected-speech).
</Callout>
