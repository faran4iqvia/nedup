## Past tense: when to use which

| Tense | Dutch | Use when |
| --- | --- | --- |
| **Simple past** | **Ik werkte** gisteren. | Stories, written narrative, some spoken past |
| **Perfect** | **Ik heb gewerkt**. | Spoken default for completed actions |
| **Both often work** | **Ik ging** / **Ik ben gegaan** | Spoken: perfect common; **ging** still frequent |

**Perfect formula:** **hebben/zijn** + participle (**ge** + stem + **d/t**)

| | **hebben** | **zijn** (movement/change) |
| --- | --- | --- |
| Example | Ik **heb** gegeten. | Ik **ben** gegaan. |

## omdat vs want (word order)

| | **omdat** (because) | **want** (because, main clause) |
| --- | --- | --- |
| Word order | **Omdat** ik **moe ben**, ga ik vroeg naar bed. | Ik ga vroeg naar bed, **want** ik **ben** moe. |
| Verb position | Verb to **end** of omdat clause | Normal **V2** in want clause |

❌ **Omdat ik ben ziek, ik blijf thuis.**\
✅ **Omdat ik ziek ben, blijf ik thuis.**

## er (three jobs at A2)

| Job | Example | English |
| --- | --- | --- |
| **There** | **Er** zijn veel fietsen. | There are many bikes. |
| **About it** | Ik praat **er** over. | I talk about it. |
| **Some** | **Er** is koffie. | There is (some) coffee. |

## Opinion phrases

| Dutch | English |
| --- | --- |
| **Ik vind dat** … | I think that … |
| **Ik vind het** + adj | I find it + adj |
| **Volgens mij** | In my opinion |
| **Ik ben het eens / oneens met** … | I agree / disagree with … |
| **Dat klopt.** / **Dat klopt niet.** | That is right / wrong |

## Comparatives (quick)

| | Comparative | Superlative |
| --- | --- | --- |
| Rule | **groter**, **interessanter** (+ **r**) | **het grootst**, **het interessantst** |
| Irregular | **beter**, **meer** | **het best**, **het meest** |
