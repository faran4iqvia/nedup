## Introduction

Wrong stress makes Dutch hard to understand even when vowels are correct. **WOON-kamer** and **woon-KA-mer** feel like different habits to a Dutch ear. This lesson belongs to the A0 stage of the Pronunciation Track. Study it alongside [A0 Lesson 8](/notes/a0/08-numbers-and-time) onward and revisit with every new long word.

Mark stress on the **first strong syllable** in new vocabulary.

## Explanation

### Default: stress on the first syllable

Most Dutch words stress the **first** syllable:

| Word | Stress pattern |
| --- | --- |
| **WOON-kamer** | first |
| **NEDER-land** | first |
| **SPRE-ken** | first |
| **FA-mi-lie** | first (familie) |
| **IN-ter-es-sant** | first (interessant) |

| Dutch | English |
| --- | --- |
| **Ik** **WOON** **in** **UT-recht**. | I live in Utrecht. |
| **We** **SPRE-ken** **Nederlands**. | We speak Dutch. |

### Prefixes are usually unstressed

These prefixes are **weak** at the start of a word:

| Prefix | Example | Stress on |
| --- | --- | --- |
| **be-** | be**STAAN**, be**GRIJP** | stem |
| **ge-** | ge**MAKEN**, ge**LOVEN** | stem |
| **ver-** | ver**STAAN**, ver**TEL** | stem |
| **ont-** | ont**MOETEN**, ont**VANG** | stem |
| **her-** | her**HALEN**, her**INNER** | stem |

❌ stress **VER-staan** → ✅ ver**STAAN**

### Compound words

Compounds stress the **first part**:

| Compound | Stress |
| --- | --- |
| **FIETS-band** | fiets |
| **KOF-fie-zet-apparaat** | koffie (first element) |
| **WOON-werk-ver-keer** | woon |
| **TREIN-station** | trein |

### Words with stress exceptions

Some words stress a **later** syllable (learn as blocks):

| Word | Stress | Note |
| --- | --- | --- |
| **uniVERsiteit** | ver | common B1 word |
| **inforMAtie** | ma | |
| **teleFOON** | foon | |
| **solliciTAtie** | tie | [B1 Lesson 17](/notes/b1/17-work-meetings-and-applications) |

When unsure, check a dictionary with audio.

### Sentence stress vs word stress

| Type | Rule |
| --- | --- |
| Word stress | fixed pattern per word |
| Sentence stress | **content words** louder: nouns, verbs, adjectives |
| Function words | **de**, **het**, **een**, **ik** = weak |

From [Connected Speech](/notes/pronunciation/05-connected-speech): in fast speech, function words shrink but content word stress stays.

## Common mistakes for English speakers

| Mistake | Fix |
| --- | --- |
| English stress on second syllable | Dutch often first: **NEDERland** |
| Stressing **ver-** in **verstaan** | ver**STAAN** |
| Equal stress on compounds | stress first part: **FIETSpad** |
| Flat sentences | stress key words: **Ik **WOON** in Amsterdam** |
