## Introduction

You already say **De stad is groot**, **Het brood is vers**, and **Die jas is te duur**. Now you learn what happens when the adjective sits **before** the noun: **de grote stad**, **het kleine huis**, **een nieuwe trui**. After this lesson you can describe rooms, food, clothes, and people with natural Dutch, not textbook English word order like **de groot stad**.

## Grammar and Language Explanation

### Two positions: after the noun vs before the noun

**Predicative** adjectives come **after** **zijn**, **blijven**, **worden**, or similar verbs. They have **no extra ending**:

| Dutch | English |
| --- | --- |
| De stad is **groot**. | The city is big. |
| Het huis is **groot**. | The house is big. |
| De trui is **mooi**. | The sweater is beautiful. |
| De schoenen zijn **te duur**. | The shoes are too expensive. |

**Attributive** adjectives come **before** the noun. They often add **e**, depending on the article:

| Dutch | English |
| --- | --- |
| **de grote** stad | the big city |
| **het grote** huis | the big house |
| **een grote** stad | a big city |
| **een groot** huis | a big house |

Same adjective, two shapes: **groot** vs **grote**. English does not show this. Dutch does.

### The basic e rule at A1

Use this table until the pattern feels automatic:

| Before | de word | het word |
| --- | --- | --- |
| **de** + noun | **de grote** stad | **het grote** huis |
| **een** + noun | **een grote** stad | **een groot** huis |
| **geen** + noun | **geen grote** stad | **geen groot** huis |
| **plural** | **grote** steden / **de grote** steden | (het words become de in plural) |

| Dutch | English |
| --- | --- |
| **De grote** stad is **rustig**. | The big city is quiet. |
| **Het kleine** appartement is **nieuw**. | The small apartment is new. |
| Ik zoek **een goedkope** jas. | I am looking for a cheap coat. |
| Wij wonen in **een groot** huis. | We live in a big house. |
| Ik koop **geen dure** schoenen. | I do not buy expensive shoes. |
| **De kleine** kamers zijn **schoon**. | The small rooms are clean. |

**Rule in one line:** add **e** with **de** words and with **het + the**; no **e** with **een + het** word or **geen + het** word.

### With deze, die, dit, dat

Demonstratives from [Lesson 9](/notes/a1/09-demonstratives) follow the same logic as **de** and **het**:

| Dutch | English |
| --- | --- |
| **deze grote** trui | this big sweater |
| **die dure** jas | that expensive coat |
| **dit kleine** brood | this small bread |
| **dat nieuwe** horloge | that new watch |
| **deze mooie** bloemen | these beautiful flowers |

| Dutch | English |
| --- | --- |
| **Deze grote** kamer is **leeg**. | This big room is empty. |
| **Dat verse** brood is **lekker**. | That fresh bread is tasty. |
| **Die goedkope** schoenen **kosten** twintig euro. | Those cheap shoes cost twenty euros. |

### Plural adjectives

Plural nouns always use **de**, so the adjective before a plural noun takes **e**:

| Dutch | English |
| --- | --- |
| **grote** huizen | big houses |
| **de grote** huizen | the big houses |
| **deze interessante** boeken | these interesting books |
| **kleine** kinderen | small children |

| Dutch | English |
| --- | --- |
| **De oude** straten zijn **mooi**. | The old streets are beautiful. |
| **Nieuwe** lampen staan in **de** kamer. | New lamps are in the room. |

### Color adjectives

Colors behave like other adjectives: **de rode auto**, **het witte huis**, **een groene trui**.

| Dutch | English |
| --- | --- |
| **de rode** auto | the red car |
| **het witte** huis | the white house |
| **een blauwe** jas | a blue coat |
| **De groene** bomen zijn **mooi**. | The green trees are beautiful. |
| **Het oranje** shirt is **nieuw**. | The orange shirt is new. |

| Dutch | English |
| --- | --- |
| rood | red |
| blauw | blue |
| groen | green |
| geel | yellow |
| wit | white |
| zwart | black |
| oranje | orange |
| bruin | brown |
| paars | purple |

**Oranje** is the same for the color and for the Dutch royal house. **Bruin** also means brown bread (**bruin brood**).

### Words that make adjectives stronger

Place these **before** the adjective. The **e** rule stays the same:

| Word | Meaning | Example |
| --- | --- | --- |
| erg | very | erg duur |
| heel | very / really | heel mooi |
| vrij | quite / fairly | vrij groot |
| best | quite / rather | best interessant |

| Dutch | English |
| --- | --- |
| **De jas is erg duur.** | The coat is very expensive. |
| **Het is een heel lekker** broodje. | It is a very tasty roll. |
| **Een vrij rustige** straat. | A fairly quiet street. |
| **Dat is best een grote** tuin. | That is quite a big garden. |

### Adjectives you already know, now before nouns

| Base | Before de noun | Before het noun (een) |
| --- | --- | --- |
| groot | de grote stad | een groot huis |
| klein | de kleine kamer | een klein kind |
| mooi | de mooie bloem | een mooi cadeau |
| duur | de dure jas | een duur horloge |
| goedkoop | de goedkope trui | een goedkoop boek |
| vers | het verse brood | een vers broodje |
| warm | de warme soep | een warm glas |
| koud | de koude melk | een koud biertje |
| jong | de jonge student | een jong kind |
| oud | de oude straat | een oud huis |

### A few common exceptions at A1

Some adjectives never add **e**: **open** (**de open deur**), **extra**, **één** (**één groot huis**). Adjectives ending in **en** usually do not add another **e**: **houten tafel**, **wollen trui**. You meet more exceptions in A2. For now, learn the main **e** table above.

<Callout title="Pronunciation Focus" type="info">
  The **e** at the end of **grote**, **mooie**, **nieuwe** is often a soft schwa, like the **e** in **de**. Do not pronounce a loud English "guh" at the end. Say **grote stad**, **mooie trui**, **nieuwe kamer** in one smooth flow. See [Vowel Sounds](/notes/pronunciation/01-vowel-sounds).
</Callout>
