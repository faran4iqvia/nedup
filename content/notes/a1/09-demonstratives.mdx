## Introduction

You have been saying **deze jas**, **die bananen**, and **Is dit van jou?** since [Lesson 8](/notes/a1/08-shopping). You also met **Dit is mijn moeder** in [Lesson 4](/notes/a1/04-family). Now you learn the full system: **deze, die, dit, dat** follow the same **de** vs **het** split you know from articles and possessives. After this lesson you can point at anything, contrast two objects, and stop guessing which word to use.

## Quick Review

**De** words take **deze** (near) and **die** (far): **deze tas**, **die schoenen**. **Het** words take **dit** and **dat**: **dit brood**, **dat huis**. From [Lesson 6](/notes/a1/06-food-and-drink): **de appel** → **deze appel**. From [Lesson 5](/notes/a1/05-possessives): **ons huis** → **dit huis**. Today we make the pattern explicit and practise it everywhere.

## Grammar and Language Explanation

### The four demonstratives at a glance

|  | Near (this) | Far (that) |
| --- | --- | --- |
| **de** words | **deze** | **die** |
| **het** words | **dit** | **dat** |
| **plural** (always de) | **deze** | **die** |

| Dutch | English |
| --- | --- |
| **deze** appel | this apple |
| **die** appel | that apple |
| **dit** brood | this bread |
| **dat** brood | that bread |
| **deze** appels | these apples |
| **die** schoenen | those shoes |

The rule mirrors **de/het**: if you know the article, you know the demonstrative form.

### deze and die with de words

| Dutch | English |
| --- | --- |
| deze tas | this bag |
| die jas | that coat |
| deze verkoper | this salesman |
| die klant | that customer |
| deze straat | this street |
| die winkel | that shop |

| Dutch | English |
| --- | --- |
| **Deze** trui is warm. | This sweater is warm. |
| **Die** schoenen zijn duur. | Those shoes are expensive. |
| Ik koop **deze**. | I buy this one. |
| Ik neem **die**. | I take that one. |

**Deze** and **die** also work alone when the noun is clear:

| Dutch | English |
| --- | --- |
| Welke jas? **Deze**. | Which coat? This one. |
| Welke schoenen? **Die**. | Which shoes? Those. |

### dit and dat with het words

| Dutch | English |
| --- | --- |
| dit brood | this bread |
| dat huis | that house |
| dit kind | this child |
| dat eten | that food |
| dit horloge | this watch |
| dat idee | that idea |

| Dutch | English |
| --- | --- |
| **Dit** is lekker. | This is tasty. |
| **Dat** is te duur. | That is too expensive. |
| **Deze** is mijn telefoon. | This is my phone. |
| **Dat** is niet mijn tas. | That is not my bag. |

### Plural: always deze and die

Plural nouns always use **de**, so plurals always use **deze** / **die**:

| Singular | Plural demonstrative |
| --- | --- |
| de appel → **deze** appels / **die** appels |
| het ei → **deze** eieren / **die** eieren |
| de schoen → **deze** schoenen / **die** schoenen |

| Dutch | English |
| --- | --- |
| **Deze** bananen zijn zoet. | These bananas are sweet. |
| **Die** aardappelen zijn goedkoop. | Those potatoes are cheap. |
| Ik neem **deze**. | I take these. |

### Near vs far: how Dutch speakers use distance

| Concept | Dutch | English |
| --- | --- | --- |
| near you | deze / dit | this |
| farther away | die / dat | that |
| here | hier | here |
| there | daar | there |

| Dutch | English |
| --- | --- |
| **Dit** boek hier is interessant. | This book here is interesting. |
| **Dat** huis daar is groot. | That house over there is big. |
| **Deze** stoel is van mij. | This chair is mine. |
| **Die** tafel bij het raam. | That table by the window. |

In real speech, Dutch people sometimes use **die** for things close by when pointing. At A1, keep the clean rule: **deze/dit** = near, **die/dat** = farther. You will hear both in cafes and shops.

### Standalone: pointing without a noun

When the thing is obvious, use **dit** or **dat** alone:

| Dutch | English |
| --- | --- |
| Wat is **dit**? | What is this? |
| Wie is **dat**? | Who is that? |
| **Dat** is Tom. | That is Tom. |
| **Dit** is mijn moeder. | This is my mother. |
| **Dat** klopt niet. | That is not correct. |
| **Dat** is een goed idee. | That is a good idea. |

| Dutch | English |
| --- | --- |
| Is **dit** jouw fiets? | Is this your bicycle? |
| Nee, **dat** is van hem. | No, that one is his. |

### In questions and answers

| Dutch | English |
| --- | --- |
| Wat is dit? | What is this? |
| Wat is dat? | What is that? |
| Wie is deze vrouw? | Who is this woman? |
| Welke tas wil je? Deze of die? | Which bag do you want? This one or that one? |
| Is dat vegetarisch? | Is that vegetarian? |

Word order from [Lesson 2](/notes/a1/02-word-order-svo) stays the same:

| Dutch | English |
| --- | --- |
| **Deze** jas is **te duur**. | This coat is too expensive. |
| **Waarom** koop je **die**? | Why do you buy that one? |
| **Vandaag** eet ik **dit** brood. | Today I eat this bread. |

### Contrasting two things

| Dutch | English |
| --- | --- |
| deze en die | this one and that one |
| niet deze maar die | not this one but that one |
| dit of dat? | this or that? |
| deze wel, die niet | this one yes, that one no |

| Dutch | English |
| --- | --- |
| Ik wil **niet deze** maar **die**. | I want not this one but that one. |
| **Deze** trui past, **die** niet. | This sweater fits, that one does not. |
| **Dit** brood is vers, **dat** brood niet. | This bread is fresh, that bread is not. |

### zo'n and zulke (such a / such)

| Dutch | English |
| --- | --- |
| zo'n dag | such a day |
| zo'n mooie jas | such a beautiful coat |
| zulke appels | such apples |
| zoiets | something like that |

| Dutch | English |
| --- | --- |
| **Zo'n** lekkere koffie! | Such tasty coffee! |
| Ik hou niet van **zulke** schoenen. | I do not like such shoes. |
| Wil je **zoiets**? | Do you want something like that? |

**Zo'n** = **zo** + **een** (such a). **Zulke** goes with plural or de words.

### Other useful patterns with dit/dat

| Dutch | English |
| --- | --- |
| op dit moment | at this moment |
| op dat moment | at that moment |
| in dit geval | in this case |
| in dat geval | in that case |
| dit keer | this time |
| dat keer | that time |
| hetzelfde | the same |

| Dutch | English |
| --- | --- |
| **Op dit moment** ben ik thuis. | At this moment I am at home. |
| **In dat geval** neem ik die. | In that case I take that one. |
| **Hetzelfde** brood, alsjeblieft. | The same bread, please. |

### Warning: die and dat have other jobs

At A1, know two other uses exist so you are not surprised later:

| Word | Other use | Example |
| --- | --- | --- |
| **die** | relative pronoun (who/which) | de man **die** daar staat |
| **dat** | conjunction (that) | Ik denk **dat** hij komt. |

In this lesson, **die** and **dat** point at things. Context makes the meaning clear.

### Demonstratives + lessons 3 to 8 combined

| Dutch | English |
| --- | --- |
| **Deze** soep is **niet** zoet. | This soup is not sweet. |
| **Die** jas is **te duur**. | That coat is too expensive. |
| **Dit** is **mijn** cadeau. | This is my present. |
| **Hoeveel kost die** trui? | How much does that sweater cost? |
| Ik eet **geen** vlees. **Dit** is vegetarisch. | I eat no meat. This is vegetarian. |

## Example Conversation

<ShadCnCard className="my-4">
  <CardHeader>
    <CardTitle className="text-lg">Choosing a gift in a shop</CardTitle>
    <CardDescription>Lisa and Tom compare two items for a birthday present.</CardDescription>
  </CardHeader>

  <CardContent>
    **In Dutch:**

    > **Lisa:** Kijk! **Deze** trui of **die** sjaal?
    >
    > **Tom:** **Deze** trui is mooi. **Die** sjaal is ook leuk.
    >
    > **Lisa:** **Dit** horloge hier is duur. **Dat** horloge daar is goedkoper.
    >
    > **Tom:** **Hoeveel kost dat**?
    >
    > **Lisa:** **Dat** kost vijfendertig euro. **Dit** kost zestig.
    >
    > **Tom:** **In dat geval** neem ik **dat**. **Dit** is te duur.
    >
    > **Lisa:** En voor je moeder? **Deze** bloemen of **dat** boek?
    >
    > **Tom:** **Die** bloemen zijn mooi. **Dat** boek kent ze al.
    >
    > **Lisa:** Oké. **Deze** bloemen en **dat** horloge voor je vader.
    >
    > **Tom:** **Beide**? Prima. **Dat** is **hetzelfde** cadeau als vorige maand, hoor.
    >
    > **Lisa:** **Zo'n** grapje! Kom, naar de kassa.

    **In English:**

    > **Lisa:** Look! This sweater or that scarf?
    >
    > **Tom:** This sweater is beautiful. That scarf is nice too.
    >
    > **Lisa:** This watch here is expensive. That watch over there is cheaper.
    >
    > **Tom:** How much does that one cost?
    >
    > **Lisa:** That one costs thirty five euros. This one costs sixty.
    >
    > **Tom:** In that case I will take that one. This one is too expensive.
    >
    > **Lisa:** And for your mother? These flowers or that book?
    >
    > **Tom:** Those flowers are wonderful. She already knows that book.
    >
    > **Lisa:** Okay. These flowers and that watch for your father.
    >
    > **Tom:** Both? Great. That is the same gift as last month, you know.
    >
    > **Lisa:** Such a joke! Come, to the checkout.

    **Deze/die** for de words, **dit/dat** for het words, **in dat geval**, **beide**, and **hetzelfde** in one shopping scene.
  </CardContent>
</ShadCnCard>
