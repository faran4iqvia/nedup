## Dutch word order (main clause)

Dutch uses **verb second (V2)**: the conjugated verb is always in **position 2**.

| Pattern | Dutch | English |
| --- | --- | --- |
| Statement | **Ik** **werk** vandaag. | I work today. |
| Time first | **Vandaag** **werk** ik. | Today I work. |
| Question (yes/no) | **Werk** jij vandaag? | Do you work today? |
| Question (open) | **Waar** **woon** je? | Where do you live? |
| Modal + infinitive | Ik **wil** koffie **drinken**. | I want to drink coffee. |
| Separable verb | Ik **sta** om zeven uur **op**. | I get up at seven. |
| Negation + separable | Ik **sta** **niet** **op**. | I do not get up. |

**Rule of thumb:** One chunk before the verb, verb in slot 2, everything else after (infinitive or prefix at the very end).

## niet vs geen

| Use | Dutch | English |
| --- | --- | --- |
| **niet** negates verbs, adjectives, adverbs, definite things | Ik **werk niet**. | I do not work. |
| **niet** before separable prefix | Ik sta **niet** op. | I do not get up. |
| **geen** negates indefinite nouns (no de/het) | Ik heb **geen** tijd. | I have no time. |
| **geen** + singular | **Geen** melk, alsjeblieft. | No milk, please. |
| **geen** + plural | **Geen** eieren vandaag. | No eggs today. |

❌ geen de melk · ✅ **geen** melk

## Quick patterns

| Need | Pattern | Example |
| --- | --- | --- |
| Like / love | **Ik hou van** + noun/infinitive | Ik hou van koffie. |
| Opinion | **Ik vind** + adj / **Ik vind dat** … | Ik vind het leuk. |
| Past (regular) | ik + **te/de** | ik werkte, ik speelde |
| Past (common irregular) | stem change | ging, kwam, had, was |
| Reflexive | verb + **me/je/zich** | Ik was **me**. |
| Polite request | **Mag ik** …? / **Kunt u** …? | Mag ik zitten? |
