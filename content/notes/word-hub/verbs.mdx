## About Dutch verbs

Regular (**weak**) verbs form the past with **-te / -de** and the participle with **ge- … -t / -d** — the choice follows the **'t kofschip** rule ([see the Grammar Track](/notes/grammar/19-past-tense-weak-strong)). Irregular (**strong**) verbs change their vowel instead, so those are listed with their full past and participle. The other groups — separable, reflexive, and modal — behave in special ways worth learning as sets.

## Separable verbs

The prefix splits off and goes to the end of the main clause.

| Dutch | English | Splits as |
| --- | --- | --- |
| opstaan | to get up | ik sta … op |
| aankomen | to arrive | ik kom … aan |
| meegaan | to come along | ik ga … mee |
| weggaan | to leave | ik ga … weg |
| opbellen | to call up | ik bel … op |
| terugbellen | to call back | ik bel … terug |
| aantrekken | to put on | ik trek … aan |
| uittrekken | to take off | ik trek … uit |
| meenemen | to bring along | ik neem … mee |
| afspreken | to arrange to meet | ik spreek … af |
| opruimen | to tidy up | ik ruim … op |
| ophangen | to hang up | ik hang … op |
| aanzetten | to turn on | ik zet … aan |
| uitzetten | to turn off | ik zet … uit |

## Modal verbs

The five modals push the main verb to the end as an infinitive.

| Dutch | English | ik-form | Past |
| --- | --- | --- | --- |
| kunnen | can / to be able | kan | kon |
| moeten | must / to have to | moet | moest |
| mogen | may / to be allowed | mag | mocht |
| willen | to want | wil | wilde / wou |
| zullen | shall / will | zal | zou |
