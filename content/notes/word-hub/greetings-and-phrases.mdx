## About this page

**~35 phrases** · These are not single dictionary words — they are **fixed chunks** you hear from day one. Use them before diving into grammar. Pairs with [A0 Greetings](/notes/a0/01-greetings-and-introductions) and [A0 Survival Phrases](/notes/a0/14-getting-by-survival-phrases).

## Greetings

| Dutch | English | When |
| --- | --- | --- |
| hallo | hello | informal, any time |
| hoi | hi | informal |
| goedemorgen | good morning | until ~12:00 |
| goedemiddag | good afternoon | ~12:00–18:00 |
| goedenavond | good evening | from ~18:00 |
| goedenacht | good night | going to bed |
| dag | bye / hi | informal hello or goodbye |
| doei | bye | informal |
| tot ziens | goodbye | see you later |
| tot straks | see you soon | same day |
| tot morgen | see you tomorrow | |
| welkom | welcome | |

## Introductions

| Dutch | English |
| --- | --- |
| ik heet … | my name is … |
| hoe heet je? / hoe heet u? | what is your name? |
| aangenaam / prettig kennis te maken | pleased to meet you |
| ik kom uit … | I come from … |
| ik woon in … | I live in … |

## Polite essentials

| Dutch | English |
| --- | --- |
| alstublieft | please (formal) / here you are |
| alsjeblieft | please (informal) |
| dank je / dank je wel | thank you (informal) |
| dank u / dank u wel | thank you (formal) |
| graag gedaan | you're welcome |
| geen probleem | no problem |
| sorry / pardon | sorry / excuse me |
| mag ik …? | may I …? |
| sorry, ik spreek nog weinig Nederlands | sorry, I don't speak much Dutch yet |

## Survival

| Dutch | English |
| --- | --- |
| ik begrijp het niet | I don't understand |
| kunt u dat herhalen? | can you repeat that? |
| langzamer alstublieft | more slowly, please |
| hoe zeg je … in het Nederlands? | how do you say … in Dutch? |
| wat betekent dat? | what does that mean? |
| ik weet het niet | I don't know |

## Small talk openers

| Dutch | English |
| --- | --- |
| hoe gaat het? | how are you? |
| goed, dank je | fine, thanks |
| en met jou? / en met u? | and you? |
| het gaat wel | so-so / OK |
| lekker weer | nice weather |

See [Particles](/notes/word-hub/particles) for **hoor**, **toch**, **even** — they appear inside many of these phrases.
