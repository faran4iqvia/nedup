## About Dutch conjunctions

The trap with Dutch conjunctions is **word order**. There are three groups: coordinating (verb keeps its normal spot), subordinating (verb goes to the **end**), and adverbial (they cause **inversion** — verb before subject). See [Conjunctions Overview](/notes/grammar/26-subordinate-want-omdat) and [Extended Subordinate Clauses](/notes/grammar/31-extended-subordinate-clauses).

## Coordinating (normal word order)

These join two main clauses without changing the verb position.

| Dutch | English | Example |
| --- | --- | --- |
| en | and | Ik werk **en** ik studeer. |
| maar | but | Ik wil, **maar** ik kan niet. |
| want | because / for | Ik blijf thuis, **want** ik ben ziek. |
| of | or | Koffie **of** thee? |
| dus | so | Het regent, **dus** ik blijf thuis. |

## Subordinating (verb to the end)

These send the verb of their clause to the **end**.

| Dutch | English | Example |
| --- | --- | --- |
| omdat | because | … **omdat** ik ziek **ben**. |
| dat | that | Ik denk **dat** het **klopt**. |
| als | if / when | **Als** het **regent**, blijf ik thuis. |
| toen | when (past) | **Toen** ik jong **was** … |
| terwijl | while | **Terwijl** ik **kook** … |
| hoewel | although | **Hoewel** het **regent** … |
| zodat | so that | … **zodat** iedereen het **weet**. |
| voordat | before | **Voordat** ik **ga** … |
| nadat | after | **Nadat** ik **at** … |
| sinds | since | **Sinds** ik hier **woon** … |
| zodra | as soon as | **Zodra** hij **komt** … |
| of | whether | Ik weet niet **of** hij **komt**. |
| tenzij | unless | … **tenzij** het **regent**. |

## Adverbial (inversion — verb before subject)

When these start a clause, the verb comes **before** the subject.

| Dutch | English | Example |
| --- | --- | --- |
| daarom | therefore | Het regent. **Daarom** blijf ik thuis. |
| toch | yet / still | **Toch** ga ik. |
| dan | then | Eerst eten, **dan** gaan we. |
| daarna | after that | **Daarna** ging ik naar huis. |
| bovendien | moreover | **Bovendien** is het duur. |
