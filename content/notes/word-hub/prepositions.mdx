## About Dutch prepositions

**~35 prepositions** · Prepositions rarely translate one-to-one, so learn each with a real example. Watch the classic pairs: **in / op** for location, **naar / in** for movement versus place, and **bij** for "at someone's / near". For fixed verb + preposition combinations, see [Prepositions of Place](/notes/grammar/22-prepositions-place).

## Place and direction

| Dutch | English | Example |
| --- | --- | --- |
| in | in | Ik woon **in** Amsterdam. |
| op | on | Het boek ligt **op** de tafel. |
| onder | under | De kat zit **onder** de stoel. |
| boven | above | De lamp hangt **boven** de tafel. |
| naast | next to | Ik zit **naast** Tom. |
| tussen | between | **Tussen** twee huizen. |
| achter | behind | De tuin is **achter** het huis. |
| voor | in front of | Ik wacht **voor** de deur. |
| naar | to (movement) | Ik ga **naar** huis. |
| uit | out of / from | Ik kom **uit** India. |
| door | through | We lopen **door** het park. |
| langs | along / past | We fietsen **langs** de rivier. |
| rond | around | We lopen **rond** het plein. |
| tegen | against | De fiets staat **tegen** de muur. |
| aan | on / at (surface, street) | Ik woon **aan** de gracht. |

## Time

| Dutch | English | Example |
| --- | --- | --- |
| om | at (clock time) | De les begint **om** acht uur. |
| op | on (day) | **Op** maandag werk ik. |
| in | in (month/year) | **In** mei / **in** 2025. |
| tot | until | **Tot** morgen! |
| vanaf | from (time) | **Vanaf** maandag ben ik vrij. |
| tijdens | during | **Tijdens** de les. |
| binnen | within | **Binnen** een uur. |
| na | after | **Na** het eten. |
| voor | before | **Voor** het weekend. |

## Other common prepositions

| Dutch | English | Example |
| --- | --- | --- |
| met | with | Ik ga **met** de trein. |
| zonder | without | Koffie **zonder** suiker. |
| van | of / from | Een vriend **van** mij. |
| bij | at / near / with | Ik ben **bij** de dokter. |
| over | about / over | Een boek **over** Nederland. |
| volgens | according to | **Volgens** mij klopt het. |
| dankzij | thanks to | **Dankzij** jou. |
| ondanks | despite | **Ondanks** de regen. |
| te | at / to (infinitive) | Ik ga **naar** school **om te** leren. |
| sinds | since (time) | **Sinds** maandag ben ik ziek. |
