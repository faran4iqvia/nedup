## Font & Language Components

<div className="mb-6 text-base text-muted-foreground">
  These components allow you to mix different fonts and languages in a single
  line. Each has three size variants: Base, Md (Medium), and Lg (Large).
</div>

### Sans Font (English)

<P>
  This is <SansBase>base size sans</SansBase> text mixed with{' '}
  <SansMd>medium size sans</SansMd> and <SansLg>large size sans</SansLg>.
</P>

### Urdu Font

<P>
  <UrduBase>یہ بیس سائز اردو ہے</UrduBase> اور{' '}
  <UrduMd>یہ درمیانی سائز ہے</UrduMd> اور <UrduLg>یہ بڑا سائز ہے</UrduLg>۔
</P>

### Arabic Font

<P>
  <ArabicBase>فَانطَلَقَا حَتَّىٰ </ArabicBase> و{' '}
  <ArabicMd>فَانطَلَقَا حَتَّىٰ </ArabicMd> و{' '}
  <ArabicLg>فَانطَلَقَا حَتَّىٰ </ArabicLg>۔
</P>

### Mixed Languages in Single Line

<P>
  <SansMd>This is English</SansMd> مع <UrduMd>اردو</UrduMd> و{' '}
  <ArabicMd>عربي</ArabicMd> في سطر واحد!
</P>
